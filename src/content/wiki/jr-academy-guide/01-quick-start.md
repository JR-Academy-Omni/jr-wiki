## 匠人学院是什么

**全球华人 AI 学习第一站**。

中文站 [jiangren.com.au](https://jiangren.com.au) 面向全球华人提供 AI 学习内容；英文站 [jiangren.com.au/en](https://jiangren.com.au/en) 面向澳洲本地求职者提供 IT 职业培训。

具体覆盖：

- **学习内容**：500+ Learn 章节、200+ CheatSheets、600+ Blog、1000+ wiki 页（你正在看的就是其中一本）
- **互动 Lab**：Frontend / AWS / Azure / Python / LLM / Prompt / Vibe Coding 7 大方向的可执行实验
- **三大工具产品**：考证匠（IT 认证刷题 + AI Tutor）、求职匠（简历 + 面试）、牛小匠（澳洲大学课程 AI 答疑）
- **Bootcamp**：AI Engineer / Web Code / Data Engineer / Cloud DevOps 等多个方向，含自学版 / 教学版 / 陪跑版三档
- **认证 + 题库**：AWS / Azure / GCP / CKA 等 IT 认证完整学习路径

## 注册方式

打开 [jiangren.com.au](https://jiangren.com.au)，右上角「登录」：

```
┌────────────────────────────┐
│  📧 邮箱 + 密码            │
│  🔵 Google 一键登录        │
│  💚 微信扫码（中文站特有） │

```

注册后送 7 天免费 Trial（试用期可用全部 BASIC 档权益），不需要立刻付费。

### 中英文站点切换

URL 上有 `/en/` 前缀就是英文版，没前缀是中文。中文是默认 locale，英文需要显式访问。每个页面右上角有语言切换按钮，部分内容只有中文版（比如本指南）。

## 移动端入口

匠人不是单纯一个网站。**考证匠 = App + 小程序 + 插件**多端形态：

### iOS / Android App（考证匠 Cert Master）

App Store / Google Play 搜「考证匠」（英文：Cert Master, 包名 `com.jracademy.certmaster`）。Phase 4 上线进度：

- iOS：App Store 上架进行中（StoreKit IAP 内购走 Apple 通道）
- Android：等 Google Play Console 账号开通

适合移动场景：通勤刷题、线上考试模考、错题回顾。

### 微信小程序（考证匠会员专属）

会员开通后微信扫描小程序码。功能聚焦移动刷题，绑定主站账号。

### Chrome 浏览器插件

匠人维护两个公开 Chrome 插件 + 一个 alpha：

- **考证匠 AI Tutor**（截屏分析考题，公开）
- **求职匠 Job Hunter**（投简历自动填表 + JD 匹配，公开）
- **牛小匠 UniMate Tutor**（大学课程 AI 答疑，T2 学期 alpha 测试中）

Chrome Web Store 搜对应英文名 / 直接从 [/tools](https://jiangren.com.au/tools) 页面找到下载链接。

## 找不到要的？

匠人 1000+ 页面找东西可能费劲。三个推荐路径：

1. **顶部搜索框**：模糊搜任何关键词，全站搜索（含 blog / wiki / lab / roadmap）
2. **`/study-center`** 学习中心：按方向（AI / Web / Data / Cloud）筛选所有 lab + learn + roadmap
3. **`/learn`** 学习方向首页：21 个学习方向（ai-engineer / prompt-master / vibe-coding 等）
