## 怎么选（三类用户场景）

### 场景 A：刷 IT 认证考试（AWS / Azure / GCP / CKA）

**选 BASIC $8**。

考证匠完整题库 + AI 解析在 BASIC 就全开。一份云认证书拿下后年薪可能涨 $20K，$8/月 拿全题库是实实在在划算。

唯一升级理由：你想同步刷面试 + 用 AI 模拟面试，那升 PLUS。

### 场景 B：本地求职 / 转行 / 找实习

**优先 PLUS $19.90**。

PLUS 比 BASIC 多的核心是**求职匠 AI 面试模拟**——用 LLM 扮演 HR / 技术面试官跟你对话，给你反馈。每次模拟 ~30 分钟，等同于花 $19.90 买 mentor 1 小时（mentor 时薪平均 $80-150）。

如果你需要真人 mentor 1:1 review 简历 + 项目 + 面试 → PREMIUM。

### 场景 C：澳洲在校大学生（UQ / UNSW / USYD / Monash 等）

**选 BASIC $8 起步**，看牛小匠用得多再升级。

BASIC 包含牛小匠每月 5 次 AI 答疑，对一周 5 节课的留学生通常够了。但要是赶 deadline / 期末，PLUS 50 次比较稳。

## 不在会员里的内容（要单独买）

**Live Cohort Bootcamp** 不含 BASIC / PLUS。这是 4-12 周由真人讲师带的密集训练营，单独定价 $1500-$8000 不等。PREMIUM 会员每年送 2 次 cohort 名额——是给已经买 PREMIUM 的额外福利，不是替代。

**Cohort vs Self-paced 区别**：

- **Cohort**：固定开课日、每周 live session、有同学群、毕业前一对一 review。**适合需要外部 deadline 推动的人**
- **Self-paced**：随时学、看视频/做 lab、社区答疑。PLUS / PREMIUM 会员包含。**适合自驱型 + 时间灵活的人**

更详细 cohort vs self-paced + 三档主课（自学/教学/陪跑） 见第 6 章。
