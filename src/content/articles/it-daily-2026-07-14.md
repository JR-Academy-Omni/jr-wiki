## 一、认证动态

### 1. AZ-204 还剩 17 天 — 7 月 31 日退役，过了只能考 AI-200

**一句话**：微软 Azure Developer Associate 认证（考试代码 AZ-204）将于 **2026 年 7 月 31 日正式退役**，最后考试窗口只剩 17 天，之后唯一替代路径是全新的 AI-200（Azure AI Cloud Developer Associate）。

这次不是小改版，而是微软对 Azure 开发者认证的一次方向性重写。AZ-204 覆盖通用 Azure 开发（Web Apps、Functions、Cosmos DB、事件总线），AI-200 从零重建，把重心转向 RAG 架构、向量数据库、Azure AI Studio、容器化 AI 工作负载，约 60% 的考点延续自 AZ-204，剩下 40% 是全新内容。

**两条路怎么选**：

| 你的备考进度 | 建议 |
|------------|------|
| 已刷完 AZ-204 题库，差不多准备好了 | 立刻预约，7 月 31 前考完，拿证有效期 1 年，可用 Microsoft Learn 免费 assessment 续期 |
| 刚开始学习 | 直接转 AI-200，没必要在即将退役的内容上浪费时间 |
| AZ-204 已过，在备考 AZ-400（DevOps） | AZ-204 作为 AZ-400 先修资格，退役后**仍旧有效** 1 年（Microsoft Q&A 官方确认） |

**考试硬参数（AZ-204，截止 7 月 31 日）**：

- 考试费：**$165 USD**（澳洲约 $255 AUD）
- 通过分：700 / 1000
- 语言：英语 / 日语 / 中文等
- 预约：Pearson VUE（线上监考或考场均可）

AZ-204 成绩认定为 Microsoft Certified: Azure Developer Associate，证书颁发后有效期 1 年，届时可用 Microsoft Learn 上的免费 online assessment 续期，**不需要再考 AI-200**。

> 来源：[Will AZ-204 still be accepted as a prerequisite for AZ-400 after it retires? — Microsoft Q&A](https://learn.microsoft.com/en-us/answers/questions/5893548/will-az-204-still-be-accepted-as-a-prerequisite-fo) · [AZ-204 vs AI-200 — CertEmpire](https://certempire.com/az-204-vs-ai-200/)

### 2. AWS 认证免考试续期「Maintain」6 月 23 日上线 Open Beta

**一句话**：AWS 正式推出第三条续期路径——**Maintain**：在 AWS Skill Builder 上完成指定数字课程 + 实操 Lab，即可把认证有效期**免考试延长 1 年**，全程在家在线完成。

AWS 认证一直是 3 年有效，之前只有两条路：①重考原证，②通过更高级别的考试。Maintain 提供了第三选项，用学习行为替代考试，更新当年最新的服务知识，适合证书快到期但没时间集中备考的工程师。

**Maintain 的运作方式**：

- 登录 [AWS Skill Builder](https://skillbuilder.aws/)，导航至 Certification → Maintain（open beta）
- 系统生成一个专属学习计划，内容涵盖你持有认证的最新知识点
- 完成数字课程 + 至少 1 个实操 Lab（在真实 AWS 环境中执行任务，不是看视频）
- 满足所有要求后，认证有效期自动延伸 **1 年**（从完成日期起算）

**目前支持的认证（2026 年 Q3）**：
- AWS Certified CloudOps Engineer – Associate（同时可维护 SysOps Administrator – Associate）
- 年内将覆盖：Data Engineer – Associate、Security – Specialty、Machine Learning Engineer – Associate

注意：Maintain 给 1 年，重考给 3 年。如果你的目标是拿证超过 1 年再考，仍建议老老实实备考续期。

> 来源：[A new way to keep your AWS Certification current — AWS Training and Certification Blog](https://aws.amazon.com/blogs/training-and-certification/a-new-way-to-keep-your-aws-certification-current/)

### 3. Linux Foundation 七月折扣码 JULY26CCCT — CKA $445 → $289.25

**一句话**：Linux Foundation 七月折扣码 **JULY26CCCT** 可在所有 Kubernetes 认证上享受 **35% off**，CKA 从 $445 降到约 $289.25 USD；捆绑包代码 **JULY26BUNCT** 最高打到 55 折（CKA+CKAD+CKS 三证包）。折扣注明「ends soon」，不保证月底还在。

**各证当前定价（折前）**：

| 认证 | 代码 | 折前价 | 35% off 后 |
|------|------|--------|-----------|
| Certified Kubernetes Administrator | CKA | $445 | ~$289 |
| Certified Kubernetes Application Developer | CKAD | $445 | ~$289 |
| Certified Kubernetes Security Specialist | CKS | $445 | ~$289 |
| Kubernetes and Cloud Native Associate | KCNA | $250 | ~$163 |
| Kubernetes and Cloud Security Associate | KCSA | $250 | ~$163 |

**同步更新：CARE 程序（6 月 18 日起）**

从 2026 年 6 月 18 日起，通过或重新认证 CKS 考试，将**自动恢复或延伸已持有的 CKA 认证**有效期——即一张考试同时维护两张证书。对于 DevOps / Platform Engineer 来说，这大幅降低了维持双证的成本。

### 4. Microsoft AI-200 & SC-500 并排上线，7 月正式开放注册

**一句话**：配合 AZ-204 退役，微软同期推出两张全新认证，均在 **2026 年 7 月**开放正式注册。

- **AI-200 Azure AI Cloud Developer Associate**：替代 AZ-204，聚焦 RAG、向量数据库、Azure AI Studio、Semantic Kernel、容器化 AI 服务

## 二、免费 / 限免课程

### 1. Google AI Essentials — 免费+完课证书，无先修要求

**适合**：完全没有 AI 背景、想用 AI 工具提效的职场人士（也适合正在备考 Google 相关认证的学员热身）

**链接**：[skills.google / AI Essentials](https://www.skills.google/)

这门课不讲模型原理，直接教你把 AI 工具落地到日常任务：用 AI 生成和迭代内容创意、辅助研究分析、加速日常重复工作流。全程免费，无需支付任何费用，完成即颁发可分享的完课证书。

Google AI Essentials 目前是 Google Skills Boost 平台上访问量最高的入门内容，预计学习时长约 **5-10 小时**（含视频、练习、测验），可随时暂停自定义节奏。

### 2. Microsoft Learn「Everyday AI Fluency」— 2 小时，免费 LinkedIn 徽章

**适合**：使用 Microsoft 365 的办公室用户，以及备考 AI-900 / AI-200 前想快速热身的学员

**链接**：[Microsoft Learn — Everyday AI Fluency](https://learn.microsoft.com/en-us/ai/)

这条学习路径 2026 年 5 月悄悄上线，目前已有超过 **3,000,000 名用户**完成，是今年 Microsoft Learn 增长最快的学习路径。内容完全基于真实 Windows / Microsoft 365 场景：在 Word 里用 AI 摘要文档、在 Outlook 里草拟邮件回复、在 Teams 里实时转录会议重点。

完成后获得数字徽章，可直接添加到 LinkedIn 个人档案，对刚进职场或转岗的学员来说是个低门槛的可见信号。全程 **约 2 小时**，完全免费，无需订阅 Microsoft 365。

**适合**：正在搭建 AI Agent、对话系统、或想加入语音交互能力的开发者

DeepLearning.AI 短课采用「第一个模块永久免费」的模式，包含视频课程、阅读材料、编程作业。如果只是想了解核心概念，免费看第一模块完全够用；完整课程需要 Coursera 订阅。

### 4. Google Cloud Skills Boost — 300+ 免费 Badge，Gemini AI 系列上新

**适合**：备考 Google Cloud Professional 认证的工程师，或想积累 GCP 实操经验的求职者

**链接**：[cloud.google.com/learn/training](https://cloud.google.com/learn/training)

截至 2026 年 5 月，Google Cloud Skills Boost 平台上有超过 **300 门课程**提供免费完课徽章或技能徽章，AI/ML 目录是近两年增长最快的方向：

### 5. AWS Skill Builder Builder Labs 免费 Plan — 10 个实操 Lab 永久免费

**适合**：AWS 认证备考新手，或想体验真实 AWS 环境但不想花钱的学员

**链接**：[skillbuilder.aws](https://skillbuilder.aws/)
