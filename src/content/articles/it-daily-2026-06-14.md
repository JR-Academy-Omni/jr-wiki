## 一、认证动态

### 1. DP-100 Azure Data Scientist Associate 已于 6 月 1 日退役，AI-300 MLOps Engineer 接棒

**一句话**：数据科学建模认证正式谢幕，Azure 把重心转移到"把模型跑稳跑好"的 MLOps 工程师角色。

DP-100（Designing and Implementing a Data Science Solution on Azure）自 2018 年上线以来是 Azure 数据科学方向的核心认证，但微软在今年 6 月 1 日将其正式退役。如果你此前拿到了 DP-100，已有的 badge 不会消失，但无法再续期。

接棒考试是 AI-300：Operationalizing Machine Learning and Generative AI Solutions，对应认证名称为 **Microsoft Certified: Machine Learning Operations Engineer Associate**。与 DP-100 侧重数据探索、模型训练、评估和部署不同，AI-300 扩大了范围，重点考察：

- **CI/CD 流水线**（Azure DevOps / GitHub Actions 触发自动化训练和部署）
- **基础设施即代码**（Bicep / ARM 管理 Azure ML 工作区）
- **可观测性与漂移检测**（监控模型性能衰减，触发再训练）
- **成本治理与生命周期管理**

考试格式：40–60 题，150 分钟，过线 700/1000。Beta 版于 3 月 2026 发布，GA 状态请在[微软官网](https://learn.microsoft.com/en-us/credentials/certifications/azure-machine-learning-operations-engineer-associate/)确认最新安排。

> 来源：[Microsoft Tech Community — New Certification for MLOps Engineers](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-certification-for-machine-learning-operations-mlops-engineers/4494111)；[CBT Nuggets — Microsoft Is Retiring the DP-100](https://www.cbtnuggets.com/blog/certifications/microsoft/tech-news-microsoft-is-retiring-the-dp-100)

### 2. PL-500 Power Automate RPA Developer Associate 6 月 30 日退役，官方无直接替代

**一句话**：传统 RPA 桌面流（Desktop Flow）认证走入历史，Power Platform 自动化方向正在向 AI Agent 切换。

PL-500 将于 **2026 年 6 月 30 日 23:59 CST** 正式退役，届时将无法报名和通过此考试获得认证，已持有者也无法续期。

与此前退役的多个 Microsoft 认证不同，PL-500 **没有官方直接替代考试**。微软推荐的方向是：

- **AB-620 AI Agent Builder Associate**：最接近 PL-500 的新认证，专注于用 Power Automate 和 Microsoft Copilot Studio 构建 AI Agent 和自动化工作流
- **AB-100 Agentic AI Business Solutions Architect Expert**：专家级，适合有 Power Platform 解决方案架构背景的人

如果你还没考 PL-500，6 月 30 日是最后窗口。考了之后——这张认证虽然留在履历上，但 3 年有效期到后便无法续。实际上建议直接把备考精力转向 AB-620。

> 来源：[CertEmpire — PL-500 Retiring June 2026](https://certempire.com/pl-500-retiring/)；[CertQueen — MB-280, PL-200, PL-500 and More Microsoft Exams Will Retire Soon in 2026](https://www.certqueen.com/news_MB-280_PL-200_PL-500_and_More_Microsoft_Exams_Will_Retire_Soon_in_2026_5372.html)

### 3. AZ-802 Windows Server Hybrid Administrator Beta 本月开放：AZ-800 + AZ-801 二合一

**一句话**：微软把 Windows Server 混合管理的两门考试合并成一门，备考路线从"两考"变"一考"。

AZ-802（Microsoft Certified: Windows Server Hybrid Administrator Expert）Beta 考试于 **2026 年 6 月**开放，正式 GA 预计 **8 月 2026**。

新考试整合了：
- **AZ-800**（Administering Windows Server Hybrid Core Infrastructure）
- **AZ-801**（Configuring Windows Server Hybrid Advanced Services）

换言之，原来需要通过两门考试才能拿到 Expert 认证，AZ-802 之后只需一门。如果你已持有 AZ-800 或 AZ-801，不需要重考——可以通过 AZ-802 续期评估完成认证维护。

Beta 期间通常折扣报名，目前具体价格和报名链接请到 [Microsoft Learn Certifications](https://learn.microsoft.com/en-us/credentials/certifications/) 搜索 AZ-802 查看。

> 来源：[Tutorials Dojo — AZ-802 Windows Server Hybrid Administrator Study Guide](https://tutorialsdojo.com/az-802-windows-server-hybrid-administrator-study-guide/)；[CertEmpire — AZ-800 vs AZ-802](https://certempire.com/az-800-vs-az-802/)

### 4. Microsoft AI Skills Fest 2026：活动已结束，但考试券兑换至 8 月 18 日仍有效

**一句话**：6 月 8–12 的活动已关闭，但如果你已在期间完成学习路径并提交了申领表，考试券还有 2 个月可以用。

Microsoft AI Skills Fest 2026 于 6 月 8–12 日在 AI Skills Navigator 平台上举行，参与者完成至少一个精选 AI 学习路径（Playlist）后提交申领表，可获得 **100% 折扣的 Microsoft 或 GitHub 认证考试券**（Certiport 和 MOS 考试除外）。

关键时间节点（均为 UTC 00:00）：
- 申领表提交截止：**6 月 12 日**（已过）
- 考试券兑换截止：**2026 年 8 月 18 日**
- 考试必须完成于：**2026 年 10 月 18 日**

如果你在活动期间参与了，现在该做的是：尽快登录 Pearson VUE 预约考试，用 100% 折扣码在 8 月 18 日前锁定考试时间。考试本身可在 10 月 18 日前安排，时间还充裕。

## 二、免费 / 限免课程

### 1. AWS Builder Labs 免费实操学习计划：10 个 Lab 覆盖 EC2 / S3 / Lambda

**适合**：AWS 新手、备考 CLF-C02（Cloud Practitioner）或 SAA-C03（Solutions Architect Associate）的同学
**链接**：[AWS Skill Builder — Builder Labs 免费学习计划](https://aws.amazon.com/blogs/training-and-certification/begin-your-aws-journey-with-new-free-aws-builder-labs-learning-plan-on-aws-skill-builder/)

AWS 在 Skill Builder 平台上架了新的 **Introduction to AWS Cloud** 免费 Builder Labs 学习计划，包含 **10 个实操 Lab 环境**，涵盖 Amazon EC2、S3、Lambda 等核心服务，**无需付费订阅**即可访问。

对备考 CLF-C02 的同学来说，刷完这 10 个 Lab 能给面试和考试添加真实操作记录，比纯刷题更有说服力。每个 Lab 在沙盒环境里运行，不会产生真实 AWS 账单。

**适合**：备考 GCP ACE 标准版或计划续期（$75）的同学
**链接**：[Cloud Engineer Learning Path](https://www.cloudskillsboost.google/paths/11)

GCP ACE 换版倒计时 16 天，Cloud Engineer Learning Path 覆盖 ACE 全部考点（VPC 网络、GKE 管理、Cloud Storage、IAM、计费等）。通过 **GEAR 计划每月 35 免费学分**可覆盖路径上大部分 Lab，无需购买 Skill Boost 订阅。

续期考试（$75，1 小时，20 题）也在 6 月 30 日同步换版——已持有 ACE 且在续期窗口内的同学，现在也要对照新 PDF 准备。

GEAR 计划报名：[Google GEAR Program](https://cloud.google.com/resources/boost-your-cloud-skills-with-google)

## 三、JR Academy 学员行动建议

1. **已持有 DP-100 的同学**：认证 badge 不消失，但无法续期。趁 AI-300 刚 GA（或仍在 Beta 窗口），现在开始备考是切换成本最低的时机，Beta 版通过率通常高于正式版。

2. **还在备考 PL-500 的同学**：6 月 30 日是最后关口，有意向的现在就预约 Pearson VUE，月底前考掉。但长远来看，备考精力建议同步转向 AB-620 AI Agent Builder，那才是 Power Platform 自动化的下一站。

3. **GCP ACE 备考中**：6 月 30 日前按旧考纲准备，下载上方 PDF 仅做参考；30 日之后才开考的，直接切新考纲。续期考试（$75）比重考（$125）划算，持有 ACE 的人优先用续期通道。

4. **参加过 AI Skills Fest 的同学**：现在登录 Pearson VUE，用领到的 100% 折扣码预约考试，考试必须在 10 月 18 日前完成，时间宽裕但不要拖。

5. **AWS 方向新手**：先进 AWS Skill Builder 免费把 10 个 Builder Labs 刷完，边做边记录操作截图，以后面试直接有实操证据。
