## 1. Cisco CCIE 实验考 5 月起强制 AI 辅助排障模块

Cisco 于 2026 年 4 月底宣布，从 **5 月 1 日起** 所有 CCIE 实验考（Enterprise Infrastructure 与 Security 方向）全面切换新格式，核心变化是在原有 8 小时实操环节中加入**强制 AI 辅助故障排查（AI-assisted troubleshooting）模块**。

旧格式下候选人完全依赖手动排障；新格式要求候选人演示如何借助 AI 工具（Cisco AI Assistant、模型驱动遥测）定位根因，并在给定提示下选择正确修复路径。Cisco 官方称此调整意在验证"人机协同运维"能力，而非单纯考察命令行熟练度。

此外 CCNP Enterprise 核心考 **350-401 ENCOR** 也更新了考纲，新增：Cisco Catalyst Center 自动化工作流、多云环境 SD-WAN 策略、6 GHz 频段无线管理三大模块。已报名旧格式场次（2026-04-30 前）仍按旧考纲计分，5 月及以后的场次均适用新规。

**备考建议**：原计划 Q2 参考 CCIE Lab 的候选人需额外补充 Cisco AI Assistant 操作实验（Cisco dCloud 有免费沙盒），建议在 Pearson VUE 重新核实预约场次的考纲版本。

来源：[Cisco Certification Exam Updates 2026 — SPOTO](https://cciedump.spoto.net/news/cisco-certification-exam-updates-2026-new-ccnp-and-ccie-changes-take-effect-in-the-us-20260429093507.html)

## 2. Linux Foundation Mega May（5 月 12–20 日）：DevOps、云开发 Bundle 大幅折扣

Linux Foundation 今年 5 月再度推出 **Mega May** 促销，窗口期为 **2026-05-12 至 2026-05-20**，即今天起还有 7 天。此次打包优惠涵盖三条主线：

- **DevOps & GitOps 路径**：含 LFS169（GitOps Fundamentals）、LFS267（GitOps at Scale）及对应的 GitOps 证书考试券
- **Cloud Career 路径**：含 LFCS（Linux Foundation Certified System Administrator）+ CKA（Certified Kubernetes Administrator）bundle
- **Cloud Developer Journey**：CKAD（Certified Kubernetes Application Developer）+ KCNA（Kubernetes and Cloud Native Associate）组合

历年 Mega May 折扣幅度通常在 30–40%，具体价格以结账页为准。CKA + CKAD Bundle 往年正价约 $595，促销期一般降至 $395 左右。

**注意**：购买后 12 个月内激活考试券，不是立即使用。考试券可自行选日期预约 PSI Online 远程监考。

来源：[Linux Foundation Mega May 2026](https://training.linuxfoundation.org/mega-may-2026/)

## 3. Microsoft AZ-500 8 月 31 日退役，SC-500 新证接棒

Microsoft 已正式公告 **Microsoft Certified: Azure Security Engineer Associate（AZ-500）** 将于 **2026 年 8 月 31 日**退役，同日停止认证续期。替代证书为即将推出的 **Cloud and AI Security Engineer Associate（SC-500）**。

变化背景：AZ-500 诞生于 2019 年，主要聚焦 Azure 原生安全服务（Defender for Cloud、Sentinel、Key Vault）；新的 SC-500 在此基础上扩大范围，加入多云安全策略（含 Microsoft Defender for DevOps、Entra Permissions Management）以及 AI 工作负载安全（Responsible AI 合规检查、Copilot Studio 权限管控）。

**时间线**：
- 现在：仍可备考 AZ-500，通过后有效期 1 年
- 2026-08-31：AZ-500 退役，已持证者徽章保留但无法续期
- SC-500 预计 2026 年 Q3 正式开放报名（官方未公布具体日期）

已计划备考 AZ-500 的候选人若能在 8 月前通过，仍有意义；若时间紧张，建议等 SC-500 考纲发布后直接转备新证。

来源：[Microsoft Certification Guide 2026](https://www.lived365.com/post/microsoft-certification-guide-2026-newexams-retirements-aiskills-and-careerimpact)，[Microsoft Learn Azure Security Engineer](https://learn.microsoft.com/en-us/credentials/certifications/azure-security-engineer/)

## 4. AWS Generative AI Developer Professional 更新：加入 Bedrock AgentCore 考点

AWS 已对 **AWS Certified Generative AI Developer – Professional** 标准版考纲完成刷新，主要变化是加入 **Amazon Bedrock AgentCore** 相关服务的验证域。

Bedrock AgentCore 于 2026 年初 GA，提供托管 Agent 运行时、内存管理（Memory）、工具（Tool）调度等能力，让开发者无需自行搭 LangChain/CrewAI 架构也能部署生产级 Agent。新考纲在 **Responsible AI 与 Multi-agent Orchestration** 两个域中新增了对 AgentCore 配置和权限模型的考察。

原有考点（Foundation Model 选型、RAG 架构、Guardrails、Bedrock Knowledge Bases）保持不变。

**备考资源**：AWS Skill Builder 上的 "Building Generative AI Applications on Amazon Bedrock" 课程已同步更新，免费层可访问。此外 AWS 官方 Exam Guide 新增了"Revisions"区块，直接标注哪些服务是新加的考点，备考效率更高。

来源：[AWS Certification Coming Soon](https://aws.amazon.com/certification/coming-soon/)，[AWS Training Blog March 2026](https://aws.amazon.com/blogs/training-and-certification/march-2026-new-offerings/)

## 5. Google Cloud 考纲将对标 Cloud Next '26：Gemini Enterprise Agent Platform 等新品入题

Google Cloud 官方宣布，所有现有认证考纲将在近期更新，以覆盖 **Google Cloud Next '26** 上发布的新产品和服务，具体包括：

- **Gemini Enterprise Agent Platform**（企业级 Agent 托管运行时）
- **BigQuery AI Studio**（数据分析与 AI 工作负载一体化）
- **Cloud Run v2 Sidecar 增强**（多容器 sidecar 生产就绪）

目前受影响最大的是 **Professional Data Engineer（PDE）** 和 **Professional Cloud Architect（PCA）** 两项考试，Gemini 相关服务和 AI 数据栈在这两个认证的权重将明显提升。具体考纲变更日期以各考试 Exam Guide 的 "Revision History" 为准，Google Cloud 官网均已可查阅。

费用提醒：Google Cloud 认证考试统一定价 $200，报名走 Webassessor 平台，部分 Qwiklabs Learning Path 完成后可领取折扣码（通常 $50 off）。

来源：[Google Cloud Certifications](https://cloud.google.com/learn/certification)，[Google Cloud Skills Boost](https://www.cloudskillsboost.google/)
