## 一、认证动态

### 1. AZ-802 Beta 开放：一张考试取代两张，折扣码 AZ802Chandler 省 80%，7 月 22 日截止

**一句话**：AZ-802 单考替代原来 AZ-800 + AZ-801 双考组合，Beta 阶段前 300 名用码 AZ802Chandler 可省 80%，报名截止 7 月 22 日，留给你的时间只剩 10 天。

Microsoft 于 7 月 9 日正式宣布推出 AZ-802 Beta，同步将认证更名——原来的 **Microsoft Certified: Windows Server Hybrid Administrator** 改名为 **Microsoft Certified: Windows Server Administrator Associate**，并将考试路径从两科（AZ-800 + AZ-801）压缩为一科（AZ-802）。Microsoft Learn 官方 X 账号原文：*"One exam. Real-world skills. 80% off (beta, first 300 only). Ends July [22]"*。

**Beta 折扣怎么用**：

- 报名考试时输入代码 **AZ802Chandler**，结账时自动减 80%（原价约 $165 USD，打折后约 $33）
- 名额仅限前 300 名，先到先得
- 必须在 **2026 年 7 月 22 日当天或之前完成考试**（不是仅仅注册）
- 折扣不适用于土耳其、巴基斯坦、印度、中国大陆

**AZ-802 考什么变了**：

原有 AZ-800 覆盖的 Windows Server 基础管理（AD DS、DHCP、DNS、Group Policy）和 AZ-801 的混合高级服务（迁移、灾难恢复、监控）都保留，但新增了一批现代混合管理内容：
- **Azure Arc**：统一管理本地 + 云上 Windows Server
- **Azure Update Manager**：取代传统 WSUS，云端统一补丁管理
- **OSConfig 安全基线**：系统配置合规自动化
- **SMB over QUIC**：不经 VPN 的安全文件共享协议
- **GPU 分区（GPU Partitioning）**：虚拟化 GPU 资源分配

AZ-800 正式退役时间：**2026 年 9 月 30 日 17:00 CST**。如果你已经持有 AZ-800 并打算顺着老路考 AZ-801，建议直接转 AZ-802，一次搞定，还能拿 Beta 折扣。

> 来源：[Microsoft Learn 官方 X 帖子](https://x.com/MicrosoftLearn/status/2075203314610794622) · [AZ Feeds 报道（2026-07-09）](https://azurefeeds.com/2026/07/09/new-exam-for-microsoft-certified-windows-server-administrator-associate-certification/)

### 2. AZ-204 七月底退役：Azure 开发者认证 19 天后停考，换 AI-200

**一句话**：AZ-204（Developing Solutions for Microsoft Azure）将于 **2026 年 7 月 31 日**永久退役，替代认证 AI-200（Azure AI Cloud Developer Associate）已开放报名，考纲从通用 Azure 开发转向 AI 驱动应用构建。

AZ-204 自 2019 年上线以来一直是 Azure 开发者的标配认证，考的是 App Service、Cosmos DB、Azure Functions、Key Vault、Event Hub 这些传统开发域。AI-200 全面改变了考察方向：

**AI-200 核心考纲模块（目前已公开的草稿）**：
- **Azure OpenAI Service**：Prompt 工程、模型微调、Safety guardrails
- **检索增强生成（RAG）**：Azure AI Search + 向量索引构建
- **AI Foundry**：Azure 统一 AI 开发平台的项目管理和 API 集成
- **向量数据库**：在 Azure Cosmos DB for NoSQL / PostgreSQL 中存储和检索 embeddings
- **Serverless AI 函数**：用 Azure Functions 封装 AI 推理管道
- **容器化 AI 工作负载**：AKS 上部署 AI 微服务

**现在怎么办**：

- **已认真备考 AZ-204 且能在 7 月 31 日前上考场**：直接去考，通过即拿 Azure Developer Associate 认证（有效期 2 年，到期可续），不会因为 AZ-204 停考而失效。
- **才刚开始备考 / 还差两三个月**：跳过 AZ-204，直接转 AI-200。AI-200 的考纲与市场需求更吻合，2026 年下半年 Azure AI 开发岗位需求持续上涨，这张证的含金量会更高。
- **持有老 AZ-203**（更早的前身）：不受影响，但建议逐步向 AI-200 路径过渡。

> 来源：[Praveen Kumar Sreeram 的技术博客（2026-06-02）](https://praveenkumarsreeram.com/2026/06/02/microsoft-az-204-certification-is-being-replaced-by-microsoft-ai-200-certification/) · [DEV Community 分析文](https://dev.to/lovelacecoding/bye-az-204-hello-ai-200-what-you-need-to-know-about-the-new-azure-developer-certification-2lo)

### 3. Google Cloud 四项认证推出"无需重考"续期方案

**一句话**：Google Cloud 于 7 月 9 日上线简化续期机制——ACE、PCA、PDE、CDL 持证者只需完成指定课程或 Skill Badge，认证自动延期 1 年，不需要再坐一次完整考试。

官方原文（来自 cloud.google.com 博客）：*"Once you've completed the required activities while your certifications are active, your certification will automatically be extended by one year."*

**4 项受益认证**：
1. **Cloud Digital Leader（CDL）** — 入门级云素养认证
2. **Associate Cloud Engineer（ACE）** — GCP 日常运维工程师
3. **Professional Cloud Architect（PCA）** — 架构设计专业认证
4. **Professional Data Engineer（PDE）** — 数据工程专业认证

**两种续期路径（任选其一）**：
- **Courses（课程）**：完成 Google 指定的更新主题课程，时间灵活，边工作边学
- **Skill Badges（技能徽章）**：完成交互式实操 Lab，已经在用新技术的人可以更快验证

**注意条件**：必须在认证**有效期内**完成活动，过期认证不适用。

这对在职 Cloud Engineer 意义明确：以前每两年要重新约考、刷题、付考试费，现在理论上可以通过持续学习实现"永续有效"。具体哪些课程 / Skill Badge 对应哪项认证，见 [support.google.com/cloud-certification](https://support.google.com/cloud-certification/answer/9907853)。

> 来源：[Google Cloud 官方博客（2026-07-09）](https://cloud.google.com/blog/topics/training-certifications/new-ways-keep-google-cloud-certifications-current)

## 二、免费 / 限免课程

### 1. Microsoft Everyday AI Fluency — 免费，已 300 万人完成

**适合**：不懂技术但每天要和 AI 打交道的职场人；想了解 AI Agent 安全边界的 IT 运维、开发者。
**链接**：[learn.microsoft.com/en-us/ai/](https://learn.microsoft.com/en-us/ai/)

这门课 2026 年 5 月悄然上线，一个月内已有超过 300 万人完成，核心内容：

- **AI Agent 安全性**：如何识别和限制 AI Agent 权限过大的风险
- **高级 Prompt 技巧**：上下文控制、角色设定、迭代精炼实操
- **负责任 AI 伦理**：微软内部 AI 使用准则的精简版
- **AI 办公场景**：Copilot for Microsoft 365 的实际工作流整合

全程自学，浏览器即可，登录 Microsoft 账号（免费注册）即可开始，没有证书但有完成徽章。对准备理解 AI-200 考纲中"Responsible AI"模块的学员，这是最快的 2 小时补课路径。

### 2. AWS Skill Builder — 10 个免费 Lab，EC2 / S3 / Lambda 实操

**适合**：零基础想上手 AWS 的学员；备考 AWS Cloud Practitioner（CLF-C02）的初学者。
**链接**：[skillbuilder.aws](https://skillbuilder.aws/)

AWS 新增了 **"Introduction to AWS Cloud"免费 Lab 学习计划**，10 个免费动手实验室，无需信用卡，包含：

- 在 EC2 上启动第一台虚拟机
- S3 存储桶创建、权限配置、静态网站托管

- IAM 用户与权限策略基础配置
- VPC 子网与安全组入门

每个 Lab 时长约 30-60 分钟，在浏览器里直接操作真实 AWS 环境（不是模拟器），实验结束后资源自动销毁。Skill Builder 免费账号同时可访问 600+ 课程视频，备考 CLF-C02 的学员可直接用免费 Lab 替代买 Udemy 实操包。

目前 Coursera 平台有 270+ 课程完全免费（无需 Financial Aid，无证书），7,000+ 课程可通过 Financial Aid 拿到免费证书。IT 方向最值得申请的：Google Data Analytics、IBM Data Science、Meta Front-End Developer。
