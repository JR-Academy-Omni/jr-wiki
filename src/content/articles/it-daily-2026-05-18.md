## 一、认证动态

### 1. DP-750 Azure Databricks Data Engineer Associate 本月 GA

**一句话**：微软新推的 Azure Databricks 认证（DP-750）已完成 Beta，本月正式 GA，是目前市面上唯一同时覆盖 Databricks 引擎 + Azure 服务集成的 Associate 级认证。

2026 年 3 月，微软发布了 DP-750 Beta 版考试，面向在 Azure Databricks 上构建数据工程解决方案的从业者。Beta 阶段前 300 名报名者可使用折扣码 **DP750Deltona** 享 80% 折扣（约 $33 USD），该优惠于 4 月 2 日截止。目前考试已进入正式版（General Availability），标准考试费约 **$165 USD**，与 Azure Data Scientist Associate、Azure Database Administrator Associate 等其他 Associate 认证定价一致。

DP-750 的核心考察内容围绕 Azure Databricks 实操四大域：

- **Delta Lake 数据管道**：表格式设计、增量读写、MERGE 操作
- **Spark 任务调度与优化**：Cluster 配置、Job 调度、查询优化
- **MLflow 模型追踪**：Experiment 管理、模型注册、部署集成
- **Azure 服务集成**：与 Azure Storage、Azure Data Factory、Azure Key Vault、Azure Monitor、Microsoft Entra ID 打通

这与昨日报道的 AI-300（MLOps Engineer）定位明确不同——DP-750 是**数据工程（Data Engineering）**方向，重点在数据流水线和 Spark 平台，而非模型上线运营。已持有 Databricks Certified Data Engineer Associate（原生 Databricks 证书）的考生，补考 DP-750 可获得微软生态体系的认可，两张证书搭配在澳洲数据工程岗招聘中组合竞争力更强。

> 来源：[Microsoft Certified: Azure Databricks Data Engineer Associate](https://learn.microsoft.com/en-us/credentials/certifications/implementing-data-engineering-solutions-using-azure-databricks/) · [Medium 认证详解](https://nidhig631.medium.com/new-microsoft-certification-just-dropped-azure-databricks-for-data-engineers-exam-dp-750-99a5427c2685)

### 2. CNCF CARE 计划：CKA/CKS/CKAD 续期从 6 月起大重构

**一句话**：Linux Foundation 的 CARE（Certification Advancement & Recertification Experience）计划 6 月全面落地，Kubernetes 系列认证的两年续期将从强制重考改为更灵活的升级机制。

CNCF 在 **KubeCon + CloudNativeCon Europe** 大会上宣布了 CARE 计划，直接回应社区多年来对 CKA/CKS/CKAD 续期体验的抱怨。传统模式下，这三张证书有效期均为 **2 年**，到期必须全额重考——CKA $395 USD、CKS $395 USD——对已在生产岗的工程师来说，时间和钱的双重成本都不低。

CARE 的改变方向：续期路径将不再强制全额重考，而是引入可通过学习模块或部分实操测试来完成的"认证升级"机制。正式细则将于 **2026 年 6 月** 公布并实施，CNCF 表示追溯自 **2026 年 1 月 1 日** 起生效（即今年之内拿到的证书都将适用新续期条款）。

对正在备考 CKA 的学员，明确建议是：**不要因为等 CARE 细则而推迟报名**。CARE 影响的是已有证书的续期流程，不改变初次考试内容。当前 CKA 考试基于 Kubernetes 1.31/1.32，题型已稳定（etcd 备份恢复、NetworkPolicy、Gateway API、RBAC、节点排障），现在通过反而能赶在 CARE 正式实施前拿到一张"有利续期条款"适用的资质。

> 来源：[Linux Foundation CKA Program Changes](https://training.linuxfoundation.org/certified-kubernetes-administrator-cka-program-changes/) · [Cloud Native Now CNCF 报道](https://cloudnativenow.com/features/cncf-revamps-certification-program-to-simplify-renewals/)

### 3. Google Cloud 完成 Pearson 切换，Cloud Next '26 后题库即将更新

**一句话**：Google Cloud 认证从 2026 年 2 月 26 日起改用 Pearson VUE（全球 5,000+ 考点），并官宣将在 Cloud Next '26 后同步更新 Professional 认证考题，现在是考旧版内容的窗口期。

Google Cloud 已完成向 **Pearson VUE** 的考试平台迁移。相比原有安排，Pearson 网络的优势在于：全球 180+ 国家、5,000+ 实体考场，悉尼/墨尔本/布里斯班均有多个 Pearson VUE 认证中心，也支持 OnVUE 在家监考模式。第一批 Pearson 考位从 **3 月 2 日** 起可预约，正常注册入口不变（cloud.google.com/certification）。

更关键的是**题库更新预告**：Google Cloud Next '26 发布了 Gemini Enterprise Agent Platform、BigQuery AI Studio 等新产品。Google 已确认认证考题将在近期相应更新。受影响较大的认证：

- **Professional Cloud Architect**：Gemini 辅助架构决策内容将新增
- **Professional Data Engineer**：BigQuery AI Studio 和 Vertex AI 新功能
- **Professional Cloud Security Engineer**：Confidential Computing 和零信任内容权重增加（本月已有题库小幅更新）

实操建议：如果你的备考资料已经准备好，这周安排考位比等题库更新后再考节省大量二次刷题时间。Google Cloud 认证费用为 **$200 USD/门**（不分级别）。

> 来源：[Google Cloud 平台切换公告](https://support.google.com/cloud-certification/answer/16803278?hl=en) · [Google Cloud Certifications 官网](https://cloud.google.com/learn/certification)

### 4. AWS AI & ML Scholars 挑战赛 6 月 24 日截止，名额已满但资源仍可用

**一句话**：AWS 与 Udacity 合作的 10 万名额 AI & ML Scholars 挑战赛名额已满，完成者可得 3 个月 Skill Builder 免费订阅，没申请到的走 AWS Skill Builder 免费版替代。

这是 AWS 2026 年最大规模的免费 AI 培训项目：**100,000 个免费名额**，两个阶段——

1. **Challenge Phase**（3 月 24 日 ~ 6 月 24 日）：在 Udacity 平台完成约 15 小时 AI 基础课程，并用 **Amazon PartyRock** 构建一个生成式 AI 原型应用

### 3. Microsoft Learn：DP-750 配套免费学习路径

**适合**：正在备考 Azure Databricks Data Engineer Associate 的数据工程师
**链接**：[https://learn.microsoft.com/en-us/training/](https://learn.microsoft.com/en-us/training/)

随着 DP-750 正式 GA，Microsoft Learn 同步上线了配套免费学习路径，覆盖考试四大域：

- **Delta Lake 与 Databricks SQL**（约 4 小时）：表格式设计、增量写入、MERGE/OPTIMIZE 操作
- **Spark 数据处理与优化**（约 5 小时）：Cluster 选型、Adaptive Query Execution、数据倾斜处理
- **MLflow 实验追踪与模型管理**（约 3 小时）：Experiment 结构、模型注册、Serving endpoint
- **Azure 服务集成**（约 3 小时）：Data Factory pipeline 触发 Databricks Job、Key Vault 密钥注入

全部免费，完成各模块后可获 Microsoft Learn badge 积分，对 Microsoft Credential 的 renewal 有帮助。不需要提前购买考试，学完再决定报名时间。

**适合**：想用开源 LLM 替代商业 API 调用的开发者和数据科学家
