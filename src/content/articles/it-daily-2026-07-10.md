## 一、认证动态

### 1. AZ-204 七月三十一日退役：别再拖了，要么本周拿下、要么直接准备 AI-200

**一句话**：AZ-204（Azure Developer Associate）距离退役仅剩 21 天，7月31日之后无法报名和参考，直接接棒的是 AI-200（Azure AI Cloud Developer Associate）。

AZ-204 自 2020 年上线以来一直是 Azure 开发者的主流认证，2026 年 7 月 31 日正式退出舞台。微软推出 AI-200 作为直接替换，考试名称从 "Azure Developer Associate" 改为 "Azure AI Cloud Developer Associate"——不是改名，是整条学习路径转向 AI 工程化开发。

**两者差异值得弄清楚：**

- 内容迁移：约 **60%** 的 AZ-204 考核内容在 AI-200 里保留，但核心新增考点是 Azure OpenAI、向量数据库、AI Foundry 和 Serverless AI 函数
- 考核语言：AZ-204 同时覆盖 C# 和 Python；AI-200 官方学习指南仅列出 Python——转考前要做好语言切换准备
- 难度预估：AI-200 对 AI 架构（RAG pipeline、embedding、模型微调流程）有更强的工程要求，不是死背文档就能过

**现在怎么办？**

| 情况 | 建议 |
|------|------|
| 已备考 AZ-204 两周以上、可在 7/31 前赶考 | 立即在 Pearson VUE 订位，先拿证再说 |
| 刚开始备考、资料还没系统过 | 跳过 AZ-204，直接学 AI-200，省时间 |
| 已持有 AZ-204（已过） | 认证在有效期内仍然有效，不受退役影响；续期时按新政策续 |

AZ-204 的另一个常见疑问：AZ-400（DevOps Engineer Expert）以 AZ-204 为前置，退役后会怎样？微软官方已在 Q&A 里确认，**退役前拿到的 AZ-204 继续作为 AZ-400 的有效前置凭证**。

来源：[Microsoft Q&A — AZ-204 prerequisite for AZ-400 after retirement](https://learn.microsoft.com/en-us/answers/questions/5893548/will-az-204-still-be-accepted-as-a-prerequisite-fo) · [DEV Community — Bye AZ-204, hello AI-200](https://dev.to/lovelacecoding/bye-az-204-hello-ai-200-what-you-need-to-know-about-the-new-azure-developer-certification-2lo)

### 2. AWS Certified Generative AI Developer – Professional：Beta 结束，GA 正式开放报名

**一句话**：AWS Certified Generative AI Developer – Professional 考试已从 Beta 阶段转为 GA，任何地区的考生现在都可以直接报名正式考试。

这个认证从 2026 年初以 Beta 形式面向受邀考生开放，经过几个月的题库优化后，目前已正式转为 General Availability。考试定位是 Professional 级别，验证考生在以下方向的工程落地能力：

- **Foundation model 选型与调优**：知道哪个 Bedrock 模型适合哪种业务场景，以及 fine-tuning vs. RAG vs. prompt engineering 的取舍判断
- **Agentic AI 系统架构**：Multi-agent workflow 的设计、Amazon Bedrock Agents 的配置与监控
- **Responsible AI**：Bedrock Guardrails 配置、输入输出过滤、幻觉控制策略
- **AWS 生态集成**：SageMaker Pipelines、Amazon Q、Kendra 和 Bedrock 的组合使用

区别于入门级的 **AWS Certified AI Practitioner（AIF-C01）**，GenAI Developer – Professional 要求考生实际上手构建过 GenAI 应用——只懂概念、没动过代码是很难通过的。

报名和考试指南：[AWS Certified Generative AI Developer – Professional](https://aws.amazon.com/certification/certified-generative-ai-developer-professional/)

### 3. AWS 四张 Microcredentials 全部免费：Serverless / Agentic AI / Application Networking / Incident Response

**一句话**：AWS 宣布将旗下所有 Microcredentials 变更为免费，无需 AWS Skill Builder 付费订阅，任何人均可直接参加。

这是今年 AWS 认证体系里动作最大的免费化举措。目前已发布的四张 Microcredentials 全部开放免费参考：

| Microcredential | 考核重点 |
|----------------|---------|
| **AWS Serverless Demonstrated** | AWS Lambda、API Gateway、Step Functions、DynamoDB 的配置与排错 |
| **AWS Agentic AI Demonstrated** | 用 Amazon Bedrock 构建 AI Agent 的集成、修复与优化 |
| **AWS Application Networking Demonstrated** | 应用交付、性能优化和现代应用架构的核心操作 |
| **AWS Incident Response Demonstrated** | AWS 安全事件的识别、隔离与修复流程 |

**和普通认证考试的区别：** Microcredentials 不是多项选择题——考生被放进模拟业务场景，直接在 **真实 AWS 环境**里操作（真正的 Lambda 函数、真正的 Bedrock Agent、真正的 IAM 配置），证明自己能干活，而不是背得住答案。

这个格式对于简历上想证明动手能力的候选人来说是加分点，因为用人单位可以直接点链接核验，且徽章显示的是"做出来了"而不是"背过了"。

更多内容和 Microcredentials 详情：[builder.aws.com — AWS Application Networking Demonstrated & Incident Response Demonstrated](https://builder.aws.com/content/3B5Y4mIAdKNnm9r8MZpPiKU4lvC/check-out-new-microcredentials-aws-application-networking-demonstrated-and-aws-incident-response-demonstrated)

### 4. AZ-900 英文版 7月20日更新：备考资料要换

**一句话**：Microsoft Certified: Azure Fundamentals（AZ-900）英文版考试于 **7 月 20 日**更新题库，正在备考的同学要确认用的是最新版官方 Study Guide。

AZ-900 是最高频次的入门认证，几乎每年都有小幅更新。这次 7 月 20 日的更新针对的是英文版，覆盖生成式 AI 与 Azure AI 服务相关的新增考点——与微软整体把 AI 内容推进各级别认证的节奏一致。

如果你正在学的 AZ-900 备考材料是半年前的版本，建议核查一下官方学习路径是否有更新：[Microsoft Learn — AZ-900 Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/az-900)

## 二、免费 / 限免课程

### 1. DeepLearning.AI「Voice for AI Agents and Applications」——免费新课，适合 AI 开发者

**适合**：已有基础 AI 应用开发经验（知道什么是 API、用过 LLM）、想给 Agent 加上语音交互的工程师  
**链接**：[deeplearning.ai/courses](https://www.deeplearning.ai/courses)

Andrew Ng 的 DeepLearning.AI 最近上线了「Voice for AI Agents and Applications」，课程使用 **Google's Agent Development Kit（ADK）** 构建实时语音 AI Agent。

课程覆盖：
- 语音流的实时捕获与处理
- 将 Speech-to-Text / Text-to-Speech 集成进 Agent 工作流
- 用 Google ADK 搭建端到端语音 Agent pipeline
- 常见延迟问题的排查思路

DeepLearning.AI 的 "Short Course" 系列一贯免费、时长 1-3 小时，这门课适合周末下午花两小时跑完 notebook 的节奏。没有结业证书，有可运行的代码。

**适合**：对 GenAI 感兴趣但不知道从哪开始、或者想用更好玩的方式复习 AWS AI 服务的学员  
**链接**：[AWS Skill Builder](https://skillbuilder.aws/generative-ai)

AWS Cloud Quest 是一个 RPG 风格的角色扮演学习游戏——你扮演一个云架构师，在虚拟城市里接任务、解问题，每完成一关就要在真实 AWS 环境里操作一次。**Generative AI Practitioner 版本完全免费**，通关后可拿到一枚可分享的数字徽章。

适合 AIF-C01（AWS AI Practitioner）的备考热身，也适合刚开始了解 Bedrock、Amazon Q、SageMaker 的新人。时间成本大约 **3-5 小时**，不需要任何预备知识。

## 三、JR Academy 学员行动建议

1. **AZ-204 持有者 / 正备考者**：7 月 31 日是硬截止线。本周就在 [Pearson VUE](https://home.pearsonvue.com/microsoft) 确认考位状态——7 月底是节假日报名高峰，考场有可能约满。若无法赶在 31 日前通过，别死磕，直接切换到 AI-200 备考，省出来的时间值钱。

2. **AWS 方向的同学**：本周把 AWS Skill Builder 账号登进去，把四张免费 Microcredentials 加入学习计划。其中 Serverless Demonstrated 和 Agentic AI Demonstrated 对 JR 课程里「Serverless 架构实战」和「Bedrock Agent 实操」的内容高度吻合——边学边拿徽章，面试时可以展示。

5. **Coursera 用户**：今天（7 月 10 日）到 7 月 13 日是 Coursera Plus 促销最后窗口。AWS、GCP、Azure 主流认证备考路径在 Coursera 上都有 DeepLearning.AI / Google / Microsoft 官方课程，如果今年计划拿两张以上认证，算一下性价比再决定。
