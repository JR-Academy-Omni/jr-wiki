## 一、认证动态

### 1. AZ-204 退役倒计时24天：Azure Developer Associate 7月31日截止，AI-200 接档

**一句话**：正在备考 AZ-204（Developing Solutions for Microsoft Azure）只剩24天——7月31日后这张证再也无法获得，替代考试 AI-200 已同月上线。

微软官方确认，**Azure Developer Associate（AZ-204）认证将于 2026年7月31日正式退役**。退役后无法报名，也无法续期已有证书。这是2026年微软认证最大规模重组的一部分——仅6月至9月就有超过12张角色型认证退役，涵盖开发、安全、数据和 AI 领域。

**替代考试：AI-200（Azure AI Cloud Developer Associate）**

新考试 AI-200 聚焦 AI 集成应用开发，与 AZ-204 的技术侧重差异明显：

| 考查方向 | AZ-204 | AI-200 |
|---------|--------|--------|
| 核心能力 | Azure 通用开发（API/Storage/Auth）| AI 集成云应用开发 |
| 重点技术 | App Service / Azure Functions / Cosmos DB | Azure OpenAI / 向量数据库 / AI Foundry |
| 新增方向 | — | Serverless AI Functions / AI 工作流编排 |

AI-200 Beta 版在2026年7月上线，正式版训练课程同期在 Microsoft Learn 开放。AZ-204 的准备者需判断：**进度超过 70%，立刻排考；进度不足 70%，直接切换 AI-200**——AZ-204 退役后，Azure 开发者市场需求已明显向 AI 集成开发倾斜，AI-200 的技能图谱与市场方向更吻合。

### 2. AZ-500 → SC-500：安全认证重组进入7月，SC-500 培训本月上线

**一句话**：AZ-500（Azure Security Engineer）8月31日退役，替代考试 SC-500（Cloud and AI Security Engineer）培训在7月起陆续开放，新旧考试的主要差距在 AI 安全和 Prompt 注入防御。

微软正式宣布，**AZ-500（Microsoft Certified: Azure Security Engineer Associate）将于 2026年8月31日退役**，替代考试为 SC-500（Cloud and AI Security Engineer Associate）。SC-500 Beta 考试已于5月开放报名，正式考试预计在培训材料稳定后数月内跟进。

SC-500 相比 AZ-500 的核心扩展：

- **AI 工作负载安全**：保护基于 Azure AI Foundry 和 Azure OpenAI 的生产环境
- **Prompt 注入防御**：识别和缓解针对 LLM 应用的越狱和注入攻击
- **AI 供应链安全**：第三方模型接入的风险审查
- **AI 监控与合规**：AI 推理结果的审计日志和异常检测

Azure 安全基础（网络安全组、Key Vault、Defender for Cloud、Sentinel）在两张证书之间高度重叠，已有 AZ-500 备考积累的学员切换 SC-500 学习曲线相对平缓。差别主要集中在新增的 AI 安全章节。

微软角色型认证考试通常定价 **$165 USD**（各地区可能有税费差异），SC-500 预计在相同价格区间。

### 3. Databricks Genie 7月6日起切换按量计费：150 DBU 免费额度约撑80次 AI 问答

**一句话**：从昨天（7月6日）起，Databricks Genie Spaces / Genie Code / Genie One 从随工作区授权改为按用量计费，每用户每月150 DBU 免费，超出部分按 LLM 实际消耗计费，Service Principal 无免费额度。

Databricks 于 2026年7月6日正式将 Genie 产品线切换至**按用量计费（Pay-as-you-go）**定价模式，影响所有使用 Genie 的企业级 Databricks 用户。

**关键数据：**

| 项目 | 数值 |
|------|------|
| 每用户免费额度 | **150 DBU / 月** |
| 免费额度约等于 | 约 80-100 次 Genie 问答，或 20-30 次 Genie Code 编程会话 |
| 免费额度折算美元（美东区） | **~$10.50 USD / 月** |
| Service Principal | **无免费额度**，从第一次请求起全额计费 |

**哪些产品受影响：** Genie Spaces（自然语言查询数据仓库）、Genie Code（AI 辅助代码生成）、Genie One（统一 Genie 入口）。

**高风险场景：** 如果你的团队用 Databricks Service Principal（非人类身份）通过 API 自动调用 Genie，这类调用**没有任何免费额度**，从第一个 DBU 起即计费。有自动化脚本或 Pipeline 调用 Genie 的团队，需立即在账户控制台的「Genie Budgets」里设置月度预算上限，避免意外账单。用量可通过 `system.billing.usage` 系统表做账单分析。
