## 一、认证动态

### 1. 微软 SC-500 Beta 上线——VistaSC500 享80折，6月8日前前300名有效

**一句话**：SC-500（Cloud and AI Security Engineer Associate）Beta 本月开考，输入代码 `VistaSC500` 考试费打到约 $33 USD，正式替代8月31日退役的 AZ-500。

微软推出 SC-500，定位是「AI 时代的云安全工程师」认证。不同于旧版 AZ-500 专注 Azure 基础安全控制，SC-500 把 AI 工作负载的安全防护纳入核心考点：

- **身份保护**：Entra ID 配置、条件访问策略、特权身份管理（PIM）
- **数据安全**：Purview 信息保护、Azure Policy 自动化合规
- **AI 模型安全**：AI 工作负载的威胁检测、Copilot 的权限与数据边界控制
- **云端威胁防御**：Microsoft Defender for Cloud AI 工作负载防护、SIEM 集成

**Beta 折扣细节：**
| 项目 | 详情 |
|------|------|
| 折扣码 | `VistaSC500` |
| 截止时间 | 2026年6月8日（前300名有效，先到先得） |
| 折扣力度 | 80%（折后约 $33 USD） |
| Associate 正式考试原价 | 约 $165 USD |
| Beta 成绩出分 | 延迟6-8周，通过即正式认证，不需重考正式版 |
| 折扣限制 | 不适用于印度、土耳其、巴基斯坦、中国 |

**退役时间表**：AZ-500 2026年8月31日停办。现有 AZ-500 持证人证书有效期不受影响，但到期续期需要转考 SC-500。SC-500 官方学习路径已同步上线 Microsoft Learn，不需要额外付费学习材料。

操作方式：访问 [Microsoft Learn 搜 SC-500](https://learn.microsoft.com/en-us/credentials/certifications/exams/sc-500/) → Pearson VUE 预约 → 付款页面输入 `VistaSC500`。

### 2. CKA 更新至 Kubernetes v1.35——Gateway API 正式进考试，旧版备考资料有盲区

**一句话**：CKA 考试已更新到 Kubernetes v1.35，Gateway API（现已 GA）和原生 Sidecar 容器是新增必考内容，考费 $445 USD 不变。

2026年3月起，CKA 全面切换到 v1.35，并在五大考试域里加入了两个重要变化：

**Gateway API 取代传统 Ingress（部分场景）：**
Gateway 资源、GatewayClass、HTTPRoute 三件套现在出现在真实考试任务里。2025年以前出品的备考材料只教 `kubectl create ingress`，遇到 Gateway API 题会直接卡住。需要额外掌握 Gateway API 的资源类型、路由规则语法以及与 Service 的绑定方式。

**原生 Sidecar 容器（v1.29 GA）：**
v1.35 里 sidecar 通过 `initContainers` 配合 `restartPolicy: Always` 定义，不同于以往在 `spec.containers` 里加一个普通容器的写法。考试里可能出现调试 sidecar 日志注入或网络代理配置的场景题。

**考试规格（v1.35 最新）：**
| 项目 | 详情 |
|------|------|
| Kubernetes 版本 | v1.35 |
| 考费 | $445 USD（含1次重考 + 两次 killer.sh 模拟器各24小时）|
| 时长 | 2小时（17-25个实操任务） |
| 通过分数 | 66% |
| 考试形式 | PSI Secure Browser，全命令行，无选择题 |
| 注册入口 | [training.linuxfoundation.org](https://training.linuxfoundation.org/certification/certified-kubernetes-administrator-cka/) |

对备考的直接影响：Udemy 上2024年以前出品的 CKA 课程（Mumshad Mannambeth 等）的 Ingress 章节需要配合官方文档补 Gateway API 部分，killer.sh 模拟器已同步 v1.35 场景。临考前两次24小时模拟器要充分利用，已包含在 $445 里。

> 来源：[GitHub – CKA Study Notes v1.35](https://github.com/theplatformlab/CKA-Certified-Kubernetes-Administrator) · [Kubernetes Certification Prep Guide 2026 – CloudSoftSol](https://cloudsoftsol.com/kubernetes/kubernetes-certification-prep-guide-2026-complete-cka-cks-roadmap-for-success/)

### 3. AZ-204 7月31日退役——AI-200 Azure AI Cloud Developer 接盘，开发者要换证了

**一句话**：AZ-204（Azure Developer Associate）2026年7月31日停办，接替者 AI-200（Azure AI Cloud Developer Associate）把 AI 集成开发列为核心，现有 AZ-204 证书有效期三年内不变。

微软的逻辑很直接：当 Azure 应用开发几乎都要集成 Azure OpenAI Service、Copilot 插件或 AI Search，单纯的「会部署 API + 配 CI/CD」不足以定义一个「Azure Developer」。

**AZ-204 vs AI-200 核心变化：**
| 对比项 | AZ-204（退役）| AI-200（接替）|
|------|------|------|
| 主要技术栈 | App Service、Functions、Storage、Cosmos DB | 上述全部 + Azure AI Studio、Azure OpenAI、AI Search |
| AI 内容比重 | <10% | 约40%（AI 集成、Responsible AI、Agent 开发）|
| 退役 / GA | 2026年7月31日退役 | 预计2026年Q3 GA |
| 当前状态 | 最后2个月可报考 | Beta 阶段，折扣约 $33 USD |

对于正在备考 AZ-204 的学员：**7月31日之前考完仍然完全有效**，证书有效期三年，不需要急着转考 AI-200。但如果还没开始备考 AZ-204，直接备考 AI-200 更有长期价值——Beta 阶段折扣约 $33（正常价 $165 的80折），且与2026-2027年 Azure 岗位招聘需求更匹配。

### 4. AWS Security Specialty SCS-C03 更新——生成式 AI 安全防护进考纲

**一句话**：AWS Certified Security – Specialty（SCS-C03）更新考纲，Amazon Bedrock 数据隔离、Bedrock Guardrails 配置、机密计算成为新考点，2024年前备考资料有覆盖缺口。

更新内容聚焦三个方向：

1. **Generative AI 安全治理**：Amazon Bedrock 的模型调用审计（CloudTrail 日志）、Bedrock Guardrails 过滤规则配置、基础模型数据隔离机制。旧考纲里这些服务根本不存在。

2. **机密计算与合规**：AWS Nitro Enclaves 配置、金融/医疗合规场景的数据隔离要求。覆盖从零信任到可信执行环境（TEE）的架构选型题。

3. **零信任 + 上下文感知访问**：IAM Identity Center + Attribute-Based Access Control（ABAC）+ VPC Lattice 的组合使用，替代传统 VPN 边界安全模型的题目比例上升。

**考试规格：**
- 考试代码：SCS-C03
- 考费：$300 USD（Specialty 级）
- 时长：170分钟，65题
- 通过分数：750/1000
- 有效期：3年

AWS Skill Builder 上的「Security Specialty Exam Prep」课程已同步 SCS-C03 考纲，需要 AWS Builder ID 注册，免费可看。建议额外补充 Bedrock Guardrails 和 Amazon Macie 的官方文档——这两个服务是新考纲里高频出题的方向。

> 来源：[AWS Training and Certification Blog – Announcements](https://aws.amazon.com/blogs/training-and-certification/category/post-types/announcements/) · [AWS Certification Roadmap 2026 – StudyTech](https://studytech.ai/blog/aws-certification-roadmap-2026)

### 5. Google Cloud Professional ML Engineer 更新——Vertex AI Agent Builder 进考纲

**一句话**：Google Cloud Professional Machine Learning Engineer 随 Google Cloud Next '26 更新，Vertex AI Agent Builder 和 Model Garden 实操是新增考点，GCP 备考资料要对应更新。

2026年更新要点：

- **Vertex AI Agent Builder**：构建多步推理 AI Agent 的设计模式，包括工具调用（Function Calling）、grounding（搜索增强）、Agent 评估方法
- **Model Garden 模型选型**：根据任务类型（文本、代码、多模态）和延迟 / 成本需求在 Model Garden 里选合适的基础模型，不同于以前只考 AutoML 和 BigQuery ML
- **生成式 AI 解决方案评估**：LLM 输出质量的量化评估方法（BLEU/ROUGE/人工评估框架），需要能在 Vertex AI Evaluation Service 里配置评估 pipeline

旧版考纲（2024年前）里这三个方向几乎不涉及。Google Cloud Skills Boost 上已有更新的官方实操实验室对应这些新考点，GEAR 计划（下一节）每月35积分可覆盖大部分实验室费用。
