## 一、认证动态

### 1. AI-200 Beta：Azure AI Cloud Developer Associate 80折，5月27日截止（还有3天）

**一句话**：前300名用代码 `AI200Piedmont` 报考 AI-200 Beta，折后约 **$33 USD**，5月27日到期——今天是5月24日，实际窗口只剩3天。

微软于5月中旬正式发布 **Microsoft Certified: Azure AI Cloud Developer Associate** 认证，对应考试代码 **AI-200**（Beta阶段）。这张证书的定位非常明确：面向在 Azure 上端到端构建和落地 AI 产品的开发者，而不是泛化的 Azure 开发者或 AI 基础概念考试。

考试覆盖的技术栈包括：

- **容器化计算**（Containerized Compute）：把 AI 推理服务跑在 Azure Container Apps、AKS 上
- **向量数据库**（Vector-Enabled Databases）：Azure AI Search、Cosmos DB for MongoDB vCore 的向量索引与 RAG 流水线
- **事件驱动 AI 管道**（Event-Driven AI Pipelines）：Azure Event Grid 触发 AI 处理链路
- **密钥管理 & 可观测性**（Secret Management, Distributed Observability）：Key Vault + Azure Monitor 集成

> 来源：[Microsoft Community Hub – New Microsoft Certified: Azure AI Cloud Developer Associate Certification](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-ai-cloud-developer-associate-certification/4494116)

**Beta 报考方式**：

1. 进入 [Microsoft Learn 认证页面](https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-cloud-developer-associate/)
2. 点击「Schedule Exam AI-200」
3. 支付时输入折扣码 `AI200Piedmont`
4. **折扣不适用于土耳其、巴基斯坦、印度、中国**

Beta 通过后，完整认证预计 **2026年7月正式发放**。对在澳洲做 Azure 相关开发的工程师，$33 拿一张关联 AI 开发的微软认证，性价比是这个档位最高的。

> 来源：[Exam AI-200 Beta | Microsoft Certification Hub](https://msfthub.com/vouchers/ai200beta/)

### 2. DP-750 正式 GA：Azure Databricks Data Engineer Associate 本月上线

**一句话**：Microsoft Certified: Azure Databricks Data Engineer Associate（考试代码 **DP-750**）5月正式退出 Beta、进入全量发布，微软与 Databricks 认证体系正式打通。

这张证书是2026年微软认证体系最具含金量的新增之一。之前 Databricks 认证（Databricks Certified Data Engineer Associate）与微软认证是完全独立的两套体系；DP-750 的推出意味着你可以在微软的 Learn 平台备考、在 Pearson VUE 考试，并拿到微软官方 badge。

**核心考点**：

- 数据集成与建模（Databricks Lakehouse 上的 Delta Live Tables、Autoloader）
- 构建和部署生产级数据管道
- 故障排查和工作负载维护
- Azure Databricks 与 Azure 生态（Data Factory、Synapse Analytics、Azure Storage）的协同

**考试参数**：

- 时长：120分钟
- 通过线：700/1000
- 考试费：$165 USD（标准微软 Associate 级定价）
- 备考资源：官方课程 DP-750T00-A（讲师引导班）+ 微软 Learn 免费自学路径

> 来源：[Microsoft Learn – Study Guide for Exam DP-750](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/dp-750)

值得注意的是：Beta 阶段的80%折扣码 `DP750Deltona` 截止4月2日已到期。现在只能按原价考，但 GA 的好处是可以立即拿到成绩和 badge，不用再等 Beta 评分周期。

> 来源：[Databricks Community – Microsoft Certified: Azure Databricks Data Engineer](https://community.databricks.com/t5/certifications/microsoft-just-announced-a-new-certification-microsoft-certified/td-p/150607)

### 3. Google Cloud Professional ML Engineer 认证6月1日大改版：Vertex AI → Gemini Enterprise Agent Platform

**一句话**：Google Cloud Professional Machine Learning Engineer 考试将于 **6月1日** 上线新版，核心变化是把 Vertex AI 相关考点整体迁移至 **Gemini Enterprise Agent Platform** 框架。

这次改版的背景是 Google Cloud Next '26（2026年4月下旬举办）上发布的重要架构变化：Gemini Enterprise Agent Platform 是 Vertex AI 的演进方向，未来 Vertex AI 的服务和路线图将统一在 Agent Platform 下交付，而不再作为独立产品更新。

> 来源：[Virtualization Review – Google Cloud Next '26: Gemini Enterprise Agent Platform Leads AI-Centric News](https://virtualizationreview.com/articles/2026/04/24/google-cloud-next-26-gemini-enterprise-agent-platform-leads-ai-centric-news.aspx)

**对正在备考的人意味着什么**：

- 现在用 Vertex AI 旧版名词体系准备的材料，6月1日后可能和新版考试有出入
- Gemini Enterprise Agent Platform 考点新增：多 Agent 编排、Agent DevOps、Agent 安全治理
- Model Garden 相关内容（包括 Gemini 3.1 Pro、Gemma 4 等）将正式成为考试场景题背景

如果你计划今年考 Google Cloud ML Engineer，建议6月1日后再查一次官方 exam guide，确认考点已同步更新。

> 来源：[Google Cloud Blog – Google Cloud Next 2026 Wrap Up](https://cloud.google.com/blog/topics/google-cloud-next/google-cloud-next-2026-wrap-up)

## 二、免费 / 限免课程

### 1. Google Cloud GEAR Get Certified：免费拿价值 $200 的 GCP 考试 Voucher

**适合**：有企业邮箱（Google Cloud 客户公司）的工程师，想备考 Google Cloud 认证但不想先掏考试费
**链接**：[developers.google.com/program/gear/getcertified](https://developers.google.com/program/gear/getcertified)

GEAR（Gemini Enterprise Agent Ready）的 Get Certified 项目是目前最系统化的免费 GCP 认证路径——不是那种"随便做几道题领张 badge"的活动，而是 **9 周结构化 cohort**：

- 每周 ~9 小时自学（官方 hands-on labs + skill badge quests）
- 每周最多 6 场 live exam review session（讲师直播答疑）
- 完成所有 milestone → 获赠 **整个认证考试费用的 voucher**（Associate Cloud Engineer 考试标准收费 $200）

**申请条件**：

- 必须使用企业邮箱（公司需是 Google Cloud 的客户）
- 个人邮箱和大学邮箱**不接受**
- 需要先注册成为 GEAR 免费成员，然后在4周报名窗口内申请 Get Certified

如果目前报名窗口已关闭，可以在页面点「Notify Me」等下一期开放。

> 来源：[Google Cloud GEAR Get Certified Program – Google for Developers](https://developers.google.com/program/gear/getcertified)

**适合**：有 Python 和 SQL 基础、打算冲 Azure Databricks Data Engineer Associate 的工程师
**链接**：[databricks.com/learn/training/home](https://www.databricks.com/learn/training/home)

随着 DP-750 正式 GA，Databricks Academy 提供的 **免费 overview courses** 是目前最省钱的备考起点：

- **Lakehouse 架构**：从 Delta Lake 原理到 Medallion 分层设计，拿完可以在 LinkedIn 挂 Databricks badge
- **Generative AI on Databricks**：覆盖 MLflow、Model Serving、Agent Framework 的基础应用
- **Data Engineering Fundamentals**：Delta Live Tables、Autoloader、Spark Structured Streaming 核心操作

这些课程完全免费，可以直接用 Databricks 账号（免注册）访问。注意：这些是 overview/prep 级别，进阶的 DP-750T00-A 讲师班需要通过公司购买或参加 Databricks 合作伙伴培训项目。

> 来源：[Databricks Blog – Getting Databricks Certified: Now Easier with Free Overview Courses](https://www.databricks.com/blog/getting-databricks-certified-now-easier-free-overview-courses)

### 3. 微软 Learn：免费备考 AI-200 的官方自学路径

**适合**：打算趁这3天内报名 AI-200 Beta、但还没系统复习过的 Azure 开发者
**链接**：[learn.microsoft.com/en-us/training/courses/ai-200t00](https://learn.microsoft.com/en-us/training/courses/ai-200t00)

微软 Learn 上的 **AI-200T00-A** 对应的自学版路径（不含讲师班、不含 Lab 环境）是完全免费的。AI-200 的考点按照官方 study guide 分为：

- **AI 解决方案规划与管理**（Plan and Manage Azure AI Solutions）
- **生成式 AI 与 Agent 实现**（Implement Generative AI and Agentic Solutions）
- **计算机视觉解决方案**（Implement Computer Vision Solutions）
- **文本分析与信息提取**（Implement Text Analysis and Information Extraction）

Beta 考试有一个特点：**评分周期比正式考试长**（通常8周），意味着今天注册、用 AI200Piedmont 折扣码报名，考完可以慢慢等成绩，不影响日常工作节奏。比等7月正式版再备考便宜约$132。

> 来源：[Microsoft Learn – Course AI-200T00-A: Develop AI Cloud Solutions on Azure](https://learn.microsoft.com/en-us/training/courses/ai-200t00)

## 三、JR Academy 学员行动建议

1. **今天就报名 AI-200 Beta**：截止5月27日，用折扣码 `AI200Piedmont` 支付，折后约$33，比7月正式版省 $130+。如果你正在做 Azure AI 项目（RAG、Agent、CV），这张证书直接对上技术栈，不用额外准备陌生考点。

2. **DP-750 纳入 Data Engineer 学习路线**：Databricks 技能在澳洲数据岗（特别是金融、电信行业）需求很高。DP-750 GA 代表这张证书正式可被雇主识别为"微软认证"，比单独的 Databricks 证书多一层背书。先上 Databricks Academy 的免费课程，三周内过完 overview，再决定是否报考。

3. **Google Cloud Professional ML Engineer：6月1日前做一次考点对齐**：如果你在学 GCP ML，把手头材料里的 Vertex AI 相关章节标注一下，6月1日后对照新版 exam guide 检查哪些 API 名称和服务架构已经变化，避免用旧知识框架应对新题型。

4. **GEAR Get Certified 等下一期报名通知**：如果你有企业邮箱且公司在用 Google Cloud，马上去 GEAR 页面注册免费成员并开启 Notify Me，等下一批 Get Certified cohort 开放，这是拿 GCP 免费 voucher 最规范的官方渠道。

5. **别等"某个证书考完再准备下一个"**：AI-200（5月27）和 SC-730（5月27）两个 Beta 窗口同天截止，但考场分开、费用独立。如果两张证书都在你的计划里，本周内两个都报进去，之后再分别备考。
