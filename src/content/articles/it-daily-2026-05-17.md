## 一、认证动态

### 1. DP-100 倒计时：6月1日退役，还有 15 天

**一句话**：Azure Data Scientist Associate（DP-100）6月1日正式退场，现在报名已买不到完整准备时间，应立即转向接班人 AI-300。

Microsoft 在 2026 年的认证大洗牌中，把最早的退役日期定在了 **6月1日**——也就是 DP-100（Azure Data Scientist Associate）的最后一天。截至今天（5月17日），只剩 15 天。

DP-100 在过去三年一直是数据科学从业者的标配，覆盖 Azure Machine Learning 服务、模型训练与部署、数据管理等内容。但微软的判断是：纯"建模"技能已不够用，2026 年市场需要的是能把模型推进生产、管理模型生命周期的 **MLOps 工程师**，所以 DP-100 让位给 AI-300。

如果你现在还在备考 DP-100，实际上应该直接跳到 AI-300。AI-300 的考试内容转向 MLOps 实操：模型监控、漂移检测、CI/CD 流水线、Azure ML Pipelines、责任 AI 框架——这些正是企业招聘时考察的技能点。

AI-300 测试版（Beta）已于 2026年3月上线，**本月（5月）正式版应可报名**，标准考试费用 $165 USD（参考 Microsoft 认证考试通常定价）。

> 来源：[Microsoft 认证退役信息](https://learn.microsoft.com/en-us/credentials/support/credential-retirement) · [PassCert 2026 认证变动汇总](https://www.passcert.com/news_Microsoft-Certification-Shake-Up-in-2026-Retired-Exams-Replacements-and-New-AI-Focused-Certifications_2846.html)

### 2. AI-200 Beta 5月开放：AZ-204 的 AI 升级版

**一句话**：AZ-204（Azure Developer Associate）将于 7月31日退役，接班人 AI-200（Azure AI Cloud Developer Associate）的 Beta 训练和考试本月开放。

AZ-204 是 Azure 开发者证书里使用最广泛的一张，但微软认为 2026 年的云开发者必须掌握 AI 集成：调用 Azure OpenAI、构建 RAG 管道、部署 Copilot 扩展。因此新的 AI-200 把 AZ-204 的传统 App Service / Functions / Storage 内容做了压缩，大幅增加 AI 服务集成场景。

关键时间节点：
- **AI-200 Beta 训练**：2026年5月上线
- **AI-200 正式考试**：预计 2026年7月
- **AZ-204 退役**：2026年7月31日

Beta 阶段报名费通常打 80% 折扣（约 $132 vs 正式版约 $165），是最低成本入场时机。如果你手上有 AZ-204 的学习材料，核心差异就是新增的 **Azure AI Studio、Azure OpenAI Service 集成、Semantic Kernel** 相关内容，约占 AI-200 考题的 30-40%。

> 来源：[Microsoft Community Hub - AI-200 公告](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-ai-cloud-developer-associate-certification/4494116) · [mscertquiz Beta 考试汇总](https://mscertquiz.com/blog/new-microsoft-beta-exams-2026)

### 3. 6月底连退：AI-900 / AI-102 同天退役，AI-901 / AI-103 接班

**一句话**：2026年6月30日，Azure AI Fundamentals（AI-900）和 Azure AI Engineer（AI-102）同日退场，接班人 AI-901 和 AI-103 已在 Beta 阶段。

这是这波退役潮里影响面最大的一步。AI-900 是入门级 AI 认证，每年大量学员用它打基础；AI-102 是实操级别的 AI Engineer 证书，两者同日退役意味着整个"原有 AI 学习路径"在6月30日断档。

- **AI-901**（接 AI-900）：更新了 Generative AI 基础、Copilot 架构、负责任 AI 原则
- **AI-103**（接 AI-102）：聚焦 Azure AI App and Agent Developer，Beta 已于3月上线，GA 预计6月

距离 6月30日只剩 44 天。如果你原计划考 AI-900 做入门，现在的选择是：
- 冲最后一次 AI-900（5月底之前考，还能拿这张证书）
- 直接看 AI-901 材料，等新版正式上线

如果你在备考 AI-102，AI-103 Beta 现在可以报名，且 Beta 费用享折扣。

### 5. AWS AI & ML Scholars：10万个免费 Nanodegree 席位，6月24日截止

**一句话**：AWS 与 Udacity 合作的 AI & ML Scholars 项目提供 10万个免费三个月 Nanodegree 名额，包含 AI Programming with Python、Machine Learning 等方向，截止 6月24日。

这个项目的含金量在于：Udacity Nanodegree 通常需要 $200+/月，三个月价值约 $600，但通过 AWS Scholars 渠道完全免费。考核条件是完成项目并提交学习进度。

具体流程：
1. 访问 [AWS AI & ML Scholars 页面](https://aws.amazon.com/about-aws/our-impact/scholars/)
2. 点击"Apply Now"，选择 AI/ML 方向的 Nanodegree
3. 申请截止 **2026年6月24日**
4. 项目开始后 3 个月内完成课程

课程内容覆盖 Python 基础、NumPy/Pandas、深度学习（PyTorch）、AWS SageMaker 部署全链路，完成后获得 Udacity 证书。对于 JR Academy 正在学数据工程或 AI 方向的同学，这是最低成本的正规证书渠道之一。

## 二、免费 / 限免课程

### 1. Coursera Plus 40% 折扣：截止 5月25日，$35.40/月

**适合**：计划在 2026 年系统拿多张专业证书（Google / Meta / IBM / DeepLearning.AI Specialization）的学员
**链接**：[https://www.coursera.org/courseraplus](https://www.coursera.org/courseraplus)

40% 折扣后：Coursera Plus 月订阅从 $59 降至 **$35.40/月**，按三个月算节省 $71。折扣期限：**2026年5月25日**到期，仅限新用户。

三个月内能拿到什么？按正常学习节奏（每周 5-7 小时）可以完成：
- **Google Data Analytics Professional Certificate**（约 6 个月，中等强度；加速后 3 个月内可完成）
- **IBM AI Engineering Professional Certificate**（6 门课，适合已有编程基础的学员）
- **DeepLearning.AI Machine Learning Specialization**（Andrew Ng 出品，3门课）

Coursera Plus 涵盖 10,000+ 课程，不限数量。只要三个月内规划清楚，$106 就能拿 2-3 张专业证书。

### 3. Google Cloud Skills Boost：GEAR 项目每月 35 学习积分（无截止日期）

**适合**：准备 GCP 认证（Associate Cloud Engineer / Professional Data Engineer 等）的学员
**链接**：[https://www.cloudskillsboost.google/paths/](https://www.cloudskillsboost.google/paths/)

Google Cloud Skills Boost 的 **GEAR（Google Enterprise AI Readiness）项目** 每月提供 35 个免费学习积分，可用于兑换平台上的 Labs、Quests 和 Learning Paths。平台共有 1,200+ 个免费课程，覆盖：
- **Associate Cloud Engineer Learning Path**（推荐 GCP 入门者）
- **Professional Data Engineer Learning Path**
- **Generative AI for Developers Learning Path**（含 Vertex AI、Gemini API 操作）

Labs 是 Skills Boost 的核心价值：每个 Lab 给你一个临时的真实 GCP 项目，直接在真实云环境里操作，而不是模拟器。每个 Lab 约消耗 1-5 个积分。35 个月度积分够完成约 10-20 个 Labs，足够覆盖 ACE 备考所需的实验量。

注册 GEAR 需要公司邮箱（gmail.com 不行），JR Academy 学员可用学校邮箱申请。

### 4. AWS Skill Builder：Cloud Practitioner Essentials 免费（无截止日期）

**适合**：完全云计算零基础、正在考虑入门 AWS 的学员
**链接**：[https://aws.amazon.com/training/](https://aws.amazon.com/training/)

AWS Skill Builder 上有一门 **AWS Cloud Practitioner Essentials** 课程完全免费，不需要订阅、不需要信用卡，直接注册 AWS 账号就能学。课程约 6 小时，涵盖 AWS 全局服务地图、核心服务（EC2、S3、RDS、Lambda）基础概念，是 CLF-C02 考试的官方入门教材。

这门课本身不带证书（免费），但作为备考 CLF-C02 的第一步非常够用。CLF-C02 考试费 $100，通过后这是所有 AWS 高级认证路径的起点。

FreeCodeCamp 在 YouTube 上也有一套 14 小时免费课程（搜"AWS Cloud Practitioner 2026 FreeCodeCamp"），配合 AWS 官方课程使用效果更好。

## 三、JR Academy 学员行动建议

**1. DP-100 → AI-300，本周内做切换决定**
距 DP-100 退役仅 15 天。如果你备考材料已齐、有信心这两周冲刺，可以最后一战；但如果备考进度不足 60%，不如直接把材料切到 AI-300，同样内容大部分重叠，还是更新的证书。

**2. AI-200 Beta 机会窗口：5月内报名享折扣**
如果你有 AZ-204 基础（或在 JR Academy 学过 Azure 后端开发），AI-200 Beta 本月开放报名，约 $132 对比正式版 $165，差 $33。Beta 通过后拿到的是正式认证，不是 Beta 版证书。

**3. Coursera Plus 折扣 5月25日截止，这周必须决定**
三个月 $106，能系统完成一套 Professional Certificate（Google / IBM / DeepLearning.AI）。如果你在 JR Academy 的数据课程已过半，用这三个月同步拿一张 Google Data Analytics 证书是性价比最高的选择。

**4. AWS Scholars 申请不要拖**
免费 Udacity Nanodegree，三个月时长，6月24日截止。现在申请，最快6月初开始，8月底拿证书。特别适合正在学习 JR Academy 数据工程或 ML 路线的学员。

**5. GCP 考生：本周注册 Skills Boost GEAR，开始刷 Labs**
Associate Cloud Engineer 大约需要 30-50 个 Labs 的实操量。每月 35 个积分，一个月就够，比买 Qwiklabs 独立包便宜 80%。
