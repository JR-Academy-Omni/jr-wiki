## 一、认证动态

### 1. AZ-204 七月底退役：Azure 开发者认证向 AI-200 全面切换

**一句话**：微软将 AZ-204（Azure Developer Associate）退役日期定在 **2026 年 7 月 31 日**，替代认证 AI-200（Azure AI Cloud Developer Associate）Beta 已于今年 4 月 2 日开放。

这是 2026 年微软最大规模认证改革的一部分——12 张 Role-based 认证将在 6 月至 9 月间陆续退役，方向是把传统云开发技能全面向 AI 生成式应用和 Agent 架构靠拢。

AZ-204 与 AI-200 的核心差别：

| 维度 | AZ-204 | AI-200（替代） |
|------|--------|----------------|
| 定位 | 全栈 Azure 开发（计算 / 存储 / 网络 / API） | AI 应用开发（RAG / Agent / Foundry / Vector DB） |
| 退役 / GA | 2026 年 7 月 31 日退役 | Beta 已开放（4 月 2 日） |
| 重叠内容 | Azure Compute / Serverless | 约 30%–40% 内容可复用 |

**实操建议**：

- **已在备考 AZ-204**：离退役还有 ~59 天，8 周刷完完全来得及——用完现有的学习资料，7 月底前交卷。
- **刚准备开始**：直接备 AI-200 Beta，Beta 价格通常比正式版低 20%–30%，并且考试结果不计入 pass rate，风险更低。
- **已持有 AZ-204**：退役不会撤销已获得的认证，证书仍然有效；但如果面试官问 AI 开发，建议尽快跟上 AI-200 内容。

此外，**DP-100（Azure Data Scientist Associate）已于昨日（6 月 1 日）正式退役**，替代考试为 AI-300（MLOps Engineer Associate）。如果你的学习计划里还有 DP-100，需要立刻换档。

> 来源：[certempire.com — 7 Microsoft Certifications Retiring in 2026（"The AZ-204 exam and Azure Developer Associate certification will retire on July 31, 2026. The suggested replacement is exam AI-200, which leads to the Azure AI Cloud Developer Associate certification"）](https://certempire.com/microsoft-certifications-retiring-2026/)；[Microsoft Learn Q&A（"Will AZ-204 still be accepted as a prerequisite for AZ-400 after it retires on July 31, 2026?"）](https://learn.microsoft.com/en-us/answers/questions/5893548/will-az-204-still-be-accepted-as-a-prerequisite-fo)

### 2. AWS Certified Generative AI Developer – Professional 标准版正式开放

**一句话**：AWS 宣布 **AWS Certified Generative AI Developer – Professional** 标准版考试正式接受报名，不再仅限 Beta 阶段，面向所有准备好的候选人开放。

这张证书定位为 Professional 级别（AWS 最高层），考查：

- **Foundation Model 选型与部署**（Amazon Bedrock 生态）
- **RAG（检索增强生成）架构设计**：向量数据库选型、embedding 策略、知识库管理
- **Responsible AI**：偏见检测、可解释性、合规边界
- **AI Agent 构建**：工具调用链路、Bedrock Agents、Lambda 集成

Professional 级别考试通常定价 **USD $300**（各地区及 GST 加成可能有差异），考试时长 170 分钟。与原 Machine Learning Specialty（已于今年 3 月 31 日退役）相比，这张证书的重心从传统 ML Pipeline 全面转向 GenAI 应用层。

JR Academy 学员如果已拿下 AWS Solutions Architect Associate 或 Developer Associate，建议把这张 Professional 证书排进今年 Q3 目标——Gen AI 岗位 JD 里出现频率正在快速上升。

> 来源：[AWS Training and Certification Blog 搜索结果（"Registration is now open for the standard exam version of the AWS Certified Generative AI Developer – Professional certification, which validates expertise in foundation models, RAG architectures, and responsible AI deployment"）](https://aws.amazon.com/blogs/training-and-certification/)

### 3. 22 天倒计时：aws.training 平台 6 月 24 日永久关停

**一句话**：旧版 AWS 培训门户 **aws.training** 将于 2026 年 6 月 24 日停止访问，所有学习记录和课程证书必须在那之前下载完毕。

具体影响：

- **历史课程完成证书**：必须在 6 月 24 日前从 aws.training 的「Transcript」页面手动下载为 PDF；关站后无法再取回。
- **课程内容**：已全部迁移至 [AWS Skill Builder（skillbuilder.aws）](https://skillbuilder.aws/)，课程目录基本一致。
- **讲师授课（ILT）报名**：从即日起通过 Skill Builder 的「Classroom Training」入口完成。

迁移步骤：登录 [aws.training](https://www.aws.training/) → 进入「Transcript」→ 下载所有完成证书 → 用相同邮箱在 [skillbuilder.aws](https://skillbuilder.aws/) 注册 AWS Builder ID 完成迁移。

**22 天不长**。现在用 aws.training 且有历史学习记录的同学，今天就把证书存档。

> 来源：[AWS Training and Certification Portal（"The legacy aws.training website will no longer be accessible from June 24th, 2026. If necessary, you can download course completion certificates from the transcript until June 24th, 2026"）](https://www.aws.training/)

### 4. CNCF CARE 程序本月落地：考过 CKA 自动续签 KCNA

**一句话**：CNCF 全新的 Certification Advancement & Recertification Experience（CARE）程序将在 **2026 年 6 月正式实施**，核心规则：通过高阶认证自动续签对应的基础认证。

具体规则：

| 通过（或续签）这张 | 自动续签这张 |
|---|---|
| CKA 或 CKAD | KCNA（Kubernetes and Cloud Native Associate） |
| CKS | KCSA（Kubernetes and Cloud Native Security Associate） |

有效期：溯及 2026 年 1 月 1 日——1 月 1 日至程序上线之间通过对应考试的候选人，如果之前已拥有基础认证，会自动追溯生效。

对已经同时持有 KCNA + CKA 的同学：下次 CKA 续费时不再需要额外续签 KCNA，节省一份续费成本（KCNA 标准续费约 USD $100）。

对正在规划 Kubernetes 学习路径的同学：先冲 KCNA 打基础，然后直接备考 CKA，通过后两张都续。路径更短，不用担心 KCNA 到期。

> 来源：[CNCF 官方博客 2026-03-23（"KCNA will automatically renew when a practitioner passes or recertifies CKA or CKAD... KCSA certification will automatically renew when an IT professional passes or recertifies CKS"）](https://www.cncf.io/blog/2026/03/23/cncf-introduces-a-new-recertification-program-as-kubestronaut-community-surpasses-3500/)；[Cloud Native Now（"The CARE program will be fully implemented in June 2026 and became effective as of January 1, 2026"）](https://cloudnativenow.com/features/cncf-revamps-certification-program-to-simplify-renewals/)

### 5. Microsoft DP-750 Azure Databricks 数据工程师认证 GA

**一句话**：微软联合 Databricks 推出 **Microsoft Certified: Azure Databricks Data Engineer Associate**，考试代码 DP-750，于 2026 年 5 月初正式 GA。

这张认证填补了 Azure 数据工程方向的空白——原来持 DP-203 的同学需要 Databricks 技能，但没有一张专门覆盖 Azure Databricks 完整工作流的微软认证。DP-750 考查的核心内容：

- Azure Databricks 工作区创建与权限管理
- Delta Lake / Delta Live Tables 管理
- Databricks Workflows 与 Databricks Asset Bundles（DABs）
- Unity Catalog 数据治理
- MLflow 实验追踪与模型服务

Databricks 平台本身也同步在 5 月 4 日更新了其 **Databricks Certified Data Engineer Associate** 考试内容，新增 Liquid Clustering、更多场景题。两张证书方向互补：DP-750 侧重 Azure 集成，Databricks 原厂证书侧重平台深度。

> 来源：[Microsoft Community Hub — New Microsoft Certified: Azure Databricks Data Engineer Associate Certification（"General availability of the Microsoft Certified: Azure Databricks Data Engineer Associate Certification is expected in early May 2026. To earn the Certification, you need to pass Exam DP-750: Implementing Data Engineering Solutions Using Azure Databricks"）](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-databricks-data-engineer-associate-certification/4494114)

## 二、免费 / 限免课程

### 1. AWS Skill Builder 免费微证书：Agentic AI / Serverless / 云安全

**适合**：正在备考 AWS 认证、想顺手拿几张有含金量技能凭证的学员
**链接**：[skillbuilder.aws](https://skillbuilder.aws/)

AWS Skill Builder 目前提供无需付费订阅的免费 Microcredentials，完成方式是在真实 AWS 环境里完成一系列 Challenge Lab（Live AWS，不是模拟器）。涵盖方向：

- **Agentic AI**（构建 AI Agent，Bedrock Agents 实战）
- **Serverless**（Lambda 架构设计 + 部署）
- **Application Networking**（ALB / API Gateway 配置）
- **Incident Response**（云环境安全事件响应流程）

课程本身（自学视频部分）全部免费，900+ 课程；有部分 Lab 需要 Team 或 Individual 订阅（约 USD $29/月）才能解锁，但上述 Microcredential Challenge Lab 在免费层可访问。

另外提醒：aws.training 里原来的课程记录，6 月 24 日前必须备份，之后直接在 Skill Builder 继续。

> 来源：[AWS Training and Certification 搜索结果（"You can earn microcredentials in serverless, agentic AI, application networking, and incident response through real challenges in a live AWS environment with no subscription required"）](https://aws.amazon.com/blogs/training-and-certification/)；[AWS Skill Builder 官方（"Access over 900 free self-paced digital courses covering every AWS service and skill level"）](https://skillbuilder.aws/)
