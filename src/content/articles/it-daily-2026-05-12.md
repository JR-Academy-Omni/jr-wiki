## 一、认证动态

### 1. SnowPro Core COF-C02 后天（5月14日）正式退役 — 新考生今天切换 COF-C03

**一句话**：Snowflake 基础认证旧版 COF-C02 后天退场，2 月 16 日上线的 COF-C03 全面接棒，新版在 AI 工具（Cortex、Document AI、ML Functions）权重上有实质变化，旧题库直接报废。

距离 COF-C02 退役还剩不到 48 小时。**已持证的同学不受影响**——旧证继续有效，到期时按 COF-C03 要求续证即可。需要立即决策的是正在备考或近期打算报考的新考生。

**COF-C02 → COF-C03 主要变化一览**：

- **AI 工具权重上升（最大变化）**：COF-C03 新增 Snowflake Cortex（Snowflake 原生 LLM 函数）、Document AI（非结构化文档解析）和 ML Functions（分类、异常检测）相关考点；COF-C02 几乎没有 AI 内容
- **考试结构基本不变**：100 道选择题、115 分钟、通过线 70%，价格维持 $175（Snowflake 官方报名）
- **Domain 权重微调**：Virtual Warehouses 和 Data Sharing 模块权重略下调，新增 AI/ML Domain 独立分类

**备考资料切换指南**：
- 市面上绝大多数 COF-C02 Udemy / Pluralsight 课程对 COF-C03 **不适用**，Cortex 相关内容缺失
- 官方免费起点：[Snowflake Learn — COF-C03 认证页](https://learn.snowflake.com/en/certifications/snowpro-core-c03/) 有最新 Exam Study Guide 下载入口
- Snowflake 官方提供免费 [Hands-On Essentials — Intro to Snowflake](https://learn.snowflake.com/) 系列，COF-C03 备考可直接作为起点

> 来源：[SnowPro Core COF-C03 官方认证页](https://learn.snowflake.com/en/certifications/snowpro-core-c03/) · [COF-C03 新增内容详解（Medium）](https://cristian-70480.medium.com/whats-new-in-the-snowpro-core-cof-c03-certification-31dd3e212ac0)

### 2. SC-500 Beta 三天后（5月15日）开考 — $45 锁定下一代 Azure 安全认证最后窗口

**一句话**：微软 Cloud and AI Security Engineer Associate（SC-500）Beta 将于本周四（5 月 15 日）开放报名，Beta 价格约 $45，较 GA 正式价约 $165 省 73%；AZ-500 同步倒计时，8 月 31 日退场。

SC-500 是为替代 AZ-500（Azure Security Engineer Associate）而设计的新一代安全认证，继承了 AZ-500 的全部云安全基础，并在此之上叠加了生成式 AI 安全与 Agentic AI 防护的全新维度。

**SC-500 Beta 版核心考点**：
- **Azure 原生安全**（AZ-500 继承）：Defender for Cloud、Sentinel SIEM、Key Vault、Network Security Group
- **生成式 AI 部署安全（新增）**：Azure AI Content Safety 配置、AOAI（Azure OpenAI）访问控制与数据隔离、模型 Prompt Injection 防御
- **Agentic AI 系统安全（新增）**：多 Agent 工作流权限边界设计、Azure AI Foundry 部署的访问审计与日志监控
- **企业零信任架构**：Conditional Access 高级策略、Microsoft Entra ID 保护机制

**报名与结果时间线**：
- 5 月 15 日起：Pearson VUE 开放 SC-500 Beta 场次，搜索「SC-500」即可报名
- Beta 考后 6-8 周：微软完成评分校准，成绩正式发布
- 2026 年 7 月（预计）：SC-500 GA，正式价约 $165 生效
- 2026 年 8 月 31 日：AZ-500 退场，停止接受新报名

对正在权衡的同学：如果你计划在年内拿 Azure 安全认证，**$45 Beta 远比 GA 后 $165 划算**；唯一代价是需要等 6-8 周才知道成绩，且 Beta 阶段题目仍在校准期，不保证一次通过率与 GA 版相同。

> 来源：[New Microsoft Beta Exams 2026: SC-500 Coming in May](https://mscertquiz.com/blog/new-microsoft-beta-exams-2026) · [SC-500 完整考试指南（A Guide to Cloud & AI）](https://www.aguidetocloud.com/cert-tracker/sc-500/)

### 3. AWS SOA-C03 CloudOps Engineer 2026 Q2 考纲更新：容器 + 多账户架构正式入题

**一句话**：AWS CloudOps Engineer Associate（SOA-C03）2026 Q2 考纲更新落地，ECS / EKS / ECR 容器运维和 AWS Organizations 多账户架构管理正式纳入必考范围，2025 年前的备考资料存在覆盖盲区。

SOA-C03 是 AWS 少数专门面向运维工程师的 Associate 级认证，侧重监控、变更管理、安全合规和成本优化。2026 年 Q2 更新后，备考方向有两处关键调整：

**① 容器运维（新增 Domain）**

| 考点 | 具体内容 |
|------|---------|
| ECS 集群管理 | 任务定义版本控制、Service 滚动更新、健康检查配置、Fargate vs. EC2 启动类型选择 |
| EKS 基础运维 | Managed Node Group 配置、kubectl 权限（RBAC + aws-auth ConfigMap）、Fargate Profile |
| ECR 镜像管理 | 生命周期策略（自动清理旧镜像）、镜像扫描（Basic vs. Enhanced）、跨区域复制 |

**② 多账户架构（权重上升）**

- AWS Organizations SCP（Service Control Policy）实际限制场景：禁止特定区域、强制加密、限制实例类型
- Control Tower + Landing Zone 合规检查：Guardrails 分类（预防性 vs 探测性）
- 跨账户 CloudTrail 聚合到中央 S3 Bucket + CloudWatch 跨账户日志订阅

**备考影响**：2025 年前出版的 SOA-C03 备考书籍和视频课程几乎没有 EKS 运维内容，若当前学习资料里容器部分不足两个完整章节，建议补充 AWS 官方文档中 ECS/EKS 运维相关 Whitepaper。Skill Builder 上的 [SOA-C03 Standard Exam Prep Plan](https://skillbuilder.aws/category/exam-prep/cloudops-engineer-associate) 已更新覆盖新考点，**免费**，建议以此对照检查学习覆盖范围。

> 来源：[AWS Certified CloudOps Engineer Associate 官方考试指南（PDF）](https://d1.awsstatic.com/onedam/marketing-channels/website/aws/en_US/certification/approved/pdfs/docs-cloudops-associate/AWS-Certified-CloudOps-Engineer-Associate_Exam-Guide.pdf) · [SOA-C03 备考指南 2026（Towards The Cloud）](https://towardsthecloud.com/blog/aws-cloudops-engineer-associate-exam-guide)

### 4. Microsoft DP-750 Azure Databricks Data Engineer Associate — GA 一周后备考实情

**一句话**：微软与 Databricks 联合背书的 DP-750（Implementing Data Engineering Solutions Using Azure Databricks）正式 GA 已过一周，考试难度定位在 Associate 与 Professional 之间，与 Databricks 原生 DE-A 存在明显差异。

DP-750 是微软认证体系中专门针对 **Azure Databricks 用户**设计的新认证，通过考试可获得 Microsoft Certified: Azure Databricks Data Engineer Associate 徽章，与 Databricks 原有的 Databricks Certified Data Engineer Associate（DE-A）是两个不同认证。

**DP-750 覆盖范围**：
- Lakehouse 架构：Delta Lake 事务日志、Time Travel、Z-Order 优化
- 数据摄取：Auto Loader、Structured Streaming、Change Data Capture（CDC）
- Unity Catalog 权限体系：Metastore 配置、Schema / Volume 权限分配
- 工作流自动化：Databricks Workflows 触发器、Cluster 策略与成本管理
- Azure 生态集成：Azure Data Factory 管道调度 Databricks Job、Event Hubs → Databricks 流式摄取

**与 Databricks DE-A 的核心区别**：

| 维度 | Databricks DE-A | Microsoft DP-750 |
|------|----------------|------------------|
| 发证方 | Databricks | Microsoft |
| 侧重 | 平台原理（Spark 原理、DLT 框架） | Azure 生态集成（ADF、Synapse、ADLS） |
| 价格 | $200 | $165（Pearson VUE 标准价） |
| 企业识别度 | 数据工程师圈子认可 | Azure 企业客户更易识别 |

对于在 Azure 环境上使用 Databricks 的数据工程师，DP-750 是更容易被企业 HR 和内部培训体系认可的选择；对于想在 Databricks 市场横向扩展的独立顾问，两个认证搭配效果更好。

> 来源：[New Microsoft Certified: Azure Databricks Data Engineer Associate（Microsoft Tech Community）](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-databricks-data-engineer-associate-certification/4494114) · [Microsoft Certification Guide 2026（lived365）](https://www.lived365.com/post/microsoft-certification-guide-2026-newexams-retirements-aiskills-and-careerimpact)
