## 1. GPT-5.6 Sol 开启预览，美政府要求限发「可信合作伙伴」

![GPT-5.6 Sol preview](https://images.unsplash.com/photo-1677442135703-1787eea5ce01?w=800&auto=format)

**一句话**: OpenAI 推出 GPT-5.6 三件套，但美国政府以国家安全为由要求仅向可信合作伙伴开放访问权限。

OpenAI 于 6 月 26 日正式预览 GPT-5.6 系列模型，包含三个层级：Sol（旗舰级，搭载最强安全防护栈）、Terra（均衡型，对标 GPT-5.5 性能但成本降低一半）、Luna（轻量快速型，专为低成本高并发场景设计）。与此同时，GPT-4.5 已于同日正式下线，标志着 OpenAI 产品线的代际切换正式完成。这是 OpenAI 近年来在单次发布中覆盖旗舰、均衡、低成本三档最完整的一次布局，意图在价格竞争白热化的背景下，用差异化分层策略同时守住高端与规模市场。

然而，此次发布伴随一个罕见的政策限制：CNBC 报道称，OpenAI 应美国政府要求，将 GPT-5.6 Sol 的初始访问权限限定在「一小批可信合作伙伴」之内，具体名单未公开。这是美国政府首次以国家安全理由，将管制范围从芯片出口延伸至模型访问权本身，意味着 AI 能力管控的颗粒度已细化至单个模型的 API 权限层面。对于正在关注 GPT-5.6 接入的开发者与企业而言，能否进入白名单将成为一个新的不确定因素。

与模型发布并行的，是 OpenAI 与 Broadcom 联合发布的自研推理芯片「Jalapeño」。这是 OpenAI 第一款专为 LLM 推理设计的 Intelligence Processor，围绕 OpenAI 自身对推理架构的判断量身打造。芯片级别的自研能力意味着 OpenAI 正在尝试摆脱对 Nvidia GPU 的单一依赖，如果 Jalapeño 量产后能稳定运行 Sol 系列，其推理成本和响应延迟将有望出现结构性改善，进一步拉开与依赖通用算力的竞争者之间的差距。

## 2. DeepSeek V4 七月中旬正式版发布，峰时 API 定价翻倍

![DeepSeek V4 release](https://images.unsplash.com/photo-1620712943543-bcc4688e7485?w=800&auto=format)

**一句话**: DeepSeek V4 官方版将于七月中旬上线，引入峰时定价翻倍机制，同步融资约 500 亿元人民币。

DeepSeek 官方宣布，V4 正式版将于 7 月中旬发布，并公布了两项核心技术创新：DeepSeek Sparse Attention（DSA）稀疏注意力机制，以及基于 token 压缩层级运行的新型注意力架构。这套组合据官方描述在长上下文场景下达到「全球领先」水准，同时相比传统注意力机制显著降低了显存占用与计算量。这意味着 DeepSeek V4 在处理超长文档、代码库、法律文书等场景时的能力和效率将有实质性提升，而非单纯在标准基准测试上刷分。

定价机制的变化同样引人注目。DeepSeek V4 Pro 和 V4 Flash 两款 API 将在非峰时段保持现有价格不变，但在工作日峰时段（9:00–12:00 和 14:00–18:00）价格翻倍。这种动态定价策略旨在通过价格信号平滑需求曲线、保证服务稳定性。对于在峰时大量调用 DeepSeek API 的企业而言，成本结构将发生显著变化，需要在工程层面做好任务调度优化，将非时间敏感型任务迁移到非峰时段执行。

在资本层面，Bloomberg 报道显示 DeepSeek 已接近完成约 500 亿元人民币（约 74 亿美元）的融资，投资方包括腾讯、宁德时代和国家人工智能产业投资基金。这笔融资规模与 Anthropic、OpenAI 近期融资轮次相当，标志着中国 AI 独角兽正在进入「第一梯队资本量级」。Ramp 6 月消费数据显示，DeepSeek 已成为美国企业增长最快的 SaaS 支出项目之一，价格优势在全球竞争中正在兑现为可见的市场份额。

## 3. Claude Fable 5 发布即遭美国出口管制，访问被迫暂停

![Claude Fable 5 export control](https://images.unsplash.com/photo-1633356122544-f134324a6cee?w=800&auto=format)

**一句话**: Anthropic 最强模型 Fable 5 发布后，随即收到美国政府出口管制指令，Fable 5 与 Mythos 5 访问全面暂停。

Anthropic 于 6 月发布了 Claude Fable 5，定位为其 Mythos 级别旗舰模型，具备常开自适应思维（adaptive thinking）、百万 token 上下文窗口和 128K token 输出能力。基准测试结果显示 Fable 5 在几乎所有已测评的基准上均达到当前最高分，涵盖真实 GitHub 代码任务、多模态推理、复杂长文本处理等实际工作场景。从技术角度看，这是 Anthropic 迄今推出的最强产品，同时也是其「安全与能力并行」理念的最新体现。

然而，据报道 Anthropic 于 6 月 12 日收到美国政府出口管制指令，被要求暂停 Fable 5 和 Mythos 5 的全球访问权限。这一事件与 OpenAI GPT-5.6 Sol 被要求限发「可信合作伙伴」发生在同一周期内，表明美国政府正在对多家前沿 AI 实验室同步施压，统一管控其最高能力模型的访问范围。这是 AI 监管从「事后审查」转向「事前管控」的重要信号，也是自芯片出口管制以来，美国在 AI 主权领域的又一次系统性行动。

在此背景下，Anthropic 仍在拓展其商业版图。Claude 正式登陆 Microsoft Azure Foundry，提供 Azure 原生的身份认证、计费与数据治理支持，Opus 4.8 和 Haiku 4.5 在 Messages API 上已可访问。加州政府达成协议，成为首个将 Claude 覆盖所有州级机构与地方政府的案例，Anthropic 以半价供应换取规模化政府采购。对于澳大利亚和亚太地区的企业用户，切换至 Opus 4.8 或通过 Azure 企业合规通道部署，是在 Fable 5 访问受限期间的可行替代方案。

## 4. Shazeer 奔 OpenAI、Jumper 转投 Anthropic，DeepMind 人才出走潮加剧

![DeepMind talent exodus](https://images.unsplash.com/photo-1522071820081-009f0129c71c?w=800&auto=format)

**一句话**: Google DeepMind 两位顶级人才相继出走，Noam Shazeer 加入 OpenAI，诺贝尔奖得主 John Jumper 转投 Anthropic。

Noam Shazeer 是 Google 内部的传奇工程师，也是 Transformer 架构奠基论文的作者之一，曾创立 Character.AI 后被 Google 重新收购。他宣布再度离开 Google DeepMind，这次选择加入 OpenAI。几乎同期，John Jumper——2024 年诺贝尔化学奖得主，因 AlphaFold 蛋白质结构预测工作获奖——宣布离开 DeepMind，转投 Anthropic。两人在同一时间窗口相继出走，在 AI 圈引发高度关注，Axios 将这波态势称为「AI 实验室的音乐椅游戏冲击 Google 最深」。

从战略价值看，Shazeer 在推理效率优化领域的深度积累，与 OpenAI 近期在自研芯片 Jalapeño 和推理成本压缩方向的重点投入高度契合，其加入可能实质性加速 OpenAI 在模型服务效率上的突破节奏。Jumper 的核心能力在于将深度学习与真实生物物理约束结合的方法论，进入 Anthropic 后若应用于科学 AI 赛道，将大幅拓展 Anthropic 的业务边界，特别是在医药研发、生命科学 AI 场景中的影响力。

对于这波离职潮，DeepMind CEO Demis Hassabis 在 6 月 23 日接受 Semafor 采访时坚称 Google DeepMind 仍然赢得人才战争。与此同时，DeepMind 于 6 月 18 日发布 AI 内部控制路线图，主动公开其对高能力 AI 的系统级安全管控框架；并联合 Schmidt Sciences 等机构发起最高 1000 万美元的多智能体安全研究资助计划。这些举措表明 DeepMind 正试图用研究影响力和安全话语权来对冲人才流失带来的品牌损耗。

## 5. Grok 4.5 进入 Tesla/SpaceX 私测，但 Grok 流量今年暴跌 22%

![Grok 4.5 private beta](https://images.unsplash.com/photo-1518770660439-4636190af475?w=800&auto=format)

**一句话**: Grok 4.5 在 Tesla 和 SpaceX 展开内部私测，声称接近 Claude Opus 水准，但 Grok 今年 web 流量同期跌幅居 AI 产品之首。

马斯克宣布 Grok 4.5 已进入 Tesla 和 SpaceX 内部私测阶段，并声称其内测成绩接近 Anthropic Claude Opus 系列，这也是 xAI 一贯的「先在自家企业内部验证，再向外部发布」策略的延续。与此同时，Grok 在 Databricks 2026 Data + AI Summit 上宣布原生集成至 Databricks Agent Bricks，Grok 4.3 也已在 Amazon Bedrock 正式上线，提供百万 token 上下文和可配置推理强度。技术和生态布局层面，xAI 的动作并不算慢。

然而与此形成强烈反差的是，Forbes 援引报告称 Grok 的 web 流量在今年 1 月至 5 月间下跌了 22%，成为同期所有主流 AI 产品中跌幅最大的一个。报告同时指出，Grok 当前流量中有相当比例由成人内容驱动，这与 xAI 希望打造的技术标杆形象存在明显落差。这一现象揭示了一个深层矛盾：Grok 作为 X 平台的原生产品，初期获得了大量因内容宽松政策带来的访问量，但这部分流量缺乏商业价值，且难以转化为开发者和企业用户的粘性。

对于在评估 Grok API 的开发者而言，4.5 版本若如期在私测后对外开放，并且宣传中的性能优势属实，其在 Bedrock 上的最低幻觉率表现将是重要参考。但 xAI 在消费端品牌形象的持续下滑，意味着生态建设和社区吸引力仍然是其与 OpenAI、Anthropic 竞争中的明显短板，API 开发者在技术之外也需要评估平台长期稳定性与合规风险。
