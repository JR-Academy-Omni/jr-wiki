## 一、认证动态

### 1. 微软 SC-730 Beta 上线 —— 5月27日前用 SC730Eclipse 打8折，仅限前300名

**一句话**：SC-730（Cybersecurity Business Professional）Beta 本月开考，前300名用代码 `SC730Eclipse` 把考试费打到约 $33 USD。

SC-730 是微软面向"非安全专业人员"设计的新认证，目标人群是每天接触数字工具却不靠网络安全吃饭的人：行政助理、数据分析师、项目经理、市场专员、销售人员。考试验证你能否识别常见网络威胁、执行基础数据保护操作、理解 LLM 和生成式 AI 带来的新型风险，并在安全事件发生时走正确的上报流程。

**折扣细节：**
- **代码**：`SC730Eclipse`
- **截止**：2026年5月27日，前300名有效
- **折扣力度**：80%（折后约 $33 USD，微软 Beta 考试原价参考 Associate 级约 $165）
- **操作方式**：Pearson VUE 预约 SC-730，付款页面输入优惠码

Beta 阶段注意：成绩延迟6-8周出，通过即为正式认证，不需要重考正式版。正式考试预计7月 GA，届时折扣不再适用。

对 JR Academy 的学员来说，这张证不需要编程背景，约60-90分钟考完，适合想在简历上补一张微软安全认证的运营、产品和业务分析岗位同学。

> 来源：[Microsoft Learn – Exam SC-730](https://learn.microsoft.com/en-us/credentials/certifications/exams/sc-730/) · [The Next Wave of AI Certifications: Three New Microsoft Beta Exams](https://mightybs.com/posts/2026-05-09-the-next-wave-of-ai-certifications-three-new-microsoft-beta-exams/)

### 2. AB-210 Beta 开考 —— Dynamics 365 Sales AI Consultant，接替7月31日退役的 MB-280

**一句话**：AB-210（Dynamics 365 Sales AI Consultant Associate）Beta 和配套学习路径5月上线，正式考试6月 GA，MB-280 持证人须单独备考新证。

AB-210 专为"AI 时代销售顾问"设计：用 Microsoft Copilot 和 Dynamics 365 Sales 构建从潜客到回款（lead-to-cash）全流程的 AI 驱动销售方案。核心考点包括：

- **Copilot 驱动的销售效率工具**：AI 辅助商机评级、邮件摘要与跟进建议
- **AI Agent 配置与生命周期管理**：在 Dynamics 365 里上线、监控、迭代 AI Sales Agent
- **负责任 AI 与数据治理**：销售数据合规处理、AI 决策审计
- **Dynamics 365 Sales 与 Microsoft 365 集成**：Teams、Outlook、Power Platform 协作配置

**时间线：**
- Beta + 学习路径：**2026年5月**（当前）
- 正式考试 GA：**2026年6月**
- MB-280 退役日期：**2026年7月31日**

注意：从 MB-280 到 AB-210 没有自动迁移，需要单独报考并通过新考试。Microsoft Learn 上的 AB-210 官方免费学习路径已同步上线，不需要购买第三方培训材料。

> 来源：[Microsoft Learn on X](https://x.com/MicrosoftLearn/status/2052554075716165934) · [Microsoft Certification Retirements 2026](https://vladtalkstech.com/microsoft-learning-and-credential-news/microsoft-certification-retirements-2026/)

### 3. AWS AIP-C01 全价 $300 正式开考 —— Bedrock 为核心的 GenAI Professional 认证

**一句话**：AWS 首张生成式 AI 专业级认证 AIP-C01（Generative AI Developer – Professional）从 Beta 转入全价阶段，$300 USD，Bedrock 模型集成与治理是主考方向。

这不是 AI Practitioner 的进阶版，而是专门验证"能把大模型落地到生产系统"的开发者工程能力。

**考试规格：**
| 项目 | 详情 |
|------|------|
| 考试代码 | AIP-C01 |
| 考试费 | $300 USD（Beta 阶段 $150，现已全价） |
| 题目数量 | 85 题（多选 + 多答案） |
| 考试时长 | 205 分钟（约3.5小时） |
| 通过分数 | 750 / 1000 |
| 有效期 | 3年，需要续期 |
| 考试语言 | 英文 |

**5大考试域权重：**
1. 基础模型集成、数据管理与合规 —— **31%**
2. 实现与集成（API 调用、RAG pipeline、Agents 构建）—— **26%**
3. AI 安全、安全性与治理框架 —— **20%**
4. GenAI 应用的运营效率与成本优化 —— **12%**
5. 测试、验证与故障排查 —— **11%**

**前置要求**：2年以上 AWS 生产应用开发经验 + 1年 GenAI 方案落地经验。Bedrock、SageMaker、Amazon Q、RAG 架构是必须能实操的服务。

> 来源：[AWS – Certified Generative AI Developer Professional](https://aws.amazon.com/certification/certified-generative-ai-developer-professional/) · [AWS Generative AI Developer Guide – Whizlabs](https://www.whizlabs.com/blog/aws-generative-ai-developer-guide/)

### 4. AWS Professional 级考试6月1日起加入 Bedrock + Amazon Q 场景题

**一句话**：6月1日后参加 AWS Solutions Architect – Professional 或 DevOps Engineer – Professional，会遇到基于 Bedrock 部署、Amazon Q 接入和 AI 治理框架的真实场景题，现有备考资料不够用。

AWS 在 Q2-Q3 2026 对旗舰认证做系列调整：

- **SAA-C04（Solutions Architect – Associate）**：「弹性架构设计」域权重升至30%；成绩出分从原来1-5个工作日缩短至**24小时以内**
- **Professional 级（SAP-C02 / DOP-C02）**：从**6月1日**起加入多步骤生成式 AI 场景判断题，涉及 Amazon Q Business 接入、Bedrock 基础模型选型与部署、AI 使用的安全治理

**对备考影响的实操建议**：6月1日前参加 Professional 级的同学按原考纲备考问题不大；6月1日后开考的同学，建议提前在 AWS Skill Builder 上过一遍 Bedrock 基础架构（有免费模块）和 Amazon Q Business 文档。

另外，AWS 从5月15日起更新 AUD、EUR、CNY 汇率，USD 考试价格不变。AUD 学员报考费用可能因汇率变化略有浮动，建议以 Pearson VUE 结账页面显示价格为准。

### 5. AI-900 六月底停办 —— 剩40天，适合零编程背景入门 Azure AI

**一句话**：微软 AI-900（Azure AI Fundamentals）6月30日退役，剩最后40天，是零编程背景进入 Azure AI 认证体系成本最低的路径。

AI-900 考试费约 $165 USD，考试时长45-60分钟，不要求编程经验，覆盖 Azure AI 服务的基础概念（Azure Machine Learning、Azure Cognitive Services、Azure Bot Service 等）。退役后，成绩永久保留在 Microsoft 认证档案，但**无法再报考，也不计入续期**。

适合人群：产品经理、运营、项目管理、业务分析师——想在简历上有一张微软 AI 背书，但不打算深入 AI 工程方向的同学，AI-900 是目前入门门槛最低的微软 AI 认证（退役后最近的替代是 AI-102，门槛高出不少）。

> 来源：[Microsoft Credential Retirement](https://learn.microsoft.com/en-us/credentials/support/credential-retirement) · [Microsoft Certification Retirements 2026 – VladTalksTech](https://vladtalkstech.com/microsoft-learning-and-credential-news/microsoft-certification-retirements-2026/)

### 1. DeepLearning.AI + Hugging Face：《Open Source Models with Hugging Face》— 免费短课约2小时

**适合**：想用 Llama、Mistral、CLIP 等开源模型搭 AI 应用，不想被商业 API 费用锁定的开发者
**链接**：https://www.deeplearning.ai/courses/open-source-models-hugging-face

DeepLearning.AI 与 Hugging Face 联合出品，平台 Beta 期间免费访问，无需订阅。约2小时课程涵盖：

- 在 Hugging Face Hub 按任务、Star 数、内存需求筛选合适的开源模型
- 用 `transformers` 库几行代码跑 NLP、语音转文字、图像识别、多模态任务
- 用 Gradio 把模型封装成 Web UI，部署到 Hugging Face Spaces（免费托管）

学完能独立构建一个功能完整的 AI demo，不依赖 OpenAI API，适合备考 AWS AIP-C01 的工程师补充开源模型知识，也适合想自己搭演示项目的求职同学。
