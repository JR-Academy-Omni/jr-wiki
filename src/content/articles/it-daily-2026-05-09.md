## 一、认证动态

### 1. 微软 AB-210 Beta 五月开放：约 $45 拿下 Dynamics 365 Sales AI 认证

**一句话**：AB-210 是 MB-280 的接替者，beta 窗口只在五月，$45 就能考正价 $225 的证书。

AB-210 Dynamics 365 Sales AI Consultant Associate 是微软 2026 年认证换代浪潮的第四张新证（前三张 AI-103、AI-200、SC-500 已在 4–5 月先后推出），beta 版于 2026 年 5 月正式开放注册。

**考点**：基于 Dynamics 365 Sales、Microsoft Copilot 和 Agent 能力，设计和配置 AI 增强型销售解决方案；涵盖 lead-to-cash 全流程、对话式智能（Conversational Intelligence）、预测性 Insights、引导式自动化和安全数据访问。

**价格时间线**：
- Beta 注册：2026 年 5 月
- Beta 考试费：约 $45（正价约 $225，80% 折扣）
- 正式版上线：预计 2026 年 6 月
- Beta 失败：不计入考试记录，没有历史污点

**前身**：MB-280 已退场，AB-210 是在 Copilot + Agent 时代重写的替代版本，考纲加入了大量生成式 AI 销售流程内容，与原版差异较大，不能混用备考材料。

对 CRM/Salesforce 方向的学员或正在学 Power Platform 的人来说，这张证加上 AI Agent 背景在简历上有明显新鲜感。Beta 窗口是成本最低的入场时机。

> 来源：[Microsoft Learn — AB-210 Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/ab-210)

### 3. CKA/CKAD/CKS 2026 考纲重大更新：Gateway API 正式入题，旧材料已过时

**一句话**：Kubernetes Gateway API 2024 年 GA、2025 年成主流，2026 年题库全面加权，光靠旧 Ingress 知识会丢分。

2026 年初 Linux Foundation 对三张 K8s 认证同步更新考纲：

- **CKA（集群管理员）**：Gateway API 目标新增，约占考试 5-7% 权重，部分原有 Ingress 规则题目被替换
- **CKAD（应用开发）**：约 5% 题目涵盖 Gateway API 路由配置和跨命名空间流量管理
- **CKS（安全专家）**：新增 Gateway 级别 TLS 策略和认证控制

**必须补的知识点**：
- `GatewayClass`、`Gateway`、`HTTPRoute`、`TLSRoute` 四个资源的关系
- Gateway API 与旧 Ingress 的对比（为什么 Gateway 更灵活）
- 跨命名空间路由配置（Gateway API 的核心优势之一）

如果你用的是 2024 年以前的备考材料（Udemy Mumshad Mannambeth 旧课、KodeKloud 旧题库），必须额外补充 Gateway API 部分。官方学习文档：[gateway-api.sigs.k8s.io/guides](https://gateway-api.sigs.k8s.io/guides/)。KodeKloud 已在 2025 年底更新题库，购买前确认版本。

> 来源：[Hakia — Kubernetes Certifications 2026](https://hakia.com/skills/kubernetes-certifications/)

### 4. AWS CloudOps Engineer Associate SOA-C03：2026 年备考必看的三个新考点

**一句话**：AWS SysOps Administrator 在 2025 年 9 月更名换题，但 2026 年市面上仍有大量旧 SOA-C02 材料，这三点一定要核对。

AWS Certified CloudOps Engineer - Associate (SOA-C03) 自 2025 年 9 月 30 日起替代 SOA-C02（SysOps Administrator），目前市面上只有 SOA-C03 可以注册参考（SOA-C02 于 2025 年 9 月 29 日退场）。

**SOA-C03 相对 SOA-C02 的三大新增方向**：

1. **容器（Containers）进入考纲**：ECS 任务定义和服务管理、EKS 基础操作和 CloudWatch 监控告警，SOA-C02 完全没有
2. **多账号 / 多区域架构加重**：AWS Organizations、Control Tower、跨区域 S3 复制策略，原 SOA-C02 只是轻描淡写
3. **自动化与 IaC 比例提升**：CloudFormation StackSets、Systems Manager Automation、EventBridge 自动响应规则，与 DevOps Engineer 方向深度交叉

**注意**：持有旧版 SysOps Administrator（SOA-C02）认证的人，在 3 年有效期内无需重考；但首次准备考试必须按 SOA-C03 备考，不能买旧套餐。Tutorials Dojo 和 Jon Bonso 的 SOA-C03 模拟题已更新，购买时看清版本。

> 来源：[AWS Certified CloudOps Engineer - Associate](https://aws.amazon.com/certification/certified-cloudops-engineer-associate/)

### 2. Google GEAR 计划：每月 35 个免费 Cloud Lab Credit，长期开放

**适合**：备考 Google Cloud Associate Cloud Engineer 或 Professional Cloud Architect 的学员  
**链接**：[skills.google](https://www.skills.google/)

GEAR（Google Earn and Reward）是 Google Cloud Skills Boost 的免费积分计划，无截止日期：

- 每月可免费领取 35 个 Lab Credit
- Credit 可用于 Google Cloud Skills Boost 全部实操 Lab（1,200+ 课程，涵盖 ML、安全、数据工程、应用开发）
- 在真实 GCP 临时账户中运行，不需要绑定信用卡
- 完成对应学习路径的 Lab 可兑换技能徽章（Skill Badge），记录在个人 Google Cloud 档案

**备考 ACE 的具体路径**：在 skills.google 选「Cloud Engineer Learning Path」→ 按顺序完成 Lab → 积累 Skill Badges → 参加 Get Certified 计划（5 月 7 日已报道，结业送 $200 考试券）。

学生和教职工身份另有独立通道，全部内容免费，不消耗 GEAR Credit。

> 来源：[Google Skills — Choose Your Path](https://www.skills.google/paths)
