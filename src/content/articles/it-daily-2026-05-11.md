## 一、认证动态

### 1. CKA 2026 改版：Kubernetes Gateway API 正式入考，通过分从 74% 降至 66%

**一句话**：2026 年初 CNCF/Linux Foundation 刷新 CKA 考纲，新增 Gateway API 目标（约 5-7% 权重），Kubernetes v1.34 为基准版本，$445 起报，全动手操作 17 道任务。

Linux Foundation 在 2026 年初完成了 CKA（Certified Kubernetes Administrator）的新一轮考纲更新，核心变化是 **Kubernetes Gateway API 正式取代 Ingress 进入必考范围**，权重约 5-7%。Gateway API 在 2024 年 GA、2025-2026 年间成为 Kubernetes 流量管理的实际标准，旧版考纲里的 Ingress 相关题目已大幅缩减。

新版 CKA 覆盖的核心域包括：etcd 备份与恢复、RBAC 权限配置、kubeadm 集群安装与升级、**Gateway / HTTPRoute 创建与 502 排错**、NetworkPolicy 隔离、以及 Kubernetes 故障排查。考试仍是纯命令行操作环境，没有选择题，2 小时内完成约 17 个任务。

**关键变化一览**：

- 考试版本：当前基于 Kubernetes v1.34（考场会在 K8s 新版发布后 4-8 周内跟进升级）
- 通过分：**66%**（旧版很多备考指南写的是 74% 或 75%，已过时，请务必注意）
- 费用：$445（仅考试）/ $625（考试 + THRIVE-ONE 年度订阅，含大量配套课程）/ $645（考试 + LFS258 Kubernetes Fundamentals 课程）
- 新增必练技能：`kubectl` 创建 Gateway 资源、挂 HTTPRoute、配置 TLS 策略、在两个 Service 间做流量分割

对正在备考 CKA 的同学：**2025 年之前的备考资料不可再直接套用**，Gateway API 部分必须补充；killer.sh 模拟环境已更新至 v1.34 题目，建议在正式考前至少刷一遍。

> 来源：[Linux Foundation CKA 认证页](https://training.linuxfoundation.org/certification/certified-kubernetes-administrator-cka/) · [DEV.to CKA 2026 实考报告](https://dev.to/suzuki0430/cka-certified-kubernetes-administrator-exam-report-2026-dont-rely-on-old-guides-mastering-the-534m) · [GitHub CKA 2026 Study Guide](https://github.com/techwithmohamed/CKA-Certified-Kubernetes-Administrator)

### 2. Google Professional Cloud Security Engineer 5月考题大更新：Assured Workloads + Confidential Computing 正式入考

**一句话**：Google Cloud 安全工程师认证 5 月迎来新一轮题库更新，新引入 Assured Workloads、Confidential Computing 两大合规与数据保护模块，零信任架构权重明显提升。

Google Professional Cloud Security Engineer 是 GCP 四大 Professional 级认证之一，侧重云原生安全架构设计。本轮 5 月更新带来了以下具体考点变化：

**新增或权重上调的考点**：
- **Assured Workloads**：为受监管行业（金融、政府、医疗）设计的 GCP 合规框架，考点包括如何配置工作负载边界、限制数据驻留地（data residency）和人员访问
- **Confidential Computing**：Google Cloud 的机密计算能力（基于 AMD SEV），考点涉及 Confidential VMs / Confidential GKE Nodes 的使用场景与限制
- **零信任 + Context-Aware Access**：BeyondCorp Enterprise 架构、访问级别（Access Level）配置、基于设备状态和用户身份的动态访问控制权重提升
- **威胁检测与事件响应**：Cloud Security Command Center（SCC）Premium、Chronicle SIEM 集成、自动化响应 Playbook 内容扩充

**对在考或计划备考的同学**：旧版备考资料里对 Assured Workloads 的覆盖往往仅一两道例题，新版考试中该模块很可能有 3-5 道题。建议重点看官方文档中 Assured Workloads 配置流程，以及 Confidential VMs 的限制条件（如不支持 Live Migration）。

> 来源：[Professional Cloud Security Engineer 考题更新（May 2026）](https://certempire.com/exam/professional-cloud-security-engineer-exam-questions/)

### 3. Google Get Certified 2026 Express Path：已持 AWS 或 Azure 证书可走 6 周速成通道，含免费 GCP 正式考试券

**一句话**：Google 的 Get Certified 2026 官方培训计划新增 6 周 Express Path，专为 AWS / Azure 持证者设计，结业后赠一张免费 GCP Professional Cloud Architect 正式考试券（市价 $200+）。

Google Cloud 的 [Get Certified 2026 计划](https://developers.google.com/program/gear/getcertified) 是面向 Google Cloud 企业客户员工的免费认证培训项目。今年新增的 **Express Path** 是其中最值得关注的变化：

- **时长**：6 周（vs. 标准通道 9 周）
- **目标人群**：已持 AWS 或 Azure 认证的多云从业者
- **课程内容**：Google 专家带教 + GCP 真实环境实操 Lab（不是模拟器）+ 专属技术导师 1-on-1
- **结业奖励**：Professional Cloud Architect（PCA）正式考试券一张，费用由 Google 承担
- **参加条件**：需以企业邮箱验证为 Google Cloud 客户的员工（个人账号不符合）
- **批次安排**：2026 年 4 月至 12 月滚动开放，每批名额有限，先报先得

如果你所在的公司是 GCP 用户，且手上已有 AWS SAA / AZ-900 以上级别的证书，这条通道可以显著压缩备考 GCP PCA 的时间成本，同时节省约 $200+ 的考试费。值得向公司 IT 团队或培训负责人推荐申请。

> 来源：[Google GEAR Get Certified 计划官页](https://developers.google.com/program/gear/getcertified)

### 4. AWS 推出双微认证徽章：Application Networking + Incident Response，无需付费订阅即可免费领

**一句话**：AWS 在 2026 年 3 月上线两枚新微认证（Microcredential），聚焦网络交付优化和安全应急响应，任何 AWS 账号无需 Skill Builder 付费订阅即可参加。

AWS Microcredential 是 AWS 在常规认证体系之外推出的轻量级能力徽章，通过在线测评后可获得 Credly 数字徽章，适合在 LinkedIn 上展示某一具体技能。本轮 3 月上线的两枚：

**① AWS Application Networking Demonstrated**
- 考核核心：应用交付策略（Load Balancer 类型选择与配置、Application-layer 路由规则）、性能优化（CDN、缓存策略、延迟排查）、现代应用架构实现（Microservices 流量管理、API Gateway 接入）
- 适合人群：有 AWS 网络基础的开发者或运维工程师，SAA-C03 过了想继续横向扩展的人

**② AWS Incident Response Demonstrated**
- 考核核心：AWS 安全事件识别（GuardDuty 告警判读）、隔离与限制影响（Security Group / IAM 快速锁定）、取证与根因分析、事件报告与恢复流程
- 适合人群：正在备考 SCS-C02（Security Specialty）或 SOC/云安全方向的工程师

两枚微认证均可在 [AWS Skill Builder](https://skillbuilder.aws/) 直接搜索名称找到，**无需订阅，免费参加**。完成评估后徽章通过 Credly 下发。

> 来源：[AWS Training & Certification 3月新增内容公告](https://aws.amazon.com/blogs/training-and-certification/march-2026-new-offerings/)

## 二、免费 / 限免课程

### 1. AWS Skill Builder：SOA-C03 + AIP-C01 官方练习题（日/韩/简中）全部免费

**适合**：正在备考 AWS CloudOps Engineer（SOA-C03）或 Generative AI Developer Pro（AIP-C01）的中日韩语言学员。
**链接**：[skillbuilder.aws](https://skillbuilder.aws/)

AWS 在 3 月更新中同步上线了以下免费官方备考资料：

- **SOA-C03**：Official Practice Exam（含评分与解析），中文版已上线
- **AIP-C01**：Official Practice Question Set + Official Pretest，均提供日语、韩语、简体中文版本

这些练习题集是官方出品（不是第三方），题目结构与真实考试格式完全一致，对拼中文考场的同学尤其有用。在 Skill Builder 里搜索「SOA-C03 Official Practice」或「AIP-C01 Practice Question Set」即可找到，无需付费订阅即可使用。

SOA-C03（CloudOps Engineer Associate）是 AWS 少数几个覆盖真实运维操作的认证，题目会考 CloudWatch 告警配置、Systems Manager Automation、OpsWorks 和 Config Rules 等实战内容，有运维经验背景的同学上手会比纯开发背景快很多。

## 三、JR Academy 学员行动建议

1. **正在备考 CKA 的同学，本周立刻更新备考资料**：找一套覆盖 Kubernetes v1.34 + Gateway API 的题库（killer.sh 已更新），今晚用 `kubectl apply -f gateway.yaml` 练习 HTTPRoute 创建和 502 排错；不要用 2024 年以前的备考资料，通过分已从 74% 降至 66%，但新考点不掌握照样失分。

2. **已持 AWS SAA / AZ-900 证书的同学，本周去申请 Google Get Certified Express Path**：用公司企业邮箱在 [developers.google.com/program/gear/getcertified](https://developers.google.com/program/gear/getcertified) 提交申请，6 周课程 + 免费 PCA 考试券，相当于省了 $200+ 的考试费，且 GCP PCA 在澳洲 AWS/Azure 主导市场里是差异化竞争优势。
