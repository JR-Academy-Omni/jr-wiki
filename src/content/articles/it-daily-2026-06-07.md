## 一、认证动态

### 1. AI Skills Fest 明日开幕：完成一条 Playlist 换任意微软 / GitHub 考试券

**一句话**：Microsoft AI Skills Fest（6 月 8-12 日）明天正式开幕，完成 AI Skills Navigator 上的一条 featured playlist 并主动提交领券表单，即可获得一张 100% 免费的微软或 GitHub 认证考试券，最晚须在 **2026 年 8 月 11 日 UTC** 前使用。

这件事在 06-04、06-05 日报里都提过，今天再说是因为：**明天是五天窗口的第一天，准备的最后时机是今晚。**

具体操作流程：

1. 打开 [aiskillsnavigator.microsoft.com](https://aiskillsnavigator.microsoft.com/events/AISF2026)，用微软账号登录
2. 在 AI Skills Fest 2026 活动页选一条 featured playlist（不要手动增减模块——改动任何一个模块会打断追踪机制，导致无法领券）
3. 把整条 playlist 按顺序刷完，每个模块标记 complete
4. **完成后当场在 dashboard 填写领券表单**——表单会动态出现在你的 event dashboard，不会自动发券，不填表单 = 没有券
5. 从 **6 月 16 日**起微软陆续以邮件方式发送券码；60 天有效期，**到期日 2026 年 8 月 11 日 UTC**

关键限制：

- **1 人 1 张**：完成 5 条 playlist 也只有 1 张券，不叠加
- 券适用范围极广——几乎覆盖所有微软认证考试（含 beta 阶段），也包含 GitHub 认证（如 GitHub Copilot、GitHub Actions 相关考试）
- 除考试券外，注册参与即自动进入抽奖池，28 个额外大奖总价值 $25,000，含 Microsoft Ignite 2026 VIP 行程名额

对 JR Academy 的学员来说，这张券可以直接抵掉 AZ-900（$165）、AI-103（Associate 级 $165）、PL-300（$165）、MS-900（$165）甚至 SC-900 的考试费。已经备考中的同学，今晚选好 playlist，明天 9 点开刷，一条基础 playlist 3-4 小时搞定，不占备考时间。

### 2. Azure AI-900 6 月 30 日退役：距今 23 天，AI-901 本月接棒

**一句话**：微软最入门的 AI 认证 AI-900（Azure AI Fundamentals）将于 **2026 年 6 月 30 日**正式退役，接替者 AI-901（同名，内容已更新）预计本月 GA，现有持证人证书继续有效至到期日，但**不会自动转换为 AI-901**。

AI-900 是数以万计 IT 人的「第一张微软证书」，入门门槛低——不要求编程背景，只考 Azure AI 概念（机器学习原理、Computer Vision、自然语言处理、Azure AI 服务概览）。退役后的对比：

| 项目 | AI-900（退役） | AI-901（接棒） |
|------|------------|------------|
| 正式名称 | Azure AI Fundamentals | Azure AI Fundamentals（新版） |
| 退役 / 上线日期 | **2026 年 6 月 30 日退役** | 预计 2026 年 6 月 GA |
| Beta 阶段 | 已结束 | 2026 年 4 月已开放 |
| 考试代码 | AI-900 | AI-901 |
| 考试定价 | USD $165（Fundamentals 级） | 预计同级别定价，GA 后官方确认 |

**考试内容变化方向**：AI-901 题库更新为微软当前 AI 产品线，涵盖 Azure OpenAI Service、Copilot Studio、Responsible AI 框架等 2025-2026 年的新服务，比 AI-900 更贴合当下企业用 Azure 做 AI 的实际场景。

**三种人群怎么做**：
- **持有 AI-900、证书尚在有效期**：不用补考，证书继续有效，等 AI-901 稳定后再评估要不要考新版
- **AI-900 备考接近尾声**：6 月 29 日前报考 AI-900 完全来得及，考完就直接过了
- **刚开始学、还没报名**：直接等 AI-901 上线，新题库与现在的 Azure AI 产品更匹配，面试考察点也更新

### 3. AB-250 + AZ-802：微软 6 月两张新认证 Beta 同步开放

**一句话**：微软本月（June 2026）同时开放两张新认证的 Beta 报名和学习材料——AB-250（Dynamics 365 Contact Center AI Engineer Associate）与 AZ-802（Windows Server Hybrid Administrator Associate），两者正式 GA 考试均预计 2026 年 8 月上线。

**AB-250 — Dynamics 365 Contact Center AI Engineer Associate**

全新的联络中心 AI 工程师认证，瞄准 CCaaS（Contact Center as a Service）场景：

- **考域重点**：使用 Dynamics 365 Contact Center 设计和运行现代联络中心解决方案，包括构建 AI Agent 驱动的自主服务（autonomous agents）、路由配置、Omnichannel 集成
- **目标人群**：负责搭建或维护 Dynamics 365 CC 平台的工程师，以及在客服 AI 落地场景工作的开发者
- **Beta 状态**：报名和培训材料 June 2026 已上线 Microsoft Learn

**AZ-802 — Windows Server Hybrid Administrator Associate**

对现有 AZ-800 + AZ-801 双考模式的整合：

- **背景**：取得 Windows Server Hybrid Administrator Associate 认证此前需分别通过 AZ-800（Core Infrastructure Part I）和 AZ-801（Part II）两场考试，学习和考试周期长
- **AZ-802 做了什么**：合并为单张 Associate 级认证，涵盖 Azure Arc 混合管理、Azure Stack HCI、混合 Windows Server 网络、身份、存储管理
- **目标人群**：负责 Azure + 本地混合基础设施的系统管理员和云工程师
- **Beta 状态**：Beta 报名 + 培训材料 June 2026 已上线；正式 GA 考试预计 2026 年 8 月

**关于 Beta 折扣**：微软 Beta 考试历史上通常提供折扣码（一般为 8 折左右），具体折扣码会由微软通过 Microsoft Learn 博客和培训合作伙伴渠道发布。如果有意参加 Beta，关注 [Microsoft Learn 博客](https://techcommunity.microsoft.com/category/microsoftlearn) 动态。

> 来源：[mscertquiz.com — New Microsoft Beta Exams 2026](https://mscertquiz.com/blog/new-microsoft-beta-exams-2026)、[certempire.com — Microsoft AB Series Certifications Guide](https://certempire.com/microsoft-ab-series-certifications/)

### 4. AWS 考完别忘领的两张隐藏福利：50% 折扣券 + 免费练习题

**一句话**：通过任意 AWS 认证考试后，AWS 会在几个工作日内邮件发送**50% 下一考折扣券**（有效期 12 个月）和**一张免费官方 Practice Exam 券**——但不主动去领就会默默过期。

这不是新政策，但 JR Academy 课群里每隔几周都有人问怎么省考试费，最简单的答案就是：**上一张证的通过本身就附带了下一张的折扣**。

- **50% 折扣券**：登录 [AWS Certification Account](https://www.aws.training/certification) → Benefits 页面领取，适用任何 AWS 认证考试，不可转让，不可与其他促销叠加
- **免费 Practice Exam**：官方题目 20-40 道，带详细解析，原价 $40，通过后免费领取，在 AWS Skill Builder 激活使用
- **典型节省路径**：通过 SAA-C03（原价 $150）→ 领 50% 券 → 报 SAA-Pro（原价 $300 → $150）→ 通过后再领券 → 报 Specialty（$300 → $150）

注意：每次考试对应一张折扣券，下一次通过后再领新的，形成链式节省。

> 来源：[tutorialsdojo.com — Free AWS Exam Retake and 25% OFF](https://tutorialsdojo.com/free-aws-exam-retake-and-25-off-any-aws-certification-exam-voucher/)、[examcert.app — AWS Certification Vouchers & Discounts 2026](https://www.examcert.app/blog/aws-certification-voucher-discount-guide/)

### 1. AWS Skill Builder Lab Maker：AI 生成即时个人化实验室

**适合**：正备考 AIP-C01 / MLS-C01 / SAA-C03，或者想快速跑通 Bedrock 和 Lambda 动手场景的工程师
**链接**：[skillbuilder.aws/learn/42V123SUXH/lab-maker](https://skillbuilder.aws/learn/42V123SUXH/lab-maker/22W4SWWYHA)
**需要**：AWS Skill Builder Team 订阅（企业购买）

2026 年 5 月，AWS 在 Skill Builder 上线了 Lab Maker——输入自然语言描述你想 build 的场景，系统自动生成分步骤指引，直接叠加在模拟 AWS Management Console 的界面上，不需要自己开 AWS 真实账号：

- **支持服务**：Amazon Bedrock、AWS Lambda、Amazon S3、Amazon DynamoDB 等主流服务
- **适合场景**：考前验证知识点（「我要在 Bedrock 上跑一个 RAG pipeline，具体步骤是？」）、练手新服务（不想开真实账号付费）
- **限制**：仅限 Team 订阅（企业授权），个人 Individual 订阅不含此功能

如果你的公司有 AWS Skill Builder 企业授权，现在就可以进入 Lab Maker 试用。

### 2. AWS + JetBrains 两条新技能路径：在 AWS 上构建 GenAI 应用

**适合**：Python 开发者、正在备考 AIP-C01 或 MLS-C01 的候选人
**平台**：AWS Skill Builder（视频和阅读模块免费；动手 lab 需订阅）
**链接**：[skillbuilder.aws](https://skillbuilder.aws/)

AWS 和 JetBrains 联名推出两条新 Skill Path：

1. **Build and Deploy Custom LLMs with Python and AWS** — 从微调到生产部署，Python 为主语言，对接 Amazon Bedrock 和 SageMaker
2. **From ML to GenAI: Building Real-World Apps on AWS** — 从传统机器学习迁移到生成式 AI 应用落地，含 Bedrock agent 实战案例

两条路径的视频和阅读内容免费访问，动手 Builder Labs 部分需要 Individual 订阅（约 $29/月 或 $449/年）或企业授权。如果只是跟着看视频掌握架构思路，0 成本可以完成。

### 3. Google Cloud Innovators：每月 35 个免费 Lab Credits

**适合**：备考 GCP ACE（6 月 30 日换新题库）、Professional Data Engineer 或 ML Engineer 的学员
**注册**：[cloud.google.com/innovators](https://cloud.google.com/innovators)（免费）

加入 Google Cloud Innovators 计划后，每月自动获得 **35 个 Google Cloud Skills Boost lab credits**，可用于兑换动手实验（标准 lab 通常消耗 1-5 credits，35 credits 约能跑 7-10 个实战 lab）。

2026 年课程库已完成更新，重点补充了 **Gemini Enterprise Agent Platform**（原 Vertex AI Agent Builder 重品牌）相关内容——这与 6 月 30 日上线的 GCP ACE 新版题库考域直接对应：ACE 新版加入了 Gemini Enterprise Agent Platform 内容，用 Innovators credits 练 lab，练完正好备考。

- Credits 每月 1 日重置，不滚入下月
- 大学生可额外申请 200 credits，教师可为班级申请 5,000 credits
- Skill Badge 课程完成后可获得数字徽章，可分享至 LinkedIn

> 来源：[Google Cloud — Boost Your Cloud Skills](https://cloud.google.com/resources/boost-your-cloud-skills-with-google)

### 4. DeepLearning.AI 新短课：Fast & Efficient LLM Inference with vLLM

**适合**：ML 工程师、想优化 LLM 推理速度的后端工程师
**平台**：DeepLearning.AI Short Courses（免费，注册即学）
**链接**：[deeplearning.ai/courses](https://www.deeplearning.ai/courses/)

DeepLearning.AI 最新上线的短课聚焦 **vLLM**——目前最主流的开源 LLM 推理加速框架，被 Mistral、Meta、百川等大量生产环境采用：

- **vLLM 解决什么问题**：通过 PagedAttention 技术大幅减少 GPU 显存碎片、支持连续批处理（continuous batching），相比 naive 推理吞吐量提升数倍
- **课程预期内容**：vLLM 架构原理、本地部署实战、与 OpenAI 兼容 API 对接、性能调优
- **时长**：DeepLearning.AI Short Courses 系列通常 1-2 小时完成
- **费用**：免费（注册 deeplearning.ai 账号即可学习，无需付费）

如果你的工作涉及把开源大模型跑在 GPU 上对外提供服务，vLLM 是 2026 年不可绕过的工具之一。

> 来源：[DeepLearning.AI Courses](https://www.deeplearning.ai/courses/)
