## 一、认证动态

### 1. SC-500 Beta 昨日开考：AZ-500 的接班人正式登场

**一句话**：微软新一代 Cloud and AI Security Engineer（SC-500）5月15日进入 Beta 测试，GA 版本预计 2026年7月上线；现有 AZ-500 持证者注意——证书**不会自动转换**，需另行报考。

SC-500 并非 AZ-500 的简单改版，而是一次方向性调整。AZ-500 覆盖传统的 Azure 基础设施安全（网络、身份、Key Vault、Defender）；SC-500 在此之上新增了 **AI 系统与模型安全**，包括保护 Azure OpenAI 部署、AI Pipeline 的访问控制、模型注入攻击防护等场景。考察的职责角色从"Azure 安全工程师"扩展为"Cloud and AI Security Engineer"。

**关键时间线**：

- Beta 考试：2026年5月15日起接受报名（Beta 价格通常打五折，约 $75 USD）
- 正式版（GA）：预计 2026年7月上线
- AZ-500 退役：**2026年9月30日**（与 AZ-204 同批次不同，AZ-500 给了更长的缓冲期）

**建议**：目前正在备考 AZ-500 的同学，如果能在 9月底前考完，拿 AZ-500 也完全有效（证书有效期通常一年可续期）。如果刚起步，直接备考 SC-500 更有前瞻性。Beta 阶段成绩批改周期约 8 周。

> 来源：[Microsoft Certification Roadmap 2026 – Whizlabs](https://www.whizlabs.com/blog/microsofts-next-generation-ai-certifications/) · [AZ-500 vs SC-500 – CertEmpire](https://certempire.com/az-500-vs-sc-500/)

### 2. AZ-204 7月31日退场：Azure 开发者认证改走 AI 路线

**一句话**：AZ-204（Developing Solutions for Microsoft Azure）将于 **2026年7月31日**正式退役，接班的 AI-200（Azure AI Cloud Developer Associate）把 AI 服务集成列为核心考点。

AZ-204 过去四年是 Azure 开发者的入场券——Cosmos DB、Service Bus、App Service 部署、函数计算。AI-200 在这些基础上叠加了 Azure AI Foundry、Azure OpenAI SDK 集成、Semantic Kernel 使用、以及智能 Agent 的设计与部署。

同一批次，微软还对四张基础级考试（AZ-900、AI-900、SC-900、MS-900）的题库做了更新，**12%–18% 的题目现在覆盖 AI 主题**，即使是非 AI 专向的考试也不再能绕开这部分内容。

| 现有证书 | 退役时间 | 接替证书 |
|---------|---------|---------|
| AZ-204 Azure Developer Associate | 2026-07-31 | AI-200 Azure AI Cloud Developer Associate |
| AZ-500 Azure Security Engineer Associate | 2026-09-30 | SC-500 Cloud and AI Security Engineer |

**建议**：如果已在备考 AZ-204，7月31日前有两个多月的窗口期，完全来得及；新人建议直接看 AI-200 学习路径，Microsoft Learn 上对应课程已在更新中。

> 来源：[Microsoft Certification Guide 2026 – lived365](https://www.lived365.com/post/microsoft-certification-guide-2026-newexams-retirements-aiskills-and-careerimpact) · [Microsoft Exam Changes 2026 – mscertquiz](https://mscertquiz.com/blog/microsoft-certification-exam-changes-2026)

### 3. AWS SysOps 正式更名为 CloudOps Engineer Associate（SOA-C03）

**一句话**：AWS Certified SysOps Administrator 已于 2025年9月30日改名为 **AWS Certified CloudOps Engineer – Associate（SOA-C03）**，考纲同步更新，**容器运维首次列入必考范围**。

这不只是改名——SOA-C03 相较于前任 SOA-C02，有三处实质性变化：

1. **容器管理进入考纲**：ECS、EKS 的监控与运维、容器日志收集等在 SOA-C03 首次出现
2. **多账户、多 Region 架构权重上升**：Organizations、Control Tower、SCP 的实操题增多
3. **IaC 自动化加重**：CloudFormation、Systems Manager Automation 占比提升

考试格式：65 题（50 道计分 + 15 道不计分），130 分钟，Pearson VUE 或在线监考，通过分 720/1000，提供简体中文、英语、日语、韩语。

如果你持有旧版 SysOps Administrator 证书，**证书名称会自动更新**为 CloudOps Engineer，不用重考。

> 来源：[AWS Certified CloudOps Engineer – Associate 官方页](https://aws.amazon.com/certification/certified-cloudops-engineer-associate/) · [Coming Soon to AWS Certification](https://aws.amazon.com/certification/coming-soon/)

### 4. AWS 全面 AI 化：SAP-C02 / DOP-C02 6月起加入 Bedrock 场景题

**一句话**：从 6月1日起，AWS Professional 级别考试 SAP-C02（Solutions Architect Professional）和 DOP-C02（DevOps Engineer Professional）将新增 Amazon Q、Amazon Bedrock AI 治理场景题，进一步拉高这两张证的难度门槛。

这一变动意味着备考 Professional 级别的同学不能再把 AI 服务当作"了解"档处理——Bedrock 的 guardrails 配置、Amazon Q Business 的部署限制、AI 工作负载的 IAM 权限设计都可能出现在题目里。

建议在 6月前已注册考试的候选人认真检查当前考试蓝图（Exam Guide）版本，部分培训资料可能还没跟上更新。

> 来源：[AWS Certification Roadmap 2026 – k21academy](https://k21academy.com/aws-cloud/aws-certification-roadmap/)

## 二、免费 / 限免课程

### 1. AWS AI & ML Scholars 2026 — 10 万个免费席位，6月24日截止

**适合**：零基础或初级 AI 学习者，想拿 AWS Certified AI Practitioner（AIF-C01）入场券的人
**链接**：[https://aws.amazon.com/about-aws/our-impact/scholars/](https://aws.amazon.com/about-aws/our-impact/scholars/) 或 [Udacity 报名入口](https://www.udacity.com/scholarships/aws-ai-ml-scholars)

AWS 与 Udacity 联合开设，全球 10 万个免费席位。程序分两个阶段：

- **Challenge 阶段**（3月24日–6月24日）：免费课程 + AWS Skill Builder 三个月会员，完成者获结业证书，并直接为 AIF-C01（AWS Certified AI Practitioner）备考。使用的工具包括 PartyRock、Amazon Bedrock、Amazon Q。
- **Udacity Nanodegree**：Challenge 阶段排名前 4,500 名自动晋级，完全免费，三条方向：**AI Programmer**、**Agentic AI Business Professional**、**Agent Developer**。

Challenge 阶段已于 3月24日开始，现在报名仍可参与剩余课程（截止 6月24日）。即使错过 Nanodegree 选拔，仅 Challenge 证书 + AI Practitioner 认证的组合在简历上已有不少含金量。

### 4. Microsoft Learn — 3,200+ 免费课程，Applied Skills 免费实操评估

**适合**：备考 Azure / Microsoft 365 / Power Platform 认证的所有层级学员
**链接**：[https://learn.microsoft.com/](https://learn.microsoft.com/)

Microsoft Learn 是微软官方免费学习平台，目前提供超过 3,600 门课程，其中 3,200+ 完全免费（包括 Learning Path、模块、沙箱实操环境）。值得特别关注的是 **Applied Skills（应用技能）**项目：每张 Applied Skills 徽章对应一个 45–60 分钟的真实沙箱评估，通过即可获得数字徽章，无考试费用，适合在备考正式认证前验证实操能力。

当前已上线的 Applied Skills 包括：Create an AI Agent（Azure AI Foundry）、Configure Copilot in Teams、Implement Azure Functions 等，面向 SC-500 备考的 AI 安全相关模块预计 5–6 月间陆续上线。

## 三、JR Academy 学员行动建议

1. **SC-500 Beta 窗口期抓住**：Beta 考试通常以五折价报名（约 $75 USD），5月15日已开放，Beta 阶段成绩约 8 周后出。有 AZ-500 基础的同学这周就可以去 Microsoft Learn 查 SC-500 Exam Skills Outline，对比差距。

2. **AZ-204 持证者提前看 AI-200**：AI-200 的核心新增内容集中在 Azure AI Foundry SDK 和 Semantic Kernel，这周可以在 Microsoft Learn 搜索"AI-200"把免费模块先刷一遍，判断备考跨度有多大。

3. **AWS AI & ML Scholars 截止 6月24日**：还没申请的同学去 [udacity.com/scholarships/aws-ai-ml-scholars](https://www.udacity.com/scholarships/aws-ai-ml-scholars) 报名，Challenge 阶段剩余约 5 周，完成后有 AWS Skill Builder 会员 + 结业证书，进入 Nanodegree 更直接拿 AIF-C01 备考资源。

4. **备考 SAP-C02 / DOP-C02 的同学**：6月1日前尽量完成报名锁定旧题库窗口；如果来不及，立刻把 Amazon Bedrock 和 Amazon Q 的官方 FAQs 加入备考清单，重点看 guardrails 和 IAM 权限。

5. **Coursera Plus 40% 折扣没有截止日期压力**（有效至 2027年2月），但如果你本月已确定要系统备考，现在订月付版本 $35.40/月，3 个月刷完一个 Professional Certificate 再取消，综合成本约 AUD $165，比单买一门正规备考课便宜很多。
