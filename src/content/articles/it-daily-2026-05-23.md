## 一、认证动态

### 1. SC-730 Beta：Cybersecurity Business Professional 80折，5月27日截止（仅剩4天）

**一句话**：前300名用代码 `SC730Eclipse` 报考 SC-730 Beta，折后约 **$33 USD**，截止5月27日——今天不动手很可能名额就满了。

微软于5月初正式发布 **Microsoft Certified: Cybersecurity Business Professional** 认证，对应考试代码 **SC-730**（Beta 阶段）。这张证书的定位非常明确：**面向职场非纯技术人员**——项目经理、业务分析师、合规官、IT 运营人员，任何在工作中需要理解并参与应对安全风险的角色。

考试范围包括四个核心域：
- 识别常见网络安全威胁（钓鱼、社会工程学、勒索软件）
- 应用日常安全最佳实践（MFA、最小权限、零信任基础概念）
- 保护敏感数据（数据分类、数据丢失预防策略）
- 安全事件响应（上报流程、业务连续性基础）

**折扣操作步骤**：进入 [Pearson VUE 官网](https://home.pearsonvue.com/microsoft) → 搜索 SC-730 → 选考场或在线监考 → 付款环节输入代码 `SC730Eclipse` → 系统自动减扣80%。标准价约 $165 USD，折后约 $33 USD。前300名先到先得，不适用于土耳其、巴基斯坦、印度和中国考生。

Beta 考试的优缺点：价格便宜，但成绩公布比正式考试晚约6-8周（等微软完成批量阅卷），期间不知道是否通过。如果不急着用证书，Beta 是性价比最高的入场方式。

To receive the discount, when you register for the exam and are prompted for payment, use SC730Eclipse." — 来源：[New Microsoft Certified: Cybersecurity Business Professional Certification — Microsoft Community Hub](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-cybersecurity-business-professional-certification/4494124)

### 2. SC-500 Beta：Cloud and AI Security Engineer 接替 AZ-500，代码 VistaSC500 截止6月8日

**一句话**：AZ-500 将于2026年8月31日退役，SC-500 是官方接替考试；Beta 期间前300名用 `VistaSC500` 享80%折扣，截止6月8日。

微软宣布 **Microsoft Certified: Cloud and AI Security Engineer Associate** 认证（考试代码 **SC-500**）正式上线 Beta，这是 AZ-500（Azure Security Technologies）的直接继任者。

**为什么更名而不是简单升级？** SC-500 的命名从 "AZ-" 前缀换到 "SC-" 前缀，标志着这张证书不再局限于 Azure 平台操作，而是扩展到更广的安全工程师角色定义：

- **保留内容**：Azure 身份安全（Entra ID、条件访问）、网络安全（NSG、防火墙、DDoS）、存储和密钥管理（Key Vault）
- **新增内容**：
  - AI 工作负载安全（Azure AI Studio 部署的模型防护、Prompt Injection 防御）
  - 安全容器和无服务器环境（AKS 安全配置、Azure Functions 权限管理）
  - 分布式可观测性（安全日志集中化、Sentinel 用例）

**关键时间线**：
- SC-500 Beta：截止2026年6月8日，折扣码 `VistaSC500`，前300名享80%折扣（约 $33 USD）
- SC-500 正式版（GA）：预计2026年7月
- AZ-500 考试和续订退役：**2026年8月31日**

持有 AZ-500 的学员：证书在到期日前仍然有效，不需要立即重考。建议在7月 GA 版上线后直接考 SC-500 正式版（而非 Beta），让成绩快点出来，方便下半年简历用。

> sourceQuote："The first 300 people who take Exam SC-500 (beta) on or before June 8, 2026, can get 80% off by using code VistaSC500 when registering for the exam and prompted for payment." — 来源：[New Microsoft Certified: Cloud and AI Security Engineer Associate Certification — Microsoft Community Hub](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-cloud-and-ai-security-engineer-associate-certification/4494117)

> sourceQuote（退役时间）："AZ-500 exam and renewal assessments will retire on August 31, 2026." — 来源：[AZ-500 vs SC-500: Which Security Exam to Take in 2026?](https://certempire.com/az-500-vs-sc-500/)

### 3. CKA 考纲5月10日更新至 Kubernetes v1.35，Gateway API 和 Native Sidecars 正式入考

**一句话**：CNCF 于2026年5月10日将 CKA 考纲更新至 Kubernetes v1.35，Gateway API GA 版本和 Native Sidecar Containers 已成为正式考点，备考资料需同步更新。

Linux Foundation / CNCF 的 **Certified Kubernetes Administrator (CKA)** 认证按季度与 Kubernetes 发版节奏同步更新，最新一次于 **2026年5月10日** 完成，对应 Kubernetes v1.35。

**本次更新的核心变化**：

**Gateway API（正式 GA）**：Kubernetes v1.31 将 Gateway API 稳定为 GA，CKA 2026年5月版正式把 Gateway API 纳入考点，替代或补充原来 Ingress 相关的题目。考生需要掌握 `GatewayClass`、`Gateway`、`HTTPRoute` 资源的创建和调试。旧版 Ingress 题目不会完全消失，但出题比例下降。

**Native Sidecar Containers**：Kubernetes v1.29 引入、v1.33 稳定的 init container sidecar 模式（通过 `initContainers` 中设置 `restartPolicy: Always`）已成为考点。这直接影响日志采集和服务网格场景题的解法。

**考试基本信息**（2026年5月更新后）：
- 时长：2小时，纯命令行实操（无选择题，无填空题）
- 费用：**$445 USD**（含一次免费补考）
- Kubernetes 版本：v1.35
- 通过分数：66分/100

> sourceQuote："The CNCF CKA exam was last updated on May 10, 2026. The CKA exam as of 2026 is based on Kubernetes v1.35, with quarterly exam updates planned to match Kubernetes releases. Recent updates include native sidecars and Gateway API becoming GA, which appeared on recent exams." — 来源：[CKA Exam Study Guide 2026 — techwithmohamed.com](https://techwithmohamed.com/blog/cka-exam-study/) | [CNCF CKA Program Changes — Linux Foundation](https://training.linuxfoundation.org/certified-kubernetes-administrator-cka-program-changes/)
