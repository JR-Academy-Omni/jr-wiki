## 一、认证动态

### 1. DP-750 正式 GA：微软 × Databricks 联合认证，5 月全面上线

**一句话**：Microsoft Certified: Azure Databricks Data Engineer Associate（考试代码 DP-750）已于 2026 年 5 月正式 GA，是微软首张专攻 Azure Databricks 的数据工程师认证。

考试正式名称为 **Exam DP-750: Implementing Data Engineering Solutions Using Azure Databricks**。今年 3 月底 beta 开放时，微软官方 @MicrosoftLearn 发推宣布：「前 300 名在 4 月 2 日前报名可享 80% 折扣（代码 DP750Deltona）」，首日名额据报数小时内告罄。Beta 窗口现已关闭，目前走标准定价。

考试考察三大能力：
- 用 **SQL 和 Python** 做数据摄取、转换和建模
- 在 Azure Databricks 上构建**生产级数据管道**
- 用 **Git 工作流**落地 SDLC（软件开发生命周期）实践

这张证书与 Databricks 官方的 Certified Data Engineer Associate 有本质区别：DP-750 考的是**在 Azure 云上用 Databricks**，偏云架构 + 平台集成；Databricks 官方认证更纯粹考核 Databricks 平台操作本身（Delta Lake / DLT / Unity Catalog）。两证可以叠加，对在澳洲做 Azure Data 方向的同学尤其值得关注。

> 来源：[Microsoft Learn on X @MicrosoftLearn](https://x.com/MicrosoftLearn/status/2032124460606673025) · [Microsoft Community Hub — 新认证公告](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-databricks-data-engineer-associate-certification/4494114)

### 2. Databricks Certified Data Engineer Associate — 5 月 4 日大改版

**一句话**：Databricks 官方数据工程师认证（非微软版）5 月 4 日发布新考试大纲，45 题 / 90 分钟 / USD $200，场景题大幅加码，Streaming + Delta Lake 决策能力成为考核重心。

新版考试大纲分五个考域：

| 考域 | 占比 |
|------|------|
| Databricks Intelligence Platform | 10% |
| Development & Ingestion | 30% |
| Data Processing & Transformations | 31% |
| Productionizing Data Pipelines | 18% |
| Data Governance & Quality | 11% |

开发/摄取 + 处理/转换两块合计占 **61%**，是考试核心。具体变化：

- **场景题升级**：不再只考「Auto Loader 是什么」，而是考「在这个生产场景下，你会用 Auto Loader 还是 COPY INTO，为什么」
- **Streaming vs Batch 决策**：明确要求考生能判断什么时候用 Structured Streaming、什么时候用批处理，以及混合架构怎么设计
- **Medallion 架构要会用**：Bronze → Silver → Gold 必须在场景里落地，不只是背定义
- **性能诊断**：考管道性能优化和成本控制，接近真实生产工作

费用 USD $200，90 分钟，45 道多选题。报名走 Databricks 官网的 Webassessor 平台。注意：5 月 4 日之前的旧版模拟题库参考价值已大幅降低，建议直接对照新版官方 Exam Guide 刷题。

> 来源：[Databricks Certified Data Engineer Associate Exam Guide May 4 2026（Scribd）](https://www.scribd.com/document/1030707656/Databricks-Certified-Data-Engineer-Associate-Exam-Guide-May-4-2026) · [Databricks 认证官网](https://www.databricks.com/learn/certification/data-engineer-associate)
