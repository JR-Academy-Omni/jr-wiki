## 一、认证动态

### 1. 微软"双退休"公告：AZ-204 七月底停考，AZ-500 八月底停考

**一句话**：两张高频考的 Azure 认证同步进入退休倒计时，备考计划现在必须调整。

微软 2026 年上半年推进整体认证体系的 AI 转型，两张核心证进入最后窗口：

**AZ-204（Azure Developer Associate）** → **2026 年 7 月 31 日最后停考**，替代考试为 **AI-200 Azure AI Cloud Developer Associate**，Beta 考试已于 4 月开放。AI-200 在原 AZ-204 Azure 服务开发基础上加入了 AI 模型部署、Azure AI Studio 工作流、Prompt Engineering 集成等新考点，定位做云开发同时接触 AI 应用的工程师。

**AZ-500（Azure Security Engineer Associate）** → **2026 年 8 月 31 日最后停考**，替代考试为 **SC-500 Cloud and AI Security Engineer Associate**，Beta 考试已于 5 月 15 日前后上线。SC-500 在 AZ-500 安全工程基础上扩展了 AI 工作负载安全、模型环境保护、Prompt Injection 防御、AI 供应链安全等新考点，升级幅度不小。

实操方向：
- 已在备 AZ-204 的同学：还有约 65 天窗口，进度过半建议冲刺完考；若刚起步直接转 AI-200。
- 计划新考 Azure 安全方向的：SC-500 Beta 已上线，Beta 阶段通常定价低于正式版（微软 Beta 考试不公示折扣幅度，但参加 Beta 的主要代价是成绩发布延迟 6–8 周）。

> 来源：[Microsoft Certification Retirements 2026 – vladtalkstech.com](https://vladtalkstech.com/microsoft-learning-and-credential-news/microsoft-certification-retirements-2026/) | [AZ-500 vs SC-500 分析 – certempire.com](https://certempire.com/az-500-vs-sc-500/)

### 2. DP-750 Azure Databricks Data Engineer Associate 5 月正式 GA

**一句话**：微软 + Databricks 联合推出的 DP-750 本月正式开考，是目前最新的 Azure 数据工程认证。

DP-750 全称 **Implementing Data Engineering Solutions Using Azure Databricks**，是微软与 Databricks 合作推出的新认证，和此前上线的 AI-300、DP-800 合称微软 2026 年新 AI/Data 证书三连。

**Beta 阶段**（截止 4 月 2 日）前 300 名考生使用代码 `DP750Deltona` 可享 80% 折扣——这个窗口已关，但正式版现已在 Pearson VUE 全面开放。

考试覆盖核心内容：
- Delta Lake 架构与 Lakehouse 设计
- 用 Azure Databricks 构建 ETL / ELT 管道
- 数据安全、治理、性能调优
- 与 Azure Data Factory / Azure Synapse 集成

对在澳洲做数据工程的同学来说，DP-750 + AZ-900 / DP-900 的组合，在金融、零售类公司数据平台岗位上的认可度正在上升，配合 Databricks 自己的 Certified Data Engineer Associate 认证可以构成"双认证"简历亮点。

> 来源：[Microsoft Community Hub DP-750 公告](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-databricks-data-engineer-associate-certification/4494114) | [DP-750 考纲 – Microsoft Learn](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/dp-750)

### 3. AWS Certified Generative AI Developer – Professional 报名通道开放

**一句话**：AWS 最新专业级 AI 认证上线报名，填补 3 月 31 日停考的 MLS-C01 空白。

AWS 今年 3 月宣布 **AWS Certified Machine Learning – Specialty（MLS-C01）于 3 月 31 日正式停考**，同时开放两条接棒路径：

1. **AWS Certified AI Practitioner（AIF-C01）**：入门级，侧重 AI/ML 核心概念 + AWS AI 服务使用，适合非纯技术背景。
2. **AWS Certified Generative AI Developer – Professional**：专业级，报名通道现已开放。考点集中在将 Foundation Models 集成进应用与业务流程，包括 Amazon Bedrock 应用开发、Prompt Engineering 进阶、RAG 架构设计、AI Agent 构建。

原本规划 MLS-C01 的同学只剩新路径可选。对做 AI 应用开发的工程师，这张证在简历上的信号量比旧 ML Specialty 更直接地对应当前市场需求。

> 来源：[AWS Training and Certification Blog – AI 认证扩张公告](https://aws.amazon.com/blogs/training-and-certification/big-news-aws-expands-ai-certification-portfolio-and-updates-security-certification/)

## 二、免费 / 限免课程

### 1. AWS Skill Builder：600+ 免费 AI / 云计算课程

**适合**：想系统补 AWS 基础或 AI 技能的同学，无需任何订阅
**链接**：[https://skillbuilder.aws/](https://skillbuilder.aws/)

AWS Skill Builder 免费层涵盖 600 多门课，其中 AI 方向可直接上手的推荐三门：

- **Foundations of Prompt Engineering**：从基础到进阶 Prompt 技巧，约 4 小时，是备考 AIF-C01 或 Generative AI Developer 的热身。
- **Low-Code Machine Learning on AWS**：用 SageMaker Canvas 跑 ML 不需要写算法代码，约 6 小时，适合数据分析师转 ML 方向。
- **Generative AI Learning Plan for Decision Makers**：三课合集，覆盖生成式 AI 项目规划和落地，面向非技术背景的产品 / 管理岗。

付费的 Skill Builder Individual 订阅（$29/月）包含全套官方考试模拟题，靠近考试时可以按月开通，考完取消。

> 来源：[AWS Skill Builder 官网](https://skillbuilder.aws/) | [AWS AI 学习资源页](https://aws.amazon.com/ai/learn/)

### 2. Google GEAR Get Certified 2026：考试费全免（价值 $200 USD）

**适合**：在 Google Cloud 客户公司任职、有公司 Google Workspace 邮箱的工程师
**链接**：[https://developers.google.com/program/gear/getcertified](https://developers.google.com/program/gear/getcertified)

Google 的 GEAR（Google for Education and Research）Get Certified 计划是目前最直接的 GCP 认证免费通道：

- 完成所有学习里程碑（labs + skill badges）后，获得**考试券一张**，覆盖完整 $200 考试费
- 培训由 Google Cloud 认证讲师每周直播 review，共 6 次 live session，可在线实时提问
- 时间投入约 9 小时/周（自学 labs + 参加 live session）

**关键限制**：须提交公司邮箱验证为 Google Cloud 客户（公司在用 Google Workspace 或 GCP 的），个人 Gmail 不符合资格。注册窗口为 4 周，错过需等下期。

若不满足资格，Google Cloud Skills Boost 的 Individual 订阅（$15/月）包含完整 labs 和 Skill Badges，是自学备考 GCP 认证性价比最高的替代方案。

> 来源：[Google GEAR Get Certified 官方页](https://developers.google.com/program/gear/getcertified) | [Tech Bytes 项目详解](https://techbytes.app/posts/google-cloud-gear-program-free-certification-2026/)

## 三、JR Academy 学员行动建议

1. **持有或在备 AZ-204 的同学**：7 月 31 日停考，剩约 65 天。进度超过 60% 建议冲刺完考；进度不足 30% 直接转备 AI-200，避免学一半换题库。SC-500 Beta 刚开，有意向做 Azure 安全方向的可以现在开始研究考纲变化。

2. **想拿 GCP 认证的同学**：马上进 [Google GEAR Get Certified 官方页](https://developers.google.com/program/gear/getcertified) 查注册窗口是否开放——公司邮箱符合资格就申请，省下 $200 考试费，同时拿到 Google 官方直播辅导。不符合资格的，直接开 Google Cloud Skills Boost 的 $15/月 订阅开刷 labs。

3. **Databricks / 数据工程方向**：DP-750 已 GA，是现在最新的 Azure 数据工程认证。配合 JR Academy 的数据工程课程，2–3 个月备考周期可行。目标澳洲金融、零售类数据平台岗位的同学把这张证加进规划。

4. **准备入手 AWS 新 AI 认证的同学**：先把 AWS Skill Builder 上「Foundations of Prompt Engineering」和「Generative AI Learning Plan for Decision Makers」两个免费课刷完（共约 8 小时），再去官网查 Generative AI Developer Professional 的 exam guide 和正式考试日期。

5. **Kubernetes 方向**：Mega May 购买窗口已关（5 月 24 日结束）。若还在犹豫，现在订阅 [Linux Foundation Newsletter](https://training.linuxfoundation.org) 是第一时间拿 Cyber November 折扣码的最快方式；平时 15–30% 的不定期折扣码通过 Newsletter 发放，不订阅容易错过。
