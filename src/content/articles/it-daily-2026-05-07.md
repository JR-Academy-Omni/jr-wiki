### 2. Databricks Certified Generative AI Engineer Associate 正式上架：2026 年新认证

**一句话**：Databricks 2026 年推出的 GenAI 工程实践认证正式开放，考的不是 Spark 算子，而是你能不能端到端交付一个生产级的 AI 应用。

这张认证的定位和 Databricks 原有体系（Data Engineer Associate、Spark Developer、ML Professional）明确区分：

- **旧认证**：Spark 优化、Delta Lake 读写、MLflow 追踪、SQL 性能调优
- **GenAI Engineer Associate**：RAG 架构设计、向量数据库接入、Agent 工具调用、模型部署监控、AI 治理和评估指标

考试结构分 6 个域：

1. **Design**：架构决策——什么场景该选 RAG、什么场景该 Fine-tune、什么时候引入 Multi-Agent
2. **Data**：向量数据库（如 Databricks Vector Search）接入方式、Chunk 策略、上下文窗口管理
3. **Assembly**：LangChain/LlamaIndex 集成、Function Calling 实现、Prompt Chain 设计
4. **Deployment**：MLflow 模型版本管理、A/B 流量切分、延迟与成本的权衡取舍
5. **Governance**：数据血缘追踪、模型卡填写、PII 数据保护
6. **Evaluation/Monitoring**：RAGAS 评估指标、答案相关性、漂移检测和 SLA 告警

市场数据：持有 Databricks 认证的数据工程师比未认证同岗位薪资高 15–30%，澳洲 Seek 平台上 2026 年带 Databricks 要求的职位数量环比持续上升。

### 4. Google Professional Cloud Architect 2026 考纲更新：Sovereign Cloud + Gemini Agent Platform 写入

**一句话**：Google Cloud Next '26 之后，Professional Cloud Architect 考纲已纳入 Sovereign Cloud 合规架构和 Gemini Enterprise Agent Platform 两个新考点，2025 年的备考资料现在有盲区。

两个新增重点：

**Sovereign Cloud（主权云）合规架构**：欧盟 AI Act、澳洲隐私法 2024 修正案等法规对跨境数据的要求越来越严，大企业在采购云服务时"数据驻留在哪"成为硬合同条款。新考点要求候选人能设计满足 Data Residency（数据居留）和监管访问限制的 GCP 架构，例如如何选择 Google Sovereign Cloud 区域、如何配置 VPC Service Controls 来阻止数据跨境流动。

**Gemini Enterprise Agent Platform**：Google 在 Cloud Next '26 发布的新平台，把多 Agent 协作写进了 GCP 正式产品线。考题方向是：如何把 Vertex AI Agent 接入企业既有工作流、多 Agent Orchestration 的架构模式、Tool Use 的安全设计。

已有 Professional Cloud Architect 认证的持证者不需要立刻重考，证书到期前正常续证即可；但如果你目前在备考，拿 2025 年版的资料准备时要专门补这两块。

考试报名：[cloud.google.com/learn/certification](https://cloud.google.com/learn/certification)，USD $200，Pearson VUE 在线/现场均可。

### 5. AWS 远程监考系统升级：AI 身份核验 4 月 15 日起全线启用

**一句话**：AWS 从 4 月 15 日起在 Pearson VUE 和 PSI 全面启用 AI 辅助身份核验，在线监考的 Check-in 流程比以前更严，第一次考 AWS 的学员注意提前熟悉。

主要变化：

1. **考前 30 分钟在线 Check-in**：通过摄像头进行 AI 身份比对，不能使用虚拟摄像头软件（如 OBS Virtual Camera）或 VPN，否则 Check-in 会被拒绝
2. **360 度环境扫描**：用手机或摄像头扫描桌面和房间四周，确认无辅助材料（书、笔记、手机均不可见）
3. **有效证件**：必须出示有照片的政府颁发证件，澳洲驾照或护照均可

如果你持有从 JR Academy 或其他渠道购入的 Voucher，证书核验流程和正常购买完全相同，Voucher 本身不受影响。新系统平均 Check-in 时间缩短到 5 分钟以内，但不熟悉流程的人第一次可能会手忙脚乱，建议考前在 Pearson VUE 账户里提前上传 ID 照片。
