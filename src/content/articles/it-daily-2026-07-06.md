## 一、认证动态

### 1. Snowflake SnowPro Advanced MLOps Engineer Beta — 7月13日截止，仅剩7天

**一句话**：Snowflake 首个 MLOps 专项高级认证（MLA-B01）Beta 窗口7月13日关闭，$188 报名，通过后直接拿3年有效期正式证。

自6月15日开放以来，Snowflake SnowPro Advanced MLOps Engineer（MLA-B01）Beta 考试一直在低调运行。今天（7月6日）距 Beta 关闭窗口还有**整整7天**，感兴趣的工程师需要立即行动。

**考试基本信息：**

| 项目 | 详情 |
|------|------|
| 考试代码 | MLA-B01（Beta） |
| 考试费 | **$188 USD** |
| 题量 | 70 题（单选 + 多选混合） |
| 时长 | **115 分钟** |
| 语言 | 仅英语 |
| Beta 报名截止 | **2026-07-13** |
| 交付平台 | Pearson VUE（线上监考 + 全球线下考场） |

**五大考试领域权重（官方 Blueprint）：**

- MLOps 基础设施与管理：**24%**
- Pipeline 编排与 CI/CD 自动化：**22%**
- 数据准备与 Feature Engineering：**20%**
- 模型服务与部署操作：**18%**
- 治理、安全与监控：**16%**

**资格要求**：至少2年以上企业级 ML 生命周期架构和运维经验，其中1年以上需在 Snowflake Data Cloud 平台实际操作（不是只用过 Snowflake 跑 SQL，而是要在 Snowflake 上构建和运维 ML Pipeline）。这不是入门认证，定位是已经在生产环境中落地过 MLOps 的资深工程师。

Beta 期间有两个额外好处：第一，Beta 考试不计入「可考次数」限制（Snowflake 正式考试有次数限制策略）；第二，Beta 通过后直接授予3年有效期的正式证书，不需要再参加正式考试。如果你用 Snowflake 做 MLOps 相关工作，$188 窗口是今年最值得把握的认证机会之一——正式考试上线后价格通常会高于 Beta。

### 2. Microsoft Applied Skills「Create agents in Copilot Studio」明日（7月8日）退役

**一句话**：这张 Microsoft Applied Skills 认证明天截止，最后24小时内完成 Lab Assessment 还能拿证。

Microsoft 官方确认，[Create agents in Microsoft Copilot Studio](https://learn.microsoft.com/en-us/credentials/applied-skills/create-agents-in-microsoft-copilot-studio/) Applied Skills 认证将于 **2026-07-08 正式退役**，退役后无法报名也无法续期。

Applied Skills 是 Microsoft 推出的场景化实操认证（Lab-based assessment），考核方式是在真实沙箱环境中完成具体操作任务，不是选择题。「Create agents in Copilot Studio」这张证书覆盖：在 Microsoft Copilot Studio 中从零搭建对话代理、配置知识源与连接器、发布到 Teams 和网站、设置 Adaptive Cards 交互界面等。整个 Lab Assessment 完成时间约 1-2 小时。

退役原因是 Microsoft 正在把 Agent 开发者认证体系升级整合，以下新 Applied Skills 已上线并作为替代方向：
- **Enhance agents with autonomous capabilities**（赋予代理自主决策能力）
- **Generate reports with AI research agents**（AI 调研代理自动生成报告）
- **Build an agent-first app**（构建以代理为核心的应用）

这些新 Applied Skills 同时也是 **Microsoft AI Agent Skills Sweepstakes**（截止2026-07-15）的参赛资格认证，完成指定技能后提交表单，有机会获得100%折扣的 Microsoft 认证考试代金券（AZ-204、DP-900 等均在使用范围内）。

> 来源：[Microsoft Applied Skills – Microsoft Learn](https://learn.microsoft.com/en-us/credentials/applied-skills/) | [Microsoft AI Agent Skills Sweepstakes](https://learn.microsoft.com/en-us/credentials/certifications/microsoft-ai-agent-skills-sweepstakes)

### 4. CKA 考试运行在 Kubernetes v1.35，2025年重写考纲后首次版本迭代稳定

**一句话**：CKA 环境已切换至 K8s v1.35，2025年初大改考纲（新增 Gateway API、Helm、CRD）现在是完整稳定版，旧备考资料需更新。

CNCF 旗下 Certified Kubernetes Administrator（CKA）认证按惯例在 K8s 新版本发布后 4-8 周内同步更新考试环境，当前考试基于 **Kubernetes v1.35**。

这次迭代的意义在于：v1.35 是 2025 年 1-2 月考纲大幅改写以来的一个稳定节点，所有新增考查内容已在实际考题中完整落地。如果你用的备考材料是2024年底或更早版本，以下内容很可能完全缺失：

**2025 年考纲新增的三大考查方向（高频出题区）：**
- **Gateway API**：Kubernetes 新一代网络抽象层，正在逐步替代传统 Ingress。CKA 考题会要求你配置 GatewayClass、Gateway、HTTPRoute 等资源。
- **Helm & Kustomize**：声明式配置管理工具。考题类型包括：安装 Helm Chart、渲染模板、用 Kustomize patch 覆盖默认值。
- **Custom Resource Definitions（CRD）& Operators**：理解 CRD schema 定义，知道 Operator 的工作原理，能排查 CR 创建失败的问题。

**其他权重上升的领域：**

| 领域 | 变化 |
|------|------|
| Storage | 动态卷供应（Dynamic Provisioning）、Volume AccessMode、Reclaim Policy |
| Troubleshooting | 网络连通性诊断、集群节点故障、CoreDNS 问题排查 |
| Workloads & Scheduling | PodDisruptionBudget、Node Affinity、HPA（水平自动扩容）|

考试格式不变：2小时，全部性能题（实操命令行），推荐备考工具：killer.sh（购买认证包含2次免费 session，每次限时36小时）。正式备考之前先把 [官方 CKA 考纲 PDF](https://training.linuxfoundation.org/certification/certified-kubernetes-administrator-cka/) 下载下来，对照当前学习资料逐项确认覆盖。

> 来源：[CKA Program Changes – Linux Foundation](https://training.linuxfoundation.org/certified-kubernetes-administrator-cka-program-changes/) | [CNCF CKA 认证官方页面](https://www.cncf.io/training/certification/cka/)
