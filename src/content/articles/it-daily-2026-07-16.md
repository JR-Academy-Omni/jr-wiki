## 一、认证动态

### 1. AZ-204 还剩 15 天退役——新人别再备考，直接冲 AI-200

**一句话**：AZ-204 Developing Solutions for Microsoft Azure 将于 **2026 年 7 月 31 日 11:59 PM CST** 正式退役，接班人是 **AI-200 Azure AI Cloud Developer Associate**，目前仍在 Beta 阶段。

如果你正在准备 AZ-204，现在有两条路：一是赶在 7 月 31 日前考完拿证（通过后认证效期不受退役影响，仍按正常一年计算，到期前可免费在线续期）；二是放弃 AZ-204，直接等 AI-200 正式版开放报名。

AI-200 的定位变化值得注意：名字从"Azure Developer"改成"Azure AI Cloud Developer"，考试方向不再是通用 Azure 应用开发，而是 AI 集成应用开发——RAG（检索增强生成）、向量数据库、容器化 AI 工作负载。如果你方向是 AI 应用开发，AI-200 比 AZ-204 更对口；如果你只是需要一个 Azure 开发者认证用来申职位、满足公司要求，且时间紧，就先拿 AZ-204。

目前 AI-200 还没有出官方练习题，Beta 考试不会立刻出分。具体开放时间持续关注 [Microsoft Learn 认证页](https://learn.microsoft.com/en-us/credentials/certifications/)。

> 来源：[Bye AZ-204, hello AI-200 — DEV Community](https://dev.to/lovelacecoding/bye-az-204-hello-ai-200-what-you-need-to-know-about-the-new-azure-developer-certification-2lo) · [Microsoft Certification Retirements 2026 — Vlad Talks Tech](https://vladtalkstech.com/microsoft-learning-and-credential-news/microsoft-certification-retirements-2026/)

### 2. AZ-900 英文版本周日（7月20日）更新——治理权重涨到 35%，AI 内容达 12-18%

**一句话**：AZ-900 Microsoft Azure Fundamentals 英文版将于 **7月20日** 更新考纲，AI 内容占比升至 12-18%，云治理和成本管理权重升至可能 35%，基础架构知识比例下调。

对正在备考的同学影响最直接：如果你用的是旧版学习材料（2025 年底或 2026 年初版），部分权重和考点分布会变。更新后的 Study Guide 会在 [Microsoft Learn AZ-900 页面](https://learn.microsoft.com/en-us/credentials/certifications/azure-fundamentals/) 同步发布。

建议：如果你这周内（7月16日–19日）就能安排考试，用现有材料备考问题不大；如果考试安排在 7月20日以后，尽快下一版新 Study Guide 对照差异，重点补 AI 基础和 Azure 成本管理这两块。

AZ-900 本身考试费 $165 USD，是进入微软认证体系最低门槛，不需要任何前置条件。

### 3. AWS Maintain Beta 上线——刷完 Skill Builder 积分不用考试续期一年

**一句话**：AWS 于 **2026 年 6 月 23 日** 推出 Maintain（开放 Beta），允许持有活跃 AWS 认证的人通过 AWS Skill Builder 上的课程和实操 Lab 积累积分，无需重新考试即可延期认证一年。

积分要求：
- **Associate 级**：500 积分（至少完成 1 个实践活动）
- **Professional 级**：700 积分

使用条件：需持有 **AWS Skill Builder 付费订阅**（个人年费约 $299 USD），且认证必须在 **90 天内到期**。完成积分后，认证自动从完成日起延期一年，相关低级别认证同步级联延期（例如：维护了 SA Professional，仍在有效期内的 SA Associate 也会自动延期）。

目前支持的认证：
- AWS Certified Solutions Architect – Professional / Associate
- AWS Certified DevOps Engineer – Professional
- AWS Certified Developer – Associate
- AWS Certified SysOps Administrator – Associate（CloudOps）

2026 年晚些时候预计加入：Data Engineer – Associate、Security – Specialty、Machine Learning Engineer – Associate。

在 Skill Builder 里找路径：Explore → Validate your skills → Recertify，然后选择你的认证。

> 来源：[A new way to keep your AWS Certification current — AWS Training Blog](https://aws.amazon.com/blogs/training-and-certification/a-new-way-to-keep-your-aws-certification-current/)

### 4. AWS MLA-C02 Beta 9月1日开放注册——MLA-C01 英文版 9月28日最后一考

**一句话**：AWS Certified Machine Learning Engineer – Associate 的更新版 **MLA-C02 Beta** 将于 **2026 年 9 月 1 日** 开放英文注册，现有 MLA-C01 英文版最后考试日期为 **2026 年 9 月 28 日**。

目前 AWS 还没有公布 MLA-C02 的详细考纲，要到 9 月 1 日才能在官网看到更多信息和备考资源。Beta 阶段只有英文版；正式 GA 后才会有韩文、日文、简体中文版本。

如果你已经在备考 MLA-C01 且计划近期考试，9 月 28 日前考完没有问题；如果还没开始，可以等 9 月 1 日 MLA-C02 考纲出来再做决定。MLA-C01 当前考试费 $150 USD。

**一句话**：**2026 年 7 月 8 日** 起，Databricks Genie 产品（AI/BI 智能查询功能）从包月模式切换为按量付费，每个用户每月获得 **150 DBU 免费额度**用于 LLM 调用。

## 二、免费 / 限免课程

### 1. AWS Skill Builder — 10 个免费实操 Lab（无需付费订阅）

**适合**：零基础入门 AWS 的学员，或者备考 AWS Cloud Practitioner / SAA 的初学者

**链接**：[skillbuilder.aws](https://skillbuilder.aws/)

AWS Skill Builder 提供 10 个完全免费的 Builder Labs，覆盖：
- Amazon EC2（虚拟服务器）
- Amazon S3（对象存储）
- Amazon VPC（虚拟私有云）
- AWS IAM（身份与权限管理）
- AWS Lambda（无服务器函数）

不需要 Skill Builder 付费订阅，注册账号直接做。Lab 在真实 AWS Console 环境里跑，不是模拟器。额外还有 600+ 免费视频课和考试备考资料。

订阅版（约 $29 USD/月，或 $299 USD/年）可以解锁 150+ 付费 Lab 和 Exam Prep 官方模拟题。

### 2. Google Cloud Skills Boost — 免费层 AI 三件套

**适合**：想了解生成式 AI 基础、准备 Google Cloud Digital Leader 或 ACE 认证的学员

**链接**：[cloud.google.com/learn/training](https://cloud.google.com/learn/training)

Google Cloud Skills Boost 免费层目前包含：
- **Introduction to Generative AI**（生成式 AI 入门）
- **Introduction to Large Language Models**（大语言模型入门）
- **Introduction to Responsible AI**（负责任的 AI）
- Cloud Digital Leader Learning Path 的入门模块
- 部分基础 Qwiklabs 实操环境

完成免费课程可以获得 Google Cloud 技能徽章，虽然徽章不等于认证，但可以作为学习记录放在 LinkedIn 上。付费订阅 $29 USD/月 可解锁全部课程和实操 Lab，包括 ACE、Professional Data Engineer 等认证对应的备考路径。

### 3. edX — Harvard CS50、IBM Python/ML 课可免费审计（证书需付费）

**适合**：想系统学编程或数据科学基础、不急着拿证书的学员

**链接**：[edx.org](https://www.edx.org/)

edX 目前约有 3,000+ 课程可以免费审计（Audit），包括：
- **CS50's Introduction to Computer Science**（哈佛大学）
- **Data Analysis with Python**（IBM）
- **Machine Learning with Python**（IBM）
- **Introduction to Linux**（Linux Foundation）
- **Python Basics for Data Science**（IBM）

审计模式可以看完全部视频和大部分测验，但不能提交评分作业，也不颁发证书。证书（Verified Certificate）通常每门课 $50–$300 USD。

注意：edX 已从 Audit（旧审计模式）过渡，部分课只提供前 1-2 周免费预览，后续内容需付费。报名前先确认该课的审计政策。

### 4. Microsoft Learn — Everyday AI Fluency（2 小时，完成得 LinkedIn 徽章）

**适合**：想了解微软 AI 工具（Copilot、Azure AI）实际用法的职场人或 Azure 方向学员

**链接**：[Microsoft Learn AI Learning Hub](https://learn.microsoft.com/en-us/ai/)

Microsoft Learn 的"Everyday AI Fluency"学习路径约 2 小时，通过 Word、Outlook、Windows Settings 等实际场景讲解 AI 工具使用。完成后可以获得一个数字徽章，支持导出到 LinkedIn 个人档案。

入口在 Microsoft Learn 的 AI Learning Hub，全部免费，不需要微软账号以外的任何订阅。如果你在备考 AI-900 Azure AI Fundamentals，这个路径的基础概念部分可以直接复用。

## 三、JR Academy 学员行动建议

1. **AZ-204 持有者或正在考 AZ-204 的学员**：7 月 31 日前考完仍然有效；如果时间来不及，等 AI-200 正式 GA（Beta 阶段不建议第一批冲），届时 JR Academy Azure 备考路径会同步更新。

2. **正在备考 AZ-900 的学员**：7 月 20 日考纲更新后，建议对照新版 Study Guide 查一下 AI 和治理两部分知识点有没有盲区，这两块权重最高，改版后更重要。

3. **AWS 证书快到期的同学**：检查证书是否在到期前 90 天内——如果是，可以尝试 AWS Maintain Beta，不用重新考试，在 Skill Builder 刷课刷 Lab 就能续期。需要先确认有没有 Skill Builder 订阅。

4. **备考 AWS ML Engineer（MLA-C01）的学员**：9 月 1 日前可以正常备考 MLA-C01，英文版最后考试 9 月 28 日。如果你 9 月前考不完，等 MLA-C02 考纲出来再决定。

5. **想低成本拿实操经验的学员**：先去 AWS Skill Builder 把 10 个免费 Lab 做一遍（EC2/S3/Lambda/IAM/VPC），再去 Google Cloud Skills Boost 做完 AI 三件套，这些都是零成本的真实环境实操，比光看视频强很多。
