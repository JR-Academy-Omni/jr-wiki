## 1. AZ-204 七月底退役，AI-200 接棒：Azure 开发者认证全面转向 AI

**截止日期警报**：Microsoft 官方已确认，AZ-204（Developing Solutions for Microsoft Azure）将于 **2026 年 7 月 31 日 23:59 CST** 停止报名和续期，Azure Developer Associate 认证随之退役。

**背景**：AZ-204 是 Azure 开发者的主力认证，覆盖 App Service、Azure Functions、Cosmos DB、Event Hub 等核心服务的开发与集成。随着 Microsoft 将开发者赛道转向 AI 优先架构，这张证书的考纲被认为不能反映当前 Azure 工作负载现实。

**替代考试 AI-200**：Microsoft 推出 [Microsoft Certified: Azure AI Cloud Developer Associate (beta)](https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-cloud-developer-associate/)，考试代码 AI-200，目前 beta 阶段，时长 120 分钟，通过线 700 分。AI-200 侧重 Azure AI Foundry、Azure OpenAI Service、AI 代理开发及负责任 AI 治理，是 AZ-204 的功能超集而非纯替代——需要重新备考，不存在直接「升级」路径。

**实操建议**：
- 已在备考 AZ-204 且进度过半的候选人：7 月 31 日之前考完，仍可取得 Azure Developer Associate 认证，该认证在 badge 中会注明获得时间，并不会被撤销。
- 刚起步的候选人：直接备考 AI-200 beta，beta 通过后正式获得 Azure AI Cloud Developer Associate，且 beta 尝试不计入总考试次数限制。
- AZ-204 是否仍可作为 AZ-400（DevOps Engineer Expert）的先修资格：Microsoft 官方 Q&A 已有专门讨论（[参见 learn.microsoft.com 讨论串](https://learn.microsoft.com/en-us/answers/questions/5893548/will-az-204-still-be-accepted-as-a-prerequisite-fo)），建议确认最新政策后再排计划。

来源：[Microsoft Certified: Azure AI Cloud Developer Associate (beta)](https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-cloud-developer-associate/) · [Microsoft Q&A — AZ-204 July 31, 2026 退役讨论](https://learn.microsoft.com/en-us/answers/questions/5893548/will-az-204-still-be-accepted-as-a-prerequisite-fo)

## 2. Google Cloud Digital Leader Beta 考试开放报名至 7 月 19 日，$59 参加

**注意窗口**：Google Cloud Digital Leader（CDL）更新版 beta 考试现正接受报名，**截至 2026 年 7 月 19 日**，之后当前非 beta 版本将于 8 月 11 日下线。

**费用**：Beta 报名费 **$59 美元**（含税则另计），官方原文：

> "Registration fee: $59, 40% discount on retail price of $99 (plus tax where applicable)"

**考试格式**：约 75 道选择题，考试时长 2 小时，仅提供英文版，支持线上监考和线下考场两种方式。**beta 阶段不公布成绩，成绩于 8 月公布**；beta 次数不计入总允许考试次数。

**CDL 是什么**：Cloud Digital Leader 是 Google Cloud 最基础的认证，面向非技术背景的业务决策者和项目经理，覆盖 Google Cloud 核心产品、数字化转型基础和数据/AI 概念。对于刚接触 Google Cloud 生态的 IT 新手或产品经理而言，是**进入 Google 认证体系成本最低的入口**。

**替代选项**：如果不想参加 beta，当前非 beta 版的 CDL 考试可报名至 **8 月 11 日**，费用 $99。Beta 版通过后正式获得 CDL 认证，等同于正式版通过，没有任何降格。

来源：[Cloud Digital Leader | Google Cloud Certification](https://cloud.google.com/learn/certification/cloud-digital-leader)（原文：*"Registration for the Cloud Digital Leader beta exam is now open through July 19, 2026."*）

## 3. Google GEAR 计划：免费 35 学习积分 + 零成本备考 Google Cloud 认证

**近期发布**：Google 宣布推出 **GEAR 计划**（由 Google Skills 驱动），任何人均可免费加入，核心权益包括：

- **每月 35 个学习积分**，可用于 Google Skills 平台的课程和动手实验室
- **Get Certified 免费通道**：Google Cloud 客户可通过该通道以零成本准备 Google Cloud 认证，官方原文：*"Google Cloud customers can join the Get Certified program to prepare for Google Cloud certifications at no cost"*
- **Discord 社群访问权限**
- **Agentic AI 精选学习路径**：Introduction to Agents / Agent Development Kit (ADK) / Deploy Production-Ready Agents / Scale Agents Across the Enterprise

**谁适合**：目前在使用 Google Cloud 的开发者和企业用户（Get Certified 通道需要 Google Cloud 客户身份）；或任何希望入门 AI Agents 开发的工程师（GEAR 基础版完全公开，无需 GCP 订阅）。

**与 CDL Beta 的组合路线**：先通过 GEAR 学习 Google Cloud 基础 → $59 参加 CDL Beta → 通过后以 Get Certified 计划备考 Professional Cloud Architect 或 Professional Data Engineer，可最大程度摊薄认证成本。

加入地址：[developers.google.com/program/gear](https://developers.google.com/program/gear)（官方注：*"Anyone can join GEAR for access to AI training and more"*）

来源：[GEAR up to get the most out of AI learning at Google Cloud Next '26](https://cloud.google.com/blog/topics/training-certifications/gear-up-to-get-the-most-out-of-ai-learning-at-google-cloud-next26)
