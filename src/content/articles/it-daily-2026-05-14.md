## 一、认证动态

### 1. 微软 AI-200 Beta 开放：前 300 名享 80% 折扣，5 月 27 日截止

**一句话**：Azure AI Cloud Developer Associate（考试代码 AI-200）Beta 阶段已上线，在 5 月 27 日前报名并完成考试的前 300 名考生享 80% 折扣，折后约 $36（正价 $165），GA 版本预计 7 月发布。

微软在本月初正式推出 **Microsoft Certified: Azure AI Cloud Developer Associate**，考试代码 **AI-200（Beta）**。这是继 AZ-204（Azure Developer Associate，7 月 31 日退役）之后，微软专为 AI 开发者设计的接班认证。

AI-200 的核心考查方向：在 Azure 上利用**容器化算力、向量数据库、事件驱动 AI Pipeline、Serverless Function、Azure Key Vault 密钥管理、分布式可观测性**，构建和监控生产级 AI 应用。简单说，就是"你能不能在 Azure 上从零搭一个 AI 应用并保证它跑得稳"。

**折扣规则**：前 300 名在 **2026 年 5 月 27 日或之前**完成 Beta 考试，享 **80% 折扣**。按历史规律，Beta 名额通常在截止日 1-2 周前被抢完，目前距截止还有 13 天，建议尽快确认报名。

> 来源：[Microsoft Learn — Exam AI-200 Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/ai-200)｜[Microsoft Community Hub 公告](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-ai-cloud-developer-associate-certification/4494116)

### 2. SC-500 Beta 同步上线：用 $33 左右锁定 AZ-500 的接班证

**一句话**：SC-500 Cloud and AI Security Engineer Associate Beta 在 5 月开放，前 300 名享 80% 折扣；AZ-500 确认 8 月 31 日退役，新证额外覆盖 AI 工作负载安全内容。

继 AI-200 之后，微软在 5 月同步推出 **SC-500（Beta）**，这是 AZ-500（Azure Security Engineer Associate）的直接替代认证。**AZ-500 退役日期：2026 年 8 月 31 日**。

与 AZ-500 相比，SC-500 新增内容：

- **AI 安全模块**：保护 Azure OpenAI 端点、防范 LLM Prompt Injection 攻击、为 AI 工作负载配置访问控制与审计
- 扩展了 Microsoft Sentinel 中针对 AI 生成异常行为的检测规则配置
- 原有 Defender for Cloud、Key Vault、零信任架构内容原样保留

Beta 折扣：类似 AI-200，前 300 名完成考试享 80% 折扣。关注 [Microsoft Certification Hub](https://msfthub.com) 获取具体折扣码和截止日期。

如果你现在的工作涉及 Azure 安全架构，SC-500 的市场定位比 AZ-500 更贴近当前"AI + 传统安全运营并行"的企业现实，含金量将更高。

> 来源：[Whizlabs — Microsoft Next-Generation AI Certifications 2026](https://www.whizlabs.com/blog/microsofts-next-generation-ai-certifications/)

### 3. DP-750 Azure Databricks Data Engineer Associate：Beta 关闭，GA 版本 5 月底发布

**一句话**：微软与 Databricks 联合推出的 DP-750 认证 Beta 阶段已于 4 月 2 日关闭，GA 版本（正价 $165）预计 5 月底至 6 月初正式发布，Databricks + Microsoft 双品牌认证体系正式落地。

今年 3 月，微软发布了首个 Databricks 联合认证：**DP-750: Implementing Data Engineering Solutions Using Azure Databricks**，获得 **Microsoft Certified: Azure Databricks Data Engineer Associate** 资格需通过该考试。

Beta 阶段给前 300 名考生提供了折扣码 `DP750Deltona`（80% 折扣，约 $33），已于 4 月 2 日截止。正式版 GA 发布后，标准价格恢复为 **$165**。

DP-750 考纲五大领域：

| 领域 | 权重 |
|------|------|
| Databricks Intelligence Platform 架构 | 10% |
| 数据摄取与开发（Auto Loader、DLT） | 30% |
| 数据处理与转换（Delta Lake、Unity Catalog） | 31% |
| 生产化数据 Pipeline（DAB、调度）| 18% |
| 数据治理与质量（Delta Sharing、Lakehouse Federation）| 11% |

如果你目前在 Azure 上用 Databricks 做数据工程，这张证是目前市场上对应职能最精准的认证之一。考虑到 Databricks 2026 年 Q3 还有 Learning Festival（50% 折扣），也可以现在备课、7 月打折时报名考试。

> 来源：[Microsoft Learn — DP-750 Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/dp-750)｜[Databricks Community Forum](https://community.databricks.com/t5/certifications/microsoft-just-announced-a-new-certification-microsoft-certified/td-p/150607)

### 4. Google Cloud 认证全面迁移至 Pearson VUE，Cloud Next '26 考纲更新 Q3 落地

**一句话**：Google Cloud 从 Kryterion 切换至 Pearson VUE 的平台迁移已完成，所有 GCP 认证现在通过 Pearson 系统预约；Professional Cloud Architect / Data Engineer / ML Engineer 三大证的考纲将在 Q3 2026 同步 Cloud Next '26 新产品内容。

2026 年 2 月底，Google Cloud 完成考台切换：**Webassessor（Kryterion）→ Pearson VUE**。对正在备考的学员有两点直接影响：

1. **预约渠道变了**：旧 Webassessor 账户的历史记录已迁移，但新考试预约需通过 [Pearson VUE — Google Cloud](https://www.pearsonvue.com/us/en/google-cloud.html) 完成。澳洲学员注意：悉尼、墨尔本的 Pearson 考场比原 Kryterion 授权中心数量更多，本地考选择更灵活了。

2. **考纲更新节奏**：Google 已宣布，跟随 Cloud Next '26 发布的新产品——包括 **Gemini Enterprise Agent Platform、BigQuery AI Analyst、Vertex AI Agent Builder 新功能**——将在 Q3 2026 更新进以下三张证的考纲：Professional Cloud Architect、Professional Data Engineer、Professional Machine Learning Engineer。

现在备考这三张证的学员，建议每周查一次官方考纲页面（[cloud.google.com/learn/certification](https://cloud.google.com/learn/certification)），一旦版本号更新立即对照新旧差异。备考时间充裕的，可以考虑在考纲更新落地后再安排考试，避免考前突击补课。

> 来源：[Google Cloud Certification — Exam Delivery Provider Change](https://support.google.com/cloud-certification/answer/16803278?hl=en)
