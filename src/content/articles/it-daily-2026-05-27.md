## 一、认证动态

### 1. 微软 AI Skills Fest 2026：完成学习路径拿 100% 考试费折扣券，截止 6 月 8 日

**一句话**：微软史上最大规模 AI 学习活动还剩 12 天报名窗口，完成一条学习路径就能拿一张全额考试费折扣券。

微软正在全球推行 **AI Skills Fest 2026**，这是一个不需要任何费用的学习激励活动：注册 → 完成 Microsoft Learn 上的指定 AI 技能学习路径（称为 "Skilling Playlist"）→ 获得一张 **100% 覆盖 Microsoft 或 GitHub 认证考试费** 的凭证码。

关键数字：
- **截止日期**：2026 年 6 月 8 日（太平洋时间）报名享有 "Early Registration Bonus"（双倍抽奖机会）；活动整体截至 **6 月 12 日**
- **考试凭证有效期**：凭证码发出后 60 天内使用，最晚 **2026 年 8 月 11 日（UTC 12:00 AM）到期**
- **适用考试范围**：Microsoft 全线认证（AZ-900、AI-900、DP-750、SC-500、AI-200 等）及 GitHub 相关认证，一人限兑一张
- **附赠抽奖**：注册即进入价值 $25,000 USD 的全球抽奖（科技大礼包 + VIP 旅行名额）

对准备在 7 月底前参加任何 Microsoft 认证的同学，这张凭证直接抵消一次考试费（AZ-900 约 $165 USD，DP-750 约 $165 USD，AI-900 约 $165 USD）。唯一代价是先花几小时完成 Microsoft Learn 上的 AI 基础课程路径。

### 2. Databricks Context Engineer Associate：全球首个 AI Agent 上下文工程认证，Beta 考试 6 月 15 日首发

**一句话**：Databricks 正式宣布业界第一张专为 AI Agent 上下文管理设计的认证，是 GenAI 方向从"用 AI"到"会工程化 AI"的分水岭。

**Databricks Certified Context Engineer Associate** 是 Databricks 自己定义的一个全新岗位认证，不同于现有的 Generative AI Engineer Associate（侧重模型集成与 RAG 管道），Context Engineer Associate 专注于：

- 设计和组装 AI Agent 使用的**上下文系统**（Context Architecture）
- 在 Databricks 平台上对 AI Agent 的上下文进行**治理与监控**
- 管理 Agent 的记忆、工具调用、上下文检索等组件

Beta 考试将在 **2026 年 6 月 15–18 日 Data + AI Summit**（旧金山 Moscone 中心）期间首发，到场参会者可以成为第一批参加 Beta 考试的候选人。Data + AI Summit 同时提供线上参会选项，Beta 考试是否同步向线上参会者开放尚待 Databricks 官网确认。

对已有 Databricks Generative AI Engineer Associate 的同学，这张证是在 AI Agent 工程化方向的直接延伸，适合正在做 LangGraph / CrewAI / Mosaic AI Agent Framework 落地项目的工程师。对刚进 AI 工程方向的同学，这两张 Databricks 认证组合（GenAI Engineer + Context Engineer）在 2026 年市场上属于稀缺组合。

> 来源：[Databricks 官方博客 — Context Engineer Associate 发布公告](https://www.databricks.com/blog/databricks-context-engineer-associate-industrys-first-certification-reliable-ai-agent-systems) | [Databricks Data + AI Summit 2026](https://www.databricks.com/dataaisummit)

### 3. Databricks Data Engineer Associate 考试 5 月 4 日大改版：Lakehouse 成必备前置，场景题全面取代定义题

**一句话**：5 月 4 日起，Databricks 数据工程师认证换了全新题型思路——不再考概念记忆，全换成真实生产场景题。

Databricks 在 2026 年 5 月 4 日发布了 **Databricks Certified Data Engineer Associate** 的重大改版，核心变化包括：

1. **Lakehouse 架构作为考试前置假设**：新题目默认考生已理解 Lakehouse 如何把数据湖和数据仓库特性结合，不再从零解释概念，而是直接考"在这个 Lakehouse 架构下你会怎么做"
2. **场景化题目取代定义题**：题目格式从"XX 是什么"换成"业务给出一个数据问题，你会怎样用 Databricks 解决"——贴近实际工作方式
3. **具体技术重点**：Auto Loader（增量摄取）、Structured Streaming（实时流处理）、Late Data 处理（延迟数据容忍策略）是新版三大实战考点

如果你正在用 2026 年 4 月前的学习材料备考，建议先下载 Databricks 官网最新的考试指南（Exam Guide），按照新版知识点核对学习进度，旧版材料在"场景化解题思路"上几乎没有覆盖，直接上考场很容易踩坑。

> 来源：[Databricks Certified Data Engineer Associate 认证页](https://www.databricks.com/learn/certification/data-engineer-associate) | [Databricks 认证总览](https://www.databricks.com/learn/training/certification)

### 4. GCP Professional Cloud Security Engineer 考纲更新：零信任 + Confidential Computing 成新核心考点

**一句话**：Google Cloud Security Engineer 认证在 2026 年大幅加入了 Confidential Computing 和上下文感知访问控制的内容，旧版备考资料不够用了。

Google Cloud Next '26 宣布了一系列数据安全和 AI 安全产品更新，随之而来的是 **Professional Cloud Security Engineer** 认证考纲的同步调整。主要变化：

- **Confidential Computing（机密计算）**：新增 Google Cloud Confidential VM 和 Confidential Space 的配置与使用场景，考点包括使用硬件级加密保护运行时数据
- **Assured Workloads**：扩展了合规工作负载配置的考点，包含如何满足特定国家监管要求（ITAR、FedRAMP 等）
- **零信任与上下文感知访问**：BeyondCorp Enterprise、Context-Aware Access 策略的设计与故障排查进入核心考点
- **威胁检测与事件响应**：Security Command Center 的 Event Threat Detection、Container Threat Detection 相关题量增加

考试报名链接和费用不变（$200 USD，Pearson VUE 全球可考）。已经购买旧版备考课程的同学需要补充 2026 年更新后的官方 Exam Guide 内容。

> 来源：[Google Cloud Certification 官方页](https://cloud.google.com/learn/certification) | [certempire.com — GCP Security Engineer 2026 更新详解](https://certempire.com/exam/professional-cloud-security-engineer-exam-questions/)

### 1. Microsoft Learn AI Skills Fest 学习路径：完成即拿 100% 考试费折扣券

**适合**：已经规划在 2026 年 8 月前考任何一张 Microsoft 认证的同学
**链接**：[https://learn.microsoft.com/en-us/credentials/support/ai-skills-fest-faq](https://learn.microsoft.com/en-us/credentials/support/ai-skills-fest-faq)

这是本周最有实际价值的"免费课程"——不是拿知识，是拿考试钱。

操作路径：

1. 登录 Microsoft Learn，进入 AI Skills Fest 注册页（通过上方 FAQ 链接找入口）
2. 选择一条 "Skilling Playlist"（Microsoft 提供多条 AI 方向路径，每条约 3–8 小时，包含视频 + 练习）
3. 完成全部模块 → 系统自动发放一次性 100% 折扣码
4. 在 6 月 8 日前完成注册还能拿到双倍抽奖机会（$25,000 奖池）

凭证码有效期 60 天（发放后），最晚用到 2026 年 8 月 11 日 UTC。

给不确定考哪张证的同学：AI-900 入门最低门槛，平时也是最多人拿来"以后备用"的过渡证——用这张凭证考 AI-900 来"解锁"后续方向是合理选择。

### 2. DeepLearning.AI × Oracle 新课：Agent Memory: Building Memory-Aware Agents（免费短课）

**适合**：有 Python 基础、正在做或学习 AI Agent 开发的工程师
**链接**：[https://www.deeplearning.ai/courses/](https://www.deeplearning.ai/courses/)

DeepLearning.AI 与 Oracle 合作新上线了短课 **"Agent Memory: Building Memory-Aware Agents"**，课程核心是：如何给 AI Agent 设计有持久性、连续性的记忆系统——这正好是备考 Databricks Context Engineer Associate 的实战基础知识。

课程内容涵盖：
- AI Agent 的记忆分类（短期记忆 vs. 长期记忆 vs. 语义记忆）
- 架构设计：如何实现 Agent 在对话轮次间"记住"用户偏好和历史操作
- 代码实现：用 Oracle 提供的工具链搭建内存感知型 Agent

DeepLearning.AI 短课一贯免费、无需注册 Coursera 账号（直接在 deeplearning.ai 平台完成），典型时长 1–2 小时，全程 Jupyter Notebook 实战，完成后无证书但有完整代码可带走。

考虑到 Databricks Context Engineer Associate 将在 6 月 15 日首发 Beta，提前用这门课打好 Agent 记忆系统的概念基础是高效的预热方式。

> 来源：[Oracle Developers Blog — Agent Memory 课程发布](https://blogs.oracle.com/developers/oracle-and-deeplearning-ai-launch-new-agent-memory-course-for-ai-developers) | [DeepLearning.AI 课程列表](https://www.deeplearning.ai/courses/)
