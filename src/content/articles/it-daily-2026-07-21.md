## 一、认证动态

### 1. AWS 专业级 GenAI 认证（AIP-C01）正式开放注册 — $300、通过分 750、含 Bedrock AgentCore 新考点

**一句话**：AWS 体系里难度最高的 AI 方向认证于 3 月 17 日结束 Beta 正式开放，今天报名即可上考，$300 报名费，通过线 750/1000。

AIP-C01（AWS Certified Generative AI Developer – Professional）是 AWS 2026 年 AI 认证扩充的旗舰产品。Beta 阶段结束时，AWS 在正式版考纲里追加了 **Amazon Bedrock AgentCore**，覆盖 Agent 的工具调用管理、持久化内存和代码执行沙盒。这直接对应现在市场上最活跃的岗位需求：把 GenAI Agent 从原型跑进生产环境。

**考试关键数字：**

| 项目 | 数值 |
|------|------|
| 考试代码 | AIP-C01 |
| 报名费用 | $300 USD |
| 通过分数 | 750 / 1000 |
| 题目数量 | 65 道计分 + 10 道不计分 = 75 道 |
| 认证有效期 | 3 年 |
| 推荐经验 | 2 年以上 AWS 生产经验 + 1 年 GenAI 落地经验 |

**五个考试领域及权重：**

1. 基础模型集成、数据管理与合规（Domain 1）—— **31%**
2. 实施与集成（Domain 2）—— **26%**
3. AI 安全、治理与合规（Domain 3）—— **20%**
4. 运营效率与 GenAI 应用优化（Domain 4）—— **12%**
5. 测试、验证与排错（Domain 5）—— **11%**

Domain 1 和 Domain 3 合计 51%，这两块是备考重点：前者考你怎么选 FM、怎么设计 RAG pipeline、怎么处理合规数据；后者考 AI 系统的访问控制、输出过滤和审计日志。

这张证和入门级 AIF-C01（$100）的层级差异显著：AIF 是概念理解，AIP 是动手实现。已有 AIF-C01 或 MLA-C01 的学员、Bedrock 生产项目经验 1 年以上，可以直接冲。没有 Bedrock 经验的学员建议先拿 AIF-C01 再规划。

> 来源：[AWS Certified Generative AI Developer – Professional 官方页面](https://aws.amazon.com/certification/certified-generative-ai-developer-professional/) · [考纲 AIP-C01 PDF](https://docs.aws.amazon.com/pdfs/aws-certification/latest/ai-professional-01/ai-professional-01.pdf) · [AWS Training and Certification Blog — AI 认证扩充公告](https://aws.amazon.com/blogs/training-and-certification/big-news-aws-expands-ai-certification-portfolio-and-updates-security-certification/)

### 2. Snowflake SnowPro Core COF-C03 — 旧版翻译本本周四（7 月 31 日）永久退役

**一句话**：2 月上线的新版 COF-C03 已于 5 月让英文 COF-C02 退役，本周四旧版的日语、韩语、法语、西语翻译考试全部关闭，全球统一切换 COF-C03。

今天是 7 月 21 日，距离截止还有 10 天。如果你正在备考 Snowflake 认证，今天需要做一件事：登录 Pearson VUE，确认你的报名显示的是 **COF-C03**，不是 COF-C02。如果用的是老版非英语题库资料，那批题目的考试已经不再提供报名。

**COF-C03 相比 COF-C02 的核心变化：**

- **Git 集成**（新考点）：如何在 Snowflake 里对接 Git 仓库、做分支管理和代码版本控制
- **Snowflake Cortex AI**（新考点）：平台内置 LLM 功能，不调外部 API 就能在 SQL 里跑 AI 查询
- **Iceberg Tables**（新考点）：基于开放格式的数据湖表，与 AWS S3、GCP Storage、Azure ADLS 互通
- **Snowflake Notebooks**（新考点）：类 Jupyter 的交互式开发环境，直接在平台内跑 Python/SQL

**COF-C03 考试信息：**

| 项目 | 数值 |
|------|------|
| 考试代码 | COF-C03 |
| 报名费用 | $175 USD（Pearson VUE） |
| 题目数量 | 100 道 |
| 考试时长 | 约 110 分钟 |
| 推荐经验 | 6 个月以上 Snowflake 实操 |
| 旧版退役日期 | 2026 年 7 月 31 日（翻译版） |

对 JR 学员的直接影响：数据工程方向的学员如果在备考 Snowflake 认证，今天检查一下报名状态，顺便更新备考资料到 COF-C03 版本。新考点里 Cortex AI 和 Iceberg Tables 预计出题比例不低。

### 3. Microsoft 四大安全专业认证 7 月底切换第三方审计模式 — 合作伙伴续签必须了解

**一句话**：7 月底，微软把 Cloud Security / Data Security / Identity & Access Management / Threat Protection 四个 Security Specialization 全部从内部 Checklist 审核切换成独立第三方审计，影响所有需要续签这些资质的微软合作伙伴。

这次变化的核心是：之前填写能力清单（Checklist）加几个客户案例就能过审，7 月底之后要由**独立第三方审计机构**进场，验证合作伙伴实际能否在客户真实生产环境交付对应的微软安全解决方案。

受影响的四个 Security Specialization：
1. **Cloud Security**（云安全专业认证）
2. **Data Security**（数据安全专业认证）
3. **Identity & Access Management**（身份与访问管理）
4. **Threat Protection**（威胁防护）

**对个人考生的影响**：这次变化针对的是**组织层面的合作伙伴资质**，不影响 SC-200、SC-300、SC-400、SC-900 等个人认证的考试和续期。如果你在考这些证，备考计划不需要变。

**需要关注的群体**：MSP、系统集成商、安全服务提供商中，持有上述 Security Specialization 的微软合作伙伴需要在续签前准备审计材料。今天的动作：把 [Microsoft Partner Center July 2026 公告](https://learn.microsoft.com/en-us/partner-center/announcements/2026-july)发给公司技术总监或合规负责人。

> 来源：[Microsoft Partner Center — July 2026 Announcements](https://learn.microsoft.com/en-us/partner-center/announcements/2026-july)
