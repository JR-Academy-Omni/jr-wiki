## 一、认证动态

### 1. 明日截止：AI-102（Azure AI Engineer Associate）今天是最后一天

**一句话**：AI-102 于 2026 年 6 月 30 日正式退役，今天（6 月 29 日）是最后一天参加考试，错过就只能转考 AI-103 重新准备。

AI-102（Azure AI Engineer Associate）退役日期为 **2026 年 6 月 30 日**，明日起该认证及对应的续期评估将全部停止接受注册和考试。证书名称是 Microsoft Certified: Azure AI Engineer Associate。

**考试信息差（避免踩坑）**：

- AI-102 退役后已购买但未使用的 Microsoft Certification Voucher，不自动转换为 AI-103 等价，退款或兑换需联系 Microsoft 认证支持
- 明日起 AI-102 相关考试预约入口关闭，当前有预约的考试今日仍可参加

**如果今天没有排上考试**：不必过分紧张。AI-103 是直接替代路径，Associate 级别不变。AI-102 覆盖的 Azure OpenAI 集成、Responsible AI 原则、Azure AI Studio 基本用法在 AI-103 里仍然考，只是 Computer Vision 和 Form Recognizer 的权重大幅下降，新增 Azure AI Foundry 统一平台和 Multi-Agent 编排。

> 来源：[AI-102 retires on 30th June 2026 - Microsoft Q&A](https://learn.microsoft.com/en-us/answers/questions/5893448/ai-102-retires-on-30th-june-2026)

### 2. AI-103（Azure AI Apps and Agents Developer Associate）：不是 AI-102 小改版，是技能方向切换

**一句话**：AI-103「Developing AI Apps and Agents on Azure」4 月已进入 Beta，预计 6 月内 GA，主考 Azure AI Foundry 和 Multi-Agent 编排，和 AI-102 是两套完全不同的技能栈。

AI-103 通过后证书名称为 **Microsoft Certified: Azure AI Apps and Agents Developer Associate**，与 AI-102 的 Azure AI Engineer Associate 是独立两张证书，不是直接改名延续。

**两张证书核心区别**：

| 项目 | AI-102 | AI-103 |
|------|--------|--------|
| 考试代号 | Exam AI-102 | Exam AI-103 |
| 证书名称 | Azure AI Engineer Associate | Azure AI Apps and Agents Developer Associate |
| 核心平台 | Azure Cognitive Services（各分散服务） | Azure AI Foundry（统一 AI 开发平台） |
| 强调技能 | 部署视觉 / 语音 / NLP 服务 | Generative AI 应用构建 + Multi-Agent 编排 |
| Beta 状态 | 退役 | 2026 年 4 月进入 Beta |
| GA 预计 | — | 2026 年 6 月内 |

考试价格以 [Microsoft Learn 官网](https://learn.microsoft.com/en-us/credentials/certifications/) 为准（Beta 期间通常有折扣）。

**给 AI-102 在读学员的判断线**：备考进度 ≥60% 且今天有排考时间 → 按计划执行 AI-102；备考进度 <40% 或今天来不及安排 → 直接转 AI-103，Azure AI Foundry 和 Agents 在 2026 年的职场需求权重更高，重新备考的内容更有现实意义。

> 来源：[AI-102 vs AI-103: Which Microsoft AI Exam in 2026 - CertEmpire](https://certempire.com/ai-102-vs-ai-103/)、[From AI-102 to AI-103: Microsoft's Big Shift Toward Generative AI and AI Agents - CertQueen](https://www.certqueen.com/news_From_AI-102_to_AI-103_Microsofts_Big_Shift_Toward_Generative_AI_and_AI_Agents_5378.html)

### 4. AWS AIP-C01 GA 版新增 Amazon Bedrock AgentCore 考点

**一句话**：AWS Certified Generative AI Developer – Professional（AIP-C01）GA 版于 3 月 17 日开放注册，相比 Beta 版新增了 Amazon Bedrock AgentCore 作为考点，用 Beta 期备考材料的学员可能漏掉这块。

AIP-C01 GA 版考试信息：

| 项目 | 详情 |
|------|------|
| 考试代号 | AIP-C01 |
| 定价 | $300 USD |
| 题目数 | 75 题（65 计分 + 10 不计分）|
| 通过线 | 750 / 1000 |
| 时长 | 170 分钟 |
| 主要考点 | Amazon Bedrock / Agents / RAG 架构 / Bedrock AgentCore / Guardrails / 负责任 AI |

**Bedrock AgentCore 是什么**：AWS 推出的托管 Agent 基础设施，用于在生产环境中部署和运行 AI Agents，提供内存管理、工具调用和会话状态维护能力。GA 版将其纳入考试范围，考题可能出现"何时应该用 Bedrock AgentCore 而不是 Bedrock Agents"类的架构选择题。

备考提示：从 [AIP-C01 官方 Exam Guide PDF](https://docs.aws.amazon.com/aws-certification/latest/examguides/ai-professional-01.html) 下载最新版，搜索"AgentCore"；如果你用的备考材料没有这个词，前往 AWS Skill Builder 搜 Generative AI Learning Plan 补课（学习计划本身免费访问）。

> 来源：[AWS Certified Generative AI Developer - Professional - Pluralsight](https://www.pluralsight.com/resources/blog/cloud/new-aws-aip-c02-exam)、[AIP-C01 Exam Guide - AWS Docs](https://docs.aws.amazon.com/aws-certification/latest/examguides/ai-professional-01.html)

### 5. GCP Professional Machine Learning Engineer 新版 6 月更新：聚焦 Gemini Enterprise Agent Platform

**一句话**：Google Cloud 在 6 月更新了 Professional Machine Learning Engineer（PMLE）考试内容，对齐 Google Cloud Next 2026（4 月）发布的 Gemini Enterprise Agent Platform（原 Vertex AI Agent Builder 的继承平台），旧版 Vertex AI 考题权重有所调整。

更新重点（基于认证培训机构汇总信息）：

- 新增：在 **Gemini Enterprise Agent Platform** 上设计 AI Agent 的架构方案
- 新增：**RAG pipeline** 设计、评估与监控方法论
- 保留：BigQuery ML、AutoML、模型监控、数据管道设计
- 权重下降：Vertex AI（旧名）各分散服务的原有用法

备考提示：前往 [Google Cloud PMLE 认证官页](https://cloud.google.com/learn/certification/machine-learning-engineer) 下载当前 Exam Guide PDF，对比自己的学习材料是否已覆盖 Gemini Enterprise 相关内容；如果材料只提到 Vertex AI Agent Builder（旧名），建议补充查阅 Google Cloud 官方文档更新。

另外提醒：GCP 认证考试已于 2026 年 2 月 26 日完成向 Pearson VUE 的迁移，现在需要通过 Pearson VUE 渠道预约并参加 GCP 认证考试。

> 来源：[Google Cloud Certification Roadmap 2026 - NicheLab](https://nicheelab.com/en/articles/gcp/certification-roadmap/)、[Upcoming Change to Exam Delivery Provider - Google Cloud Certification Help](https://support.google.com/cloud-certification/answer/16803278?hl=en)

### 1. Anthropic × Coursera：5 门 AI Fluency 免费课（Community Impact，注册即免费）

Anthropic 与 Coursera 合作于 2026 年 5 月上线 5 门 AI Fluency 课程，属于 Coursera 的 **Community Impact** 类别（专为解决社会紧迫需求设计，不需要 Coursera Plus 订阅，注册账号即可免费学习并拿证书）。

**5 门课程清单**：

| 课程名 | 目标人群 | 大致时长 |
|--------|----------|----------|
| AI Fluency: Framework and Foundations | 所有人，入门级 | 约 3 小时 |

### 2. DeepLearning.AI「Building Code Agents with HF smolagents」（免费）

**适合**：有 Python 基础、想动手构建 AI Agent 的开发者；备考 AWS AIP-C01 里 Agent 架构题的学员

**链接**：[Building Code Agents with Hugging Face smolagents - DeepLearning.AI](https://www.deeplearning.ai/courses/building-code-agents-with-hugging-face-smolagents)

课程由 **Thomas Wolf**（Hugging Face 联合创始人兼 CSO）和 **Aymeric Roucher**（smolagents Project Lead）共同讲授，使用 Hugging Face 的 smolagents 框架搭建 Code Agent——即 Agent 以直接编写并执行 Python 代码的形式完成工具调用，而不是传统的 JSON 工具调用模式。

**和备考 AIP-C01 的关联**：smolagents 的 Code Agent 模式让 Agent 的推理和工具调用都通过代码执行，可审计、可复现；和 Amazon Bedrock Agents 的 JSON-based 工具调用是两种不同的 Agent 实现路径。学完这门课，在遇到 AIP-C01 里"比较不同 Agent 架构的适用场景"类考题时会有更清晰的判断依据。

DeepLearning.AI 短课程（Short Courses）系列目前在 Beta 阶段全部免费访问，但免费窗口不定期关闭，建议尽早注册领取。

> 来源：[Building Code Agents with Hugging Face smolagents - DeepLearning.AI](https://www.deeplearning.ai/courses/building-code-agents-with-hugging-face-smolagents)

### 3. AWS Skill Builder Builder Labs + Lab Maker：真实 AWS 环境实操免费开放

**适合**：备考 AWS SAA-C03、AIP-C01、SysOps Administrator 等需要动手经验的学员

**链接**：[AWS Skill Builder](https://skillbuilder.aws/)

AWS 近期将 **Builder Labs**（原付费内容）对所有 Skill Builder 账户免费开放。Builder Labs 在**真实 AWS 生产环境**（而非模拟器）中完成实操任务，包括配置 VPC、部署 Lambda、设置 S3 访问控制等，完成后环境自动回收，不会产生费用。

同步上线的还有 **Lab Maker**（2026 年 5 月发布）：一个 AI 驱动的 Lab 生成器，输入你要练习的 AWS 服务，自动生成对应的动手实验场景，无需人工编排。

**备考 AIP-C01 的直接价值**：Builder Labs 里包含 Amazon Bedrock 和 Bedrock Agents 专项实操 Lab，练完基本搞清楚 Bedrock AgentCore、Agents、Knowledge Base 三者的区别，对应到考题里的架构选择题会有明确判断。600+ 门免费数字课程可在注册 AWS Builder ID 后直接访问（无需付费）。
