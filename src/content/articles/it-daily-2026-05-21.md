## 一、认证动态

### 1. AWS Certified Generative AI Developer – Professional 标准版正式开放注册

**一句话**：Beta 期结束，标准考试自 2026年3月17日起全面开放注册，新增 Amazon Bedrock AgentCore 考点。

这是 AWS 认证体系里目前最新的 Professional 级别考试，代号 AIP-C02（或沿用旧 Beta 叫法），考察的是用 AWS 服务构建并部署生产就绪 AI 解决方案的能力。考试重点域：

- **Foundation Models 选型与评估**：在 Amazon Bedrock 上按业务需求选、微调和评估基础模型
- **RAG 架构实现**：向量数据库、Knowledge Bases for Bedrock、检索策略调优
- **AI Agents 开发**：Amazon Bedrock AgentCore（3月新加考点）——创建、部署、监控 AI Agents
- **Responsible AI**：防护栏（Guardrails for Bedrock）、提示注入防御、合规合规审计

考试费用 $300 USD（标准价），没有 Beta 折扣。AWS Skill Builder 上有官方「Exam Prep Plan: AWS Certified Generative AI Developer – Professional」，包括考域分析、SimuLearn 实操和模拟题，**全部免费**（无需 Skill Builder Plus 订阅）。

对 JR Academy 学员的判断：这个认证适合已持 AWS Developer Associate 或 AWS ML 相关背景的人。AWS ML Specialty 已于2026年3月31日退役，这个 Pro 级别的 Gen AI 认证是最直接的升级路径。

> 来源：[AWS Certified Generative AI Developer – Professional（官网）](https://aws.amazon.com/certification/certified-generative-ai-developer-professional/) · [AWS Training Blog 3月更新](https://aws.amazon.com/blogs/training-and-certification/march-2026-new-offerings/)

### 2. AWS 云运维认证已改名：SysOps Admin → CloudOps Engineer（SOA-C03）

**一句话**：2025年9月30日起，AWS SysOps Administrator - Associate 正式更名为 AWS Certified CloudOps Engineer - Associate，旧版 SOA-C02 已退役，新版考试代码 SOA-C03。

很多人还在用旧名备考，这里明确几个关键点：

- **SOA-C02 最后一天**：2025年9月29日（已过）
- **SOA-C03 第一天**：2025年9月30日（已在考）
- **名称适用规则**：只有通过 SOA-C03 才会获得「CloudOps Engineer」称谓；持旧版 SOA-C02 证书的人，证书上的名称不会自动更新，但无需重考（有效期和续期规则不变）

SOA-C03 相比 SOA-C02 的核心变化：
- **容器正式入考**：ECS、EKS 基础操作纳入考点
- **多账户/多区域**：AWS Organizations、Control Tower 的实操权重提升
- **IaC 自动化**：CloudFormation、Systems Manager Automation 的更深考查
- **现代服务覆盖**：EventBridge、Step Functions 等新增服务

考试费 $150 USD，2小时65题，Associate 级别（不需要 Professional 前置）。如果你在 Udemy 刷的是「SysOps」的老课，注意核查课程版本是否已覆盖 SOA-C03 考纲。

> 来源：[AWS CloudOps Engineer 认证页](https://aws.amazon.com/certification/certified-cloudops-engineer-associate/) · [AWS Blog 改名公告](https://aws.amazon.com/blogs/training-and-certification/exam-update-and-new-name-for-operations-certification/)

### 3. 微软 DP-750 Azure Databricks Data Engineer Associate 本月 GA

**一句话**：Microsoft Certified: Azure Databricks Data Engineer Associate 认证（对应考试 DP-750）2026年5月正式 GA，Beta 期间前300名享80%折扣（代码 DP750Deltona，截止2026年4月2日，已过）。

DP-750 是微软和 Databricks 联合发布的新认证，填补了 Azure Data Engineer（DP-203）和纯 Databricks DEA 之间的空白。考点围绕 Delta Lake + Unity Catalog + Azure 数据管道：

- **Delta Lake 架构**：ACID 事务、Time Travel、Change Data Feed
- **Unity Catalog 权限管理**：在 Azure Databricks 工作区里做精细化数据访问控制
- **Azure 数据集成**：Databricks 连接 Azure Data Factory、Event Hubs、ADLS Gen2
- **MLflow 在 Azure ML 上的应用**：实验追踪、模型注册、部署

考试费 $165 USD。现在 Beta 折扣已过，但可以等 **7月 Databricks Learning Festival**（见下条）。备考材料：Microsoft Learn 上有对应学习路径（免费），Databricks Academy 有官方自学课（部分免费）。

> 来源：[微软技术社区公告](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-databricks-data-engineer-associate-certification/4494114)

### 4. Databricks Learning Festival 7月开跑：50% 认证折扣码预告

**一句话**：Databricks 每季度举办 Learning Festival，下一届是 2026年7月，参与者完成一门自定进度课程即可拿 50% 认证折扣码——Databricks DEA、DPA 等考试原价 $200 USD，折后仅 $100。

具体机制：
- **时间**：固定四次 / 年——1月、4月、7月、10月
- **参与条件**：在 Databricks Academy 上完成至少一门免费自定进度课（无需 Databricks 企业账户）
- **折扣范围**：所有 Databricks 认证考试（DEA Databricks Certified Data Engineer Associate、DPA Machine Learning Professional 等）
- **拿到折扣码后**：通常可在 1 个月内使用，不能叠加其他优惠

目前 5月没有折扣活动。如果你计划考 Databricks DEA 或 DP-750，建议 6月底开始备考，7月 Festival 期间拿码，7月 / 8月报名参考。

> 来源：[Databricks 社区 Learning Festival 讨论](https://community.databricks.com/t5/certifications/inquiry-regarding-discount-vouchers-for-databricks-data-engineer/td-p/155347)
