## 一、认证动态

### 1. 微软 AI-103 Beta 接近尾声，GA 预计 6 月：替代 AI-102 的 Agentic AI 认证来了

**一句话**：AI-103 是微软今年最重要的新认证，替代 AI-102，专注 Azure AI Foundry + 多智能体工作流，GA 预计 6 月，现在备考正当时。

完整名称：**Microsoft Certified: Azure AI Apps and Agents Developer Associate**，考试代码 **AI-103**。这张证书定位于替代现行 AI-102（Azure AI Engineer Associate），考点从传统 Cognitive Services（语音、视觉、文本分析）全面转向 generative AI 与 agentic 工作流。

**考试规格**：
- 时长：120 分钟
- 过线分数：700 分（满分 1000）
- 当前状态：Beta 阶段（GA 预计 2026 年 6 月）

**考试内容覆盖**：Azure AI Foundry 端到端开发流程、RAG（检索增强生成）实现、function calling、agent orchestration、多模态场景，以及 responsible AI 防护措施与部署标准。

Beta 阶段曾有 80% 折扣窗口：前 300 名在 5 月 7 日前使用代码 **AI103Claxton** 可享 80% 折扣——截止日已过，折扣不再有效。Beta 考试目前是否仍开放报名取决于 Microsoft 安排，在 GA 前通常会保留一段时间。GA 预计 6 月落地，这意味着：**在 GA 前考过的人是全球首批持证人，早期采用者优势明显**。

与 AI-102 的核心区别：AI-102 侧重调用 Azure 认知服务 API（语音转文字、LUIS、Form Recognizer 等），AI-103 的核心场景是在 Azure AI Foundry 上编排 agent、构建 RAG pipeline、把 LLM 接入真实业务流程。如果你已有 AI-102，下一张就该考 AI-103；如果还没考，可以直接跳过 AI-102 冲 AI-103。

> 来源：[Microsoft Community Hub — New Microsoft Certified: Azure AI Apps and Agents Developer Associate](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-ai-apps-and-agents-developer-associate/4494126)；[mscertquiz.com 引用官方公告：「The first 300 people who take Exam AI-103 (beta) on or before May 7, 2026, can get 80% off using code AI103Claxton」](https://mscertquiz.com/blog/new-microsoft-beta-exams-2026)；[Microsoft Learn AI-103 Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/ai-103)

### 2. CompTIA SecAI+（CY0-001）：行业首个 AI 安全认证，2 月 17 日正式发布，$359 USD

**一句话**：CompTIA 2026 年最大的认证动作之一——行业首个专门面向 AI 系统安全的独立认证，代码 CY0-001，价格 $359，60 分钟考试。

**CompTIA SecAI+** 不是 Security+ 的子集，而是独立认证，覆盖两个能力维度：

1. **保护 AI 系统**：模型安全、训练数据投毒防御、对抗样本攻击、AI pipeline 加固、模型权限控制
2. **在网络安全工作中使用 AI 工具**：AI 辅助威胁检测、安全自动化、AI 驱动的 SOC 分析流程

**官方考试规格**（来自 CompTIA 官网）：
- **考试代码**：CY0-001
- **发布日期**：2026 年 2 月 17 日
- **考试时长**：60 分钟
- **题目数量**：最多 60 题（选择题 + PBQ 基于性能题型）
- **建议经验**：3-4 年 IT 工作经验 + 2 年以上网络安全实操
- **价格**：约 $359 USD（单次考试）；含一次重考的套餐约 $408

为什么值得关注？AI 已经全面渗透红队 / 蓝队、SOC、漏洞扫描等安全岗位，但市场上几乎没有专门验证「AI × 安全」能力组合的认证。SecAI+ 填补了这个空白，且 CompTIA 品牌在雇主侧认可度高——澳洲、美国、新加坡的安全岗 JD 里 CompTIA 认证出现频率仅次于 CISSP。

对 JR Academy 学员而言：如果你是 Security+ 持证人、想往 AI Security Engineer 方向发展，SecAI+ 是比 CISSP 更聚焦、更快取得的短期证书目标。

> 来源：[CompTIA SecAI+ 官方产品页（comptia.org — Pre-Order Now, Launching February 17, 2026）](https://www.comptia.org/certifications/secai)；[CompTIA 发布公告引用：「The SecAI+ exam runs 60 minutes with a maximum of 60 multiple-choice and performance-based questions, and CompTIA recommends candidates have three to four years of experience in IT with more than two years of hands-on cybersecurity experience」](https://www.networkworld.com/article/4133427/comptia-launches-secai-certification.html)

### 3. AWS Certified Generative AI Developer Professional（AIP-C01）：Beta 结束，$300 GA 版正式开放

**一句话**：AWS 首个 Professional 级 GenAI 认证 Beta 于 3 月 31 日关闭，GA 标准价格 $300 USD，现在正常报名。

**AWS Certified Generative AI Developer – Professional（AIP-C01）** 是 AWS 认证体系中的新高度——第一个明确针对 GenAI 开发者的 Professional 级别认证，定位在入门级 AWS Certified AI Practitioner（AIF-C01）之上。

关键事实：
- **Beta 价格**：$150 USD（截止 2026 年 3 月 31 日，已过期）
- **GA 标准价格**：$300 USD
- **当前状态**：GA 版本正常开放，可直接在 AWS 认证官网报名

考试内容覆盖：foundation model 选型策略、RAG 架构设计、Amazon Bedrock 开发与部署、Agents for Amazon Bedrock 编排、fine-tuning 与 RLHF 策略，以及负责任 AI 合规与 bias 检测。对比 AIF-C01 只要求概念性理解，AIP-C01 需要真实编码和架构设计能力，难度接近 Solutions Architect Professional（SAP-C02）。

如果你已持有 SAP-C02 或 MLS-C01，AIP-C01 是 2026 年下半年 ROI 最高的一张证书——澳洲 IT 招聘平台上 GenAI 相关职位已把这个认证列入「优先考虑」条件。

> 来源：[AWS Certified Generative AI Developer – Professional 官方认证页](https://aws.amazon.com/certification/certified-generative-ai-developer-professional/)；[tutorialsdojo.com 引用：「The last day to take the beta version of the exam is March 31, 2026. The price is expected to be $300 USD once the full version is live.」](https://tutorialsdojo.com/free-aws-exam-retake-and-25-off-any-aws-certification-exam-voucher/)

### 4. Databricks Data Engineer Associate 官方考纲 5 月 4 日全面更新

**一句话**：Databricks 自家认证（非微软 DP-750）考纲 5 月 4 日重大刷新，Unity Catalog 与 Structured Streaming 成核心考点，旧版备考资料需更新。

注意区分：这里说的是 **Databricks 官方认证"Databricks Certified Data Engineer Associate"**，不是微软的 DP-750（Azure Databricks Data Engineer Associate）——两者由不同机构颁发，是完全独立的考试体系。

2026 年 5 月 4 日，Databricks 官方发布新版考试大纲。核心变化：

| 考点域 | 权重 |
|------|------|
| Databricks Intelligence Platform | 10% |
| Development and Ingestion | 30% |
| Data Processing & Transformations | 31% |
| Productionizing Data Pipelines | 18% |
| Data Governance & Quality | 11% |

最大变化集中在两处：

1. **Unity Catalog 成为独立考核项**：数据血缘（data lineage）、细粒度权限控制（column-level security）、跨 workspace catalog 管理——这些在 2024 年的旧考纲里几乎没有，2026 年版大幅提权。

2. **Structured Streaming + Auto Loader 的实战深度**：新考纲明确要求考生能处理生产环境中的 late data 问题，Auto Loader 与 checkpoint 管理纳入标准考点。

题型方面从「定义解释」转向「业务场景 → 最优方案」的情景题，纯背概念已经不够用。

**考试规格（未变）**：45 题，90 分钟，$200 USD。

如果你在用 2025 年底前的备考资料，Unity Catalog 和 Structured Streaming 的章节很可能是空白或过时的。建议直接从 Databricks 官网下载最新 Exam Guide（May 4, 2026 版）对标考点后再刷题。

> 来源：[Databricks Certified Data Engineer Associate 官方认证页](https://www.databricks.com/learn/certification/data-engineer-associate)；[Scribd 上的官方考纲文档「Databricks Certified Data Engineer Associate Exam Guide May 4 2026」](https://www.scribd.com/document/1030707656/Databricks-Certified-Data-Engineer-Associate-Exam-Guide-May-4-2026)——题目结构、权重和规格均来自该文件。

### 5. Google Cloud 认证全面迁移至 Pearson VUE：旧渠道已停用，请重新预约考场

**一句话**：Google Cloud 认证从 3 月 2 日起改由 Pearson VUE 负责考试配送，原 PSI/Kryterion 渠道停用，备考 ACE / PCA 的学员需在新平台重新预约。

这条消息已于 3 月 2 日生效，但仍有不少学员不知道——Google Cloud 认证的考试配送商已从 Kryterion/PSI 切换至 **Pearson VUE**（与 Microsoft、CompTIA、AWS 使用同一家考试平台）。

实操影响：
- 考试预约入口：登录 [home.pearsonvue.com](https://home.pearsonvue.com) 搜索 Google Cloud
- 历史备考记录和认证成绩不受影响
- 考试题库会逐步纳入 Google Cloud Next '26 发布的新产品内容（Gemini Enterprise Agent Platform、最新数据分析栈），具体生效时间按考试而定

建议在报名前先查当前 exam guide 是否有「updated」标注，避免备考内容对不上最新题库。

> 来源：[Google Cloud 官方支持页面 — Upcoming Change to Exam Delivery Provider（「Google Cloud certification program is partnering with Pearson as the new test delivery provider... Access to exam registration resumed with the first available test date of March 2, 2026」）](https://support.google.com/cloud-certification/answer/16803278?hl=en)

## 二、免费 / 限免课程

### 1. Microsoft「AI Agents for Beginners」—— 12 课 GitHub 开源课程，完全免费

**适合**：想从零理解并动手构建 AI agent 的开发者、后端工程师、应届毕业生
**链接**：[https://microsoft.github.io/ai-agents-for-beginners/](https://microsoft.github.io/ai-agents-for-beginners/)

Microsoft 2026 年 4 月发布的开源学习项目，GitHub 仓库 `microsoft/ai-agents-for-beginners`，无账号墙，无限制。12 课结构化覆盖：

- AI agent 定义与核心架构
- Tool use 与 function calling
- Planning 与 reasoning chain
- Memory 管理（短期 / 长期）
- Multi-agent 协作系统
- Responsible AI for agents

每节课配 Jupyter Notebook 代码练习，使用 Azure AI Foundry 和 GitHub Model Catalogs 运行 LLM。如果没有 Azure 订阅，GitHub Models 的免费 tier 足够跑完所有练习。

特别推荐原因：课程内容与 AI-103 考点高度重叠（agent orchestration、Azure AI Foundry、responsible AI 防护）——既是免费学习资源，又是 AI-103 备考辅助材料，一举两得。

> 来源：[Microsoft Dev Blog — AI Agents for Beginners Course: 10 Lessons teaching you how to start building AI Agents](https://devblogs.microsoft.com/agent-framework/ai-agents-for-beginners-course-10-lessons-teaching-you-how-to-start-building-ai-agents/)；GitHub 仓库页面显示「12 Lessons to Get Started Building AI Agents」

### 2. Hugging Face AI Agents Course —— smolagents 实战，免费无门槛

**适合**：有 Python 基础、想在本地或云端跑自己的 AI agent 的学员
**链接**：[https://huggingface.co/agents-course](https://huggingface.co/agents-course)

Hugging Face 官方出品，完全免费，内容持续更新。课程带你用 **smolagents** 框架从头搭一个能思考、能调工具的 AI agent，核心章节：

- smolagents 框架架构与工作原理
- Tool calling 与自定义工具开发
- Multi-agent 协作：一个 agent 指挥另一个
- 在 Hugging Face Hub 部署和分享 agent

与 Microsoft「AI Agents for Beginners」的差别：微软课程用 Azure 生态（Azure AI Foundry）；Hugging Face 课程用开源生态（smolagents、开源 LLM、Hub），更适合想在本地或低成本环境跑 agent 的学员。两个都学是最优解——各自覆盖企业级和开源侧。

每个章节都有 code notebook，边看边跑，估计 15-20 小时可以完成全部内容。

> 来源：[huggingface.co/agents-course](https://huggingface.co/agents-course)（Hugging Face 官方课程页）；[ClassCentral 2026 年 Hugging Face 课程目录](https://www.classcentral.com/institution/hugging-face)
