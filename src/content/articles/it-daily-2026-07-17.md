## 一、认证动态

### 1. Google Cloud Digital Leader Beta 考试报名截止后天（7月19日）——今天行动还来得及

**一句话**：Google Cloud Digital Leader Beta 考试报名窗口**截止 2026 年 7 月 19 日**，仅剩两天，全程英文，通过即可获得三年期认证（或为现有认证续期三年）。

这是 GCP 更新 Cloud Digital Leader 考纲之后发布的 Beta 版，主要反映了 Google Cloud Next '26 上发布的新产品方向，包括 Gemini Enterprise Agent Platform 和最新的数据分析栈。Beta 阶段通常不会立即出分，成绩会在数周后统一公布。

为什么值得考虑：
- Cloud Digital Leader 本身是 GCP 认证门槛最低的一张，没有前置条件，不需要写代码
- 正式版考试收费 $200 USD，Beta 阶段部分考场或有折扣（具体看报名页面）
- 通过后直接拿三年效期，比后续参加续期活动省事
- 如果你手上已有 Cloud Digital Leader 认证，通过 Beta 考试可以直接续期三年

报名入口在 Google Cloud 官方认证页面，选 Cloud Digital Leader，再选 Schedule exam：[cloud.google.com/learn/certification/cloud-digital-leader](https://cloud.google.com/learn/certification/cloud-digital-leader)

> 来源：[Cloud Digital Leader — Google Cloud](https://cloud.google.com/learn/certification/cloud-digital-leader)

### 2. Google Cloud 7月8日上线"跳过考试"续期机制——四张证书可用技能徽章换一年

**一句话**：Google Cloud 于 **2026 年 7 月 8 日** 正式推出替代续期路径——持有以下四张认证的学员，在 Google Skills 上完成指定课程和技能徽章后，认证会**自动延期一年**，无需重新参加全程监考考试。

适用认证（目前仅以下四张）：
- Cloud Digital Leader
- Associate Cloud Engineer
- Professional Cloud Architect
- Professional Data Engineer

操作路径：登录 [skills.google.com](https://skills.google.com)，选择与你的认证对应的学习路径，完成指定课程和实操 Skill Badge 后，延期自动触发。要求是你的认证必须处于**有效期内**才能使用这个机制。

背景：Google 官方博客的原文提到，技术技能的"半衰期"已经从约 6 年缩短到约 2.5 年，这是 GCP 引入这一机制的根本原因——与其要求工程师每两年重新坐大考，不如用持续学习来保持认证的时效性。这个机制和 AWS 于 6 月 23 日上线的 AWS Maintain Beta 思路是一致的，两家云厂商几乎同期推出了无需重考的续期方案。

注意事项：
- 只有四张认证目前支持，其他 Professional 级别暂未纳入
- 完成学习后延期幅度是一年（不是两年）
- 目前的 Cloud Digital Leader Beta 考试（上一条）可以续期三年，比这个路径多出两年——如果你手上有 Cloud Digital Leader 且还想省事，Beta 考试反而更划算

> 来源：[New ways to keep Google Cloud certs current — Google Cloud Blog](https://cloud.google.com/blog/topics/training-certifications/new-ways-keep-google-cloud-certifications-current)

### 3. Snowflake COF-C03 已接替 COF-C02——旧版翻译版本月底退场，还在用旧材料的学员速切换

**一句话**：Snowflake SnowPro Core 认证的新版本 **COF-C03** 已于 **2026 年 2 月 16 日** 正式上线（英文版），COF-C02 英文版已于 **5 月 14 日** 退役，COF-C02 的各语言翻译版将于 **7 月 31 日** 彻底退场。

COF-C03 考试格式：
- 100 道题（选择题、多选题、判断题混合）
- 110 分钟作答时间
- 及格线：750 分（满分 1000 分，卷面按比例换算）

相比 COF-C02，COF-C03 增加了哪些新内容：

| 新增/扩充考点 | 说明 |
|---|---|
| Snowflake Cortex AI | 覆盖 AI 功能，但不考 API 写法 |
| Git Integration & CI/CD | Snowflake 上的代码版本控制与持续集成 |
| Cortex ML Models & AI Applications | 机器学习模型和 AI 应用的基本概念 |
| FinOps / Cost Attribution | 成本分配与使用治理 |
| Secondary Roles & Advanced RBAC | 更细粒度的访问控制 |
| Data Sharing 增强功能 | 跨账户 / 跨云数据共享的新机制 |
| Performance Optimization & Query Profiling | 查询性能调优与诊断 |

COF-C02 删除的内容：Delta Sharing 和 Lakehouse Federation 这两块在旧版里有独立考点，COF-C03 中已移除或合并。

如果你现在还在用 COF-C02 的备考材料（无论是 Udemy、官方文档还是第三方题库），需要立刻切换到 COF-C03 版本。旧题库的题型和考点分布已经对不上了。

> 来源：[What's new in the SnowPro Core COF-C03 — Cristian Scutaru, Medium](https://cristian-70480.medium.com/whats-new-in-the-snowpro-core-cof-c03-certification-31dd3e212ac0) · [Snowflake SnowPro Core Quick Facts (2026) — certificationpractice.com](https://certificationpractice.com/exam-overviews/snowflake-snowpro-core-quick-facts)

### 4. AWS AIP-C01 Generative AI Developer Professional Beta 结束，标准考试正式开放

**一句话**：AWS Certified Generative AI Developer – Professional（AIP-C01）的 Beta 阶段已于 **2026 年 3 月 31 日** 结束，标准版本正式开放报名，考试费 **$300 USD**（AWS Professional 级别统一定价）。

考试基本参数：
- **试题数量**：65 道计分题 + 10 道不计分样题，共 75 道
- **及格线**：750 分（满分 1000 分）
- **效期**：通过后认证有效 3 年
- **语言**：英文（其他语言版本陆续推出中）

这张认证考什么：验证候选人将 Foundation Models（FM）集成进应用和业务工作流的实操能力，包括 RAG（检索增强生成）、GenAI 应用在 AWS 上的落地、安全和负责任 AI 的实施。不是考 AWS 通用服务，而是专门针对生成式 AI 应用开发场景。

和 AZ-204 → AI-200 这条 Azure 路线的变化类似，AWS 也在把开发者认证的重心从通用云开发移向 AI 应用开发。AIP-C01 是 AWS 认证矩阵里目前 AI 方向最高级别的专业认证。

备考资源：AWS Skill Builder 上有对应的 Exam Prep 官方备考路径，部分内容可免费访问（订阅版解锁全部），入口在 [skillbuilder.aws](https://skillbuilder.aws)，搜索 "AIP-C01"。

> 来源：[AWS Certified Generative AI Developer – Professional — AWS](https://aws.amazon.com/certification/certified-generative-ai-developer-professional/) · [AWS Certified Generative AI Developer Certification Cost — Dumpsgate](https://dumpsgate.com/aws-generative-ai-certification-cost/)

### 5. Databricks 数据工程师认证 5 月大改版——7 模块结构、AI Agent 新证即将到来

**一句话**：Databricks Certified Data Engineer Associate 认证于 **2026 年 5 月 4 日** 完成了一次大规模改版，考试由原来的 5 个模块重构为 7 个，Governance 权重从约三分之一大幅压缩至 15%，新增了 Lakeflow Jobs、CI/CD、Troubleshooting 三个独立模块。

新旧考试结构对比：

| 旧考试（5 模块） | 新考试（7 模块） |
|---|---|
| Databricks Tooling | Databricks Tooling（保留） |
| ELT with Apache Spark and Python | ELT with Spark（保留） |
| Incremental Data Processing | Incremental Processing（保留） |
| Production Pipelines | **Lakeflow Jobs**（扩充为独立模块） |
| Data Governance（原占比约 33%） | Data Governance（降至 15%）|
| — | **CI/CD**（新增独立模块） |
| — | **Troubleshooting / Monitoring / Optimization**（新增独立模块） |

术语变化：
- Asset Bundles → **Declarative Automation Bundles**
- Repos → **Git Folders**

如果你的备考材料是 2025 年或 2026 年 5 月之前的版本，题库的权重分配和部分术语会和实际考试对不上。Databricks 官方认证页面已更新对应的 Exam Guide。

此外，有消息指出 Databricks 正在准备一张新的 **Context Engineer Associate** 认证，专门针对 Databricks 平台上的 AI Agent 系统设计（信息收集、组装与治理），具体 GA 时间尚未确认，可持续关注 [Databricks 认证页面](https://www.databricks.com/learn/certification)。

> 来源：[Clarification on Updated Data Engineering Associate Exam Content — Databricks Community](https://community.databricks.com/t5/certifications/clarification-on-updated-data-engineering-associate-exam-content/td-p/126760) · [Databricks DE Associate 2026 Syllabus: What Changed — alexcole.net](https://www.alexcole.net/databricks-data-engineer-associate-2026-syllabus-changes/)

## 二、免费 / 限免课程

### 1. IBM SkillsBuild — AI、网络安全、编程方向免费数字证书

**适合**：想零成本获得一张有认可度的 AI 或网络安全方向入门证书的学员
**链接**：[skillsbuild.org](https://skillsbuild.org)

IBM SkillsBuild 是 IBM 的免费学习平台，部分课程同时也在 edX 上提供。2026 年持续提供以下方向的免费课程和数字徽章：

- **AI 方向**：AI Foundations、Generative AI for Developers、Machine Learning with Python
- **网络安全方向**：Cybersecurity Fundamentals、IBM Cybersecurity Analyst（可通过 edX 访问）
- **编程方向**：Python for Data Science、SQL for Data Science、DevOps Fundamentals

IBM SkillsBuild 课程完成后颁发**数字技能徽章（Digital Skill Badge）**，可以上传到 LinkedIn，也可以通过 Credly 分享。徽章包含课程内容、技能描述和验证链接，比普通完成证书更有可信度。

免费的部分是课程本身和技能徽章，部分 edX Professional Certificate 如果要拿证书需要额外付费（$100–$200 USD 不等）。

IBM 的 Cybersecurity Analyst 认证备考路径（含 7 门课）在 edX 上可以完整旁听（Audit），不想花证书费用的学员可以单纯完成课程积累知识。

### 2. Hugging Face Learn — NLP / LLM / Agents 全套课程免费

**适合**：想学习 Transformers、LLM 微调、AI Agents 实操的技术学员
**链接**：[huggingface.co/learn](https://huggingface.co/learn)

Hugging Face 的 Learn 平台提供几门主干课程，全部免费，不需要注册会员：

- **NLP Course**：覆盖 Transformers 库、Datasets、Tokenizers 的使用，从文本分类到文本生成，2026 年 7 月初刚完成了一轮内容更新。课程会教你直接用 `transformers` 库在真实模型上操作，不是讲理论。
- **Deep Reinforcement Learning Course**：从零到强化学习实战，课程用的是 Unity ML-Agents 和 Gymnasium 环境，有专属 Discord 社群。
- **Agents Course**：专注 AI Agent 构建，内容包括 smolagents 框架、工具调用、多 Agent 编排。2026 年是 Agent 相关认证（AWS、Databricks）大量扩充的一年，这门课可以直接对标这批新认证的 Agent 知识模块。

这些课程没有传统意义上的证书，但 Hugging Face 在结业后提供完成标记和 Discord 角色。对于备考 AWS AIP-C01（GenAI Developer Pro）或 Databricks Context Engineer（即将推出）的学员，Hugging Face Learn 的内容覆盖度很高。

### 3. Coursera — Google Cloud Professional Data Engineer 证书备考路径（可免费旁听）

**适合**：准备 Google Cloud Professional Data Engineer 认证，或想系统学 GCP 数据工程的学员
**链接**：[coursera.org/professional-certificates/gcp-data-engineering](https://www.coursera.org/professional-certificates/gcp-data-engineering)

这条专项课程（Professional Certificate）由 Google Cloud 官方在 Coursera 上提供，内容对应 Google Cloud Certified Professional Data Engineer 考试的知识范围。

覆盖主题：
- BigQuery 数据处理和查询优化
- Cloud Dataflow / Dataproc 流式和批式处理
- Vertex AI 机器学习集成
- Pub/Sub 消息系统
- Cloud Composer（托管 Airflow）DAG 编排
- 数据安全与合规

**如何免费旁听**：在 Coursera 注册账号后，选择 Audit，可以访问视频和大部分阅读材料，不能提交评分作业，也不拿证书。如果通过 Financial Aid 申请（提前 15 天提交，写清楚为什么需要资助），审批通过后可以免费获得完整访问权限包括证书。

注意：Coursera 从 2025 年中开始把部分课程的免费预览缩限到 1-2 周，所以优先用 Audit 模式或 Financial Aid，不要指望 7 天试用能完成整条路径。

## 三、JR Academy 学员行动建议

1. **今天报名 GCP Digital Leader Beta**：报名截止 2026 年 7 月 19 日（本周日）。没有前置条件，有 Cloud Digital Leader 认证的学员通过后直接续期三年；没有的学员通过后从零拿到认证。报名入口：[cloud.google.com/learn/certification/cloud-digital-leader](https://cloud.google.com/learn/certification/cloud-digital-leader)。

2. **正在备考 Snowflake 的学员**：立刻确认你用的题库/课程是 COF-C03 版本，不是 COF-C02。COF-C02 英文版早在 5 月就退役了，翻译版也在 7 月 31 日结束。旧版材料里 Delta Sharing 和 Lakehouse Federation 的题不再考，而 Cortex AI 和 CI/CD 是新增高权重考点。

3. **想备考 AWS AI 路线的学员**：AIP-C01 现在已经是标准考试，可以正式开始备考。先去 AWS Skill Builder 免费区域搜 AIP-C01，看 Exam Guide 里的技术域权重，再决定是否需要订阅付费备考题。

4. **Databricks 学员**：如果你的备考资料是 5 月之前的，重新下一版官方 Exam Guide（databricks.com/learn/certification/data-engineer-associate），重点补 Lakeflow Jobs 和 CI/CD 这两个新独立模块，Governance 部分可以减少备考时间了。

5. **想快速拿一张能上 LinkedIn 的免费证书**：IBM SkillsBuild 的网络安全或 AI 方向课程最快 10-15 小时可以完成，通过后直接得到 Credly 数字徽章，可以公开验证。不需要考场，纯线上，零成本入口：[skillsbuild.org](https://skillsbuild.org)。
