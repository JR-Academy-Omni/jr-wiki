## 一、认证动态

### 1. SC-500 Beta 上线：80% 折扣到6月8日，前300名限额

**一句话**：AZ-500 的继任证书 SC-500（Cloud and AI Security Engineer Associate）已于5月15日开放 Beta 报名，折扣码 `VistaSC500` 可让前300名考生享受80%折扣，截止6月8日。

微软在5月中旬正式宣布 **Microsoft Certified: Cloud and AI Security Engineer Associate** 认证，对应考试代码 **SC-500**。这张证书被定位为 AZ-500（Azure Security Engineer Associate）的全面替代——AZ-500 将于2026年8月31日正式退役，届时不再接受新报名或续期。

SC-500 与 AZ-500 最关键的区别在于考点覆盖范围：除了传统的 Azure 云安全基础（身份保护、数据安全、网络防护），SC-500 把 **AI 模型安全**单独列为核心考点，包括 AI 管道的访问控制、模型训练环境安全、以及应对 LLM 特有威胁的防御手段。微软的逻辑是：企业现在普遍把 AI 工作负载跑在云上，负责云安全的工程师不懂 AI 安全就不算合格。

**Beta 报考操作**：

1. 前往 [Microsoft Certified: Cloud and AI Security Engineer Associate](https://learn.microsoft.com/en-us/credentials/certifications/cloud-and-ai-security-engineer-associate/)
2. 点击「Schedule Exam SC-500」
3. 支付页面输入折扣码 `VistaSC500`
4. 前300名可享80%折扣，微软 Associate 级考试标准定价约 **$165 USD**，折后约 **$33 USD**
5. 折扣截止 **6月8日**，或300名额用完为止（以先达者为准）

Beta 考试说明：Beta 阶段的成绩评分周期比正式考试长（通常6-8周），等分数出来后正式认证才会发放；正式 GA 版本预计 **2026年7月** 上线。

> 来源：[New Microsoft Certified: Cloud and AI Security Engineer Associate Certification | Microsoft Community Hub](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-cloud-and-ai-security-engineer-associate-certification/4494117)

### 2. CNCF CARE 新续期制度：6月正式实施，CKA 过了 KCNA 自动续期

**一句话**：CNCF（云原生计算基金会）将于2026年6月正式推行 CARE（Certification Advancement & Recertification Experience）制度，核心变化是：通过高级认证后，相关基础认证自动续期，不再需要单独维护。

目前 CKA（Certified Kubernetes Administrator）和 CKS（Certified Kubernetes Security Specialist）的有效期管理一直是从业者的痛点——CKA 有效期3年，CKS 有效期2年，过了就要重考。很多人手上同时持有 KCNA + CKA，到期后得分别续期。

CARE 制度的关键变化：

- **CKA 或 CKAD 过了/续期 → KCNA 自动续期**，不用再单独重考 Kubernetes and Cloud Native Associate
- **CKS 过了/续期 → KCSA 自动续期**（Kubernetes and Cloud Native Security Associate）
- 2026年1月1日之后拿到相关认证的人，均可纳入 CARE 体系（6月之前考过的，只要之前也持有对应基础认证，可自动并入）

同期，CKA 考试环境已对应 **Kubernetes v1.34**，同时 Pod Security Policy（PSP）在2026年正式从考纲中删除，替换为 Pod Security Standards（PSS）+ Admission Controllers 相关考点。正在备考 CKA 的同学，删掉所有关于 PSP 的笔记，重点改看 PSS 和 `kube-apiserver` 的 admission plugin 配置。

> 来源：[CNCF Introduces CARE Program | CNCF](https://www.cncf.io/blog/2026/03/23/cncf-introduces-a-new-recertification-program-as-kubestronaut-community-surpasses-3500/)

> 来源：[CNCF Revamps Certification Program to Simplify Renewals | Cloud Native Now](https://cloudnativenow.com/features/cncf-revamps-certification-program-to-simplify-renewals/)

### 3. Google Cloud 完成 Pearson VUE 迁移：全球5,000+ 考点，24小时取消政策

**一句话**：Google Cloud 认证已于2026年3月2日完成从 Kryterion 向 Pearson VUE 的全面迁移，全球可用考点从零散增至5,000+，取消/改期截止线从72小时缩短至24小时。

这次迁移是今年 GCP 认证最实质的体验改善。Kryterion 时代，部分地区（包括澳洲一些城市）可选考点非常少；切换到 Pearson VUE 后，悉尼、墨尔本、布里斯班等主要城市的考点选项明显增多，在线监考（OnVUE）也已同步上线。

对正在备考的人有影响的变化：

- **改期窗口**：原来用 Kryterion 需要至少72小时前改期（否则付费），Pearson VUE 只需 **24小时前** 操作，不额外收费
- **考试内容不变**：此次迁移只影响考试平台和界面，题目、通过分数线、时长均无变化
- **在线考试要求**：Pearson OnVUE 的环境检测比 Kryterion 更严格，首次用在线方式考前建议提前跑一次系统测试

GCP 各考试费用维持不变：Associate 和 Professional 级别均为 **$200 USD**。

> 来源：[Upcoming Change to Exam Delivery Provider | Google Cloud Certification Help](https://support.google.com/cloud-certification/answer/16803278?hl=en)

> 来源：[Google Cloud certification exam | Pearson VUE](https://www.pearsonvue.com/us/en/google-cloud.html)

| AZ-204 (Developing Solutions) | 2026年7月31日 | AI-200 (Azure AI Cloud Developer Associate) |

1. **SC-500 Beta 报名窗口只到6月8日**：如果你在做 Azure 安全相关的工作，或者正打算冲 AZ-500 但还没报名，现在用 `VistaSC500` 报 SC-500 Beta 是更合算的路线——折后约$33，比正式版省$130+，而且 AZ-500 反正8月就退役，现在备 AZ-500 不如直接备 SC-500。

2. **今天完成 Microsoft AI Skills Fest 注册**：进 [learn.microsoft.com/en-us/credentials/microsoft-credentials-ai-challenge](https://learn.microsoft.com/en-us/credentials/microsoft-credentials-ai-challenge) 注册，6月8日截止早注册，学习路径按自己节奏跑，完成后领考试券。如果你本周已经报了 SC-500 Beta，AI Skills Fest 的学习路径内容和 SC-500 备考有部分重叠，一石二鸟。
