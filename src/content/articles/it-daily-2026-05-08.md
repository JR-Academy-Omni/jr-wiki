## 一、认证动态

### 1. Microsoft SC-500 Beta 5 月 15 日开放：$45 拿 Cloud and AI Security Engineer Associate

**一句话**：全新 SC-500 beta 定档 5 月 15 日，Beta 折扣约 80% off，正价约 $225 的考试 beta 阶段约 $45，7 月正式发放证书。

SC-500 是 Microsoft 今年推出的新一代安全认证，把传统云安全职责扩展到 AI 模型保护领域。考纲核心：

- **AI 安全设计**：为 Azure AI Studio 和 Copilot Studio 工作负载设计安全边界
- **身份与访问**：基于 Microsoft Entra ID 实施最小权限原则，管理 AI agent 的服务主体和托管身份
- **监控与响应**：用 Microsoft Sentinel 建立 AI 相关的检测规则，结合 Defender for Cloud 做漏洞管理
- **数据与网络隔离**：Private Endpoint + 网络分段，防止模型被旁路访问

Beta 名额有限、先到先得。目前官方备考资源（Microsoft Learn 路径）已同步上线，建议提前 2-3 周刷完。7 月正式考试预计恢复全价约 $225。

持有 SC-200（Security Operations Analyst）或 SC-300（Identity and Access Administrator）的人，有约 40% 的考纲重叠，备考周期可压缩到 2 周。

### 2. AI-200 Azure AI Cloud Developer Associate：与 SC-500 同批 Beta 开放

**一句话**：AI-200 beta 5 月开放，目标人群是在 Azure 上构建、集成、监控 AI 解决方案的开发者，全价约 $165，beta 约 $33。

AI-200 考核的是实际开发技能，不是概念理解：Azure AI Foundry 部署、Azure OpenAI Service 调用（含 RAG 模式）、AI Agent 编排、模型监控与内容过滤配置。

与 AI-900（Azure AI Fundamentals）的区别：AI-900 是概念向，AI-200 要求写代码、配 SDK、理解 deployment 生命周期。已持有 AZ-204（Azure Developer）的人上手快。

**操作建议**：
1. Microsoft Learn 搜索 "AI-200" → 官方学习路径已上线
2. 注册 beta 需要 Microsoft Certification Profile，报名后 3 个工作日内会收到考试链接
3. Beta 考试结果一般 6-8 周后出来，证书注册日期回溯到考试当天

### 4. Google Cloud 完成向 Pearson Vue 迁移：6 月起支持 Chromebook 在线监考

**一句话**：Google Cloud 认证已于 3 月完成从 Kryterion 到 Pearson Vue 的迁移，6 月正式开放 Chromebook 兼容的在线监考模式。

这次平台切换对考生的三个实际影响：

1. **预约入口变了**：全部改走 Pearson VUE（https://home.pearsonvue.com/Google），Kryterion 的账号不再受理 Google Cloud 考试
2. **Chromebook 用户的好消息**：此前 Chromebook 无法参加 Google Cloud 在线监考，6 月起 Pearson OnVUE 将支持 Chromebook，消除了这一限制
3. **考试中心照常**：全球 Pearson 考试中心均已接受 Google Cloud 考试，无需额外操作

当前 Professional Cloud Architect 考纲已加大云原生技术（GKE、Cloud Run、Anthos）的比重至 35%，较上一版本提升约 10 个百分点，刷旧题库的同学需注意。

> 来源：[Upcoming Change to Exam Delivery Provider – Google Cloud Certification Help](https://support.google.com/cloud-certification/answer/16803278?hl=en)

### 5. AWS SCS-C03（Security Specialty）考纲新增 GenAI 安全专项

**一句话**：AWS Certified Security – Specialty 更新为 SCS-C03，专门增加一节生成式 AI 和机器学习安全考点，旧版 SCS-C02 正在退出。

2026 年 AWS 认证家族的最大变化之一是把安全认证与 AI 趋势挂钩。SCS-C03 新增考纲包括：Bedrock 模型的权限隔离与 VPC 终端节点配置、SageMaker 数据加密与模型治理、生成式 AI 应用的 prompt injection 防护基础。

备考时间预估：有 AWS 实操经验 + 持 SAA/SOA 的人约需 4-6 周；从零开始约需 10-12 周。考试费用 $300，无官方折扣码，但购买 Udemy 模拟题可在 Learn Cantril / Neal Davis 课程里找到折扣入口（经常 $10-15）。
