## 1. Google Cloud Digital Leader 更新 Beta 开考：7月19日截止，$59 打四折

Google Cloud 正在对 Cloud Digital Leader 认证进行内容更新，在过渡期内以 **Beta 考试**形式开放报名，**截止日期为 2026 年 7 月 19 日**，距今仅剩 10 天。

**Beta 考试具体数据：**

- 费用：**$59 美元**（标准考试原价 $99，约打四折）
- 题量：约 **75 道**多项选择题
- 考试时长：**2 小时**
- 交付方式：线上监考或线下考场均可
- Beta 成绩公布时间：约 8 月中旬（Beta 结束后数周）

**两类特殊情况值得关注：**

已持有 Cloud Digital Leader 证书的考生：通过 Beta 考试后，证书续期 **3 年**（从通过日起算），相当于花 $59 完成续期，性价比远高于等到到期再付标准费用。Beta 考试不占年内考试次数限制（attempt limits 计数不适用于 Beta），考砸了不计入正式考试失败记录。

Cloud Digital Leader 是 Google Cloud 的入门级商业认证，考核对象是需要理解云计算商业价值的管理者、产品经理和云迁移决策者。技术门槛低于 Associate Cloud Engineer，但更侧重云战略决策与业务对齐，对有意向往 Cloud PM 或技术管理方向发展的候选人有实际认证价值。

打算报考的同学现在就可以在官方页面注册，不要拖到最后几天：

来源：[Google Cloud Digital Leader – Beta Exam](https://cloud.google.com/learn/certification/cloud-digital-leader)

官方原文：*"Registration period: Open through July 19, 2026. Beta exam cost: $59 (40% discount off retail price)."*

## 2. GH-600 GitHub Certified Agentic AI Developer：Beta 尾声，GA 版本本月上线

微软 / GitHub 今年推出 **GitHub Certified: Agentic AI Developer（考试代码 GH-600）**，聚焦开发者在整个软件开发生命周期（SDLC）中操作、监督和集成 AI Agent 的能力。

**当前状态（2026 年 7 月）：**

- Beta 阶段早鸟折扣（80% off，限前 100 名考生）已于 5 月 31 日结束
- **GA 版本预计本月正式发布**；官方训练课 **GH-600T00（Developing in Agentic AI Systems）**于 **7 月 20 日**开放，可作为备考主线材料
- Beta 阶段部分地区（土耳其、巴基斯坦、印度、中国大陆）不可参考，GA 版地区政策请以官方最新公告为准

**GH-600 考核内容：**

- AI Agent 在 CI/CD 和代码审查流程中的集成与监控
- 用 GitHub Copilot、Agent Mode 和 MCP 协议构建、调试 Agentic 工作流
- AI Agent 权限管理与风险控制——包括沙箱策略、人工审批节点设计和日志可观测性

这是 GitHub 首批 AI Agent 专项认证，对正在给团队落地 AI 辅助开发工具链（Copilot + CI/CD + Agent）的工程师和技术 Lead 有直接参考价值。GA 考试没有免费 Beta 优惠了，但 7 月 20 日上线的官方 Learn 模块可以免费学习。

来源：[GitHub Certified: Agentic AI Developer（learn.microsoft.com）](https://learn.microsoft.com/en-us/credentials/certifications/agentic-ai-developer/) · [Study Guide for Exam GH-600](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/gh-600)

## 3. SC-500 Cloud and AI Security Engineer Associate：7月 GA 上线，AI 安全专项认证来了

微软新发布 **Microsoft Certified: Cloud and AI Security Engineer Associate（考试代码 SC-500）**，Beta 阶段于 2026 年 5 月开放（限前 300 名，6 月 8 日截止），GA 版本预计 **2026 年 7 月**正式开放报名。

考试全名 *Implementing End-to-End Security Controls for Cloud and AI Workloads*，覆盖：

- 身份与访问保护（Identity & Access Management）
- 数据安全与分类
- **AI 模型安全与治理**——覆盖 AI 工作负载的输入输出过滤、提示注入防护和模型访问审计，是与现有 Azure 安全证书的最大差异点
- 云端威胁检测与响应（Microsoft Defender for Cloud、Microsoft Sentinel）

**目标受众：** 负责保护云端与混合环境的安全工程师，尤其是系统中承载 AI 工作负载的团队。

**与现有 Azure 安全证书的关系：** SC-500 是独立的新认证，不是对现有证书的直接替换。已持有 AZ-500（Azure Security Engineer Associate）的工程师会发现 SC-500 在 AI 安全域上是明显的扩展补充——特别是 AI 模型和 Copilot 工作流的安全防护部分在 AZ-500 中几乎没有覆盖。
