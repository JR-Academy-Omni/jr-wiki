## 一、认证动态

### 1. AI-200 Azure AI Cloud Developer Associate Beta：80折代码本周五（5月27日）到期

**一句话**：微软 AI-200 Beta 考试首300名用代码 AI200Piedmont 享80%折扣，折后约 $33 USD，截止5月27日，正式版定价 $165，窗口只剩5天。

AI-200（Developing AI Cloud Solutions on Azure）是 AZ-204 的直接接班人。AZ-204 将于7月31日退役，AI-200 的考试重心已完全移向生产级 AI 应用开发，与 AZ-204 相比有三处明显变化：

- **核心技术栈**：不再考传统 Web API 开发，转向 Azure AI Foundry 的 Foundation Model 部署与管理
- **工作负载类型**：重点考容器化 AI 服务的弹性伸缩，以及通过 Azure Service Bus、Event Hub 整合后端异步消息
- **安全与监控**：AI 应用特有的内容安全、Prompt 注入防护、成本监控成为新考域

正式 GA 预计2026年7月。Beta 折扣规则：仅限前300名报名并在5月27日前完成考试，不适用于土耳其、巴基斯坦、印度、中国大陆。

目前 Microsoft Learn 上已同步发布对应的免费学习路径，沙盒实操环境不需要个人 Azure 订阅，直接用学习路径里的免费 Lab 凭证即可。如果你正在备考 AZ-204，本周改考 AI-200 Beta 是性价比最高的选择——内容转型趋势一致，价格是正式版的两成。

> 来源：[Microsoft Community Hub – New Microsoft Certified: Azure AI Cloud Developer Associate Certification](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-azure-ai-cloud-developer-associate-certification/4494116)

### 2. AI-103 Beta 已开考，AI-102（Azure AI Engineer Associate）6月30日退役

**一句话**：AI-103（Azure AI Apps and Agents Developer Associate）Beta 已开放注册，接替6月30日退役的 AI-102，正式 GA 预计2026年6月，考纲完全对齐 AI Foundry + Agent 生态。

AI-102 是目前市场认可度最高的 Azure AI 开发者认证之一，但微软已正式宣布其6月30日关考。继任者 AI-103 的考纲变化幅度是近两年最大的：

| 维度 | AI-102（退役中） | AI-103（Beta 开考）|
|------|----------|---------|
| 核心技术 | Azure Cognitive Services、Bot Service、Form Recognizer | Azure AI Foundry、Agent 编排、Prompt Flow |
| 考试重点 | 传统认知服务 API 调用与配置 | 多智能体系统设计、RAG 架构、Function Calling |
| AI 治理 | 基础 Responsible AI 原则 | 完整评估框架（Azure AI Evaluation SDK）+ 内容安全护栏 |
| 正式定价 | 已关考 | Beta 约 $99，GA 预计 $165 |

五大考域权重：Plan and Manage Azure AI Solutions（25–30%）、Implement Generative AI and Agentic Solutions（30–35%）、Computer Vision Solutions（10–15%）、Text Analysis（10–15%）、Information Extraction（10–15%）。

Beta 折扣代码 AI103Claxton 已于5月7日到期（限额用完），当前 Beta 考试按标准 Beta 价格（约 $99）报名，结果与正式版完全等效。正式版 GA 后预计恢复 $165，现在考有明显价格优势。持有 AI-102 的学员需留意——证书在手，但5月前没有备好 AI-103 的话，6月30日之后 AI-102 就正式停止颁发，只能考 AI-103 补上去。

### 3. Databricks Data Engineer Associate 考纲5月4日大改版：场景题 + Unity Catalog 成必考

**一句话**：5月4日起 DEA 考试全面改版，从"记概念"转向"解业务场景"，Unity Catalog 细粒度权限管理和 PySpark 性能优化成为必考新核心，45题90分钟通过线80%。

Databricks 5月4日正式推出改版后的 Data Engineer Associate 考试，相比旧版有三处根本性变化：

**1. 全面转向场景化实操题**
旧版会问"什么是 Medallion Architecture"，新版问"你有 500GB/天的 IoT 事件流，需要支持 CDC（Change Data Capture），用 Delta Live Tables 还是 Auto Loader？为什么？"选项之间的区别需要真正做过才能区分。

**2. Unity Catalog 成为独立考域**
旧版几乎不考 Unity Catalog，新版新增 Data Governance & Quality 占11%：
- 细粒度访问控制（Column-level Security、Row Filters）
- 数据血缘追踪（Lineage API + Unity Catalog Lineage UI）
- Catalog/Schema/Table 三层权限模型的设计与排查

**3. PySpark 性能优化考实际决策**
不再只考语法，考场景判断：广播 Join vs. Sort-Merge Join 各适合什么数据规模、分区策略对 Shuffle 开销的影响、Spark 读写 Delta 时的 File Skipping 优化。

**考试结构（45题，90分钟，通过线80%）**：

| 考域 | 权重 |
|------|------|
| Databricks Intelligence Platform | 10% |
| Development and Ingestion | 30% |
| Data Processing & Transformations | 31% |
| Productionizing Data Pipelines | 18% |
| Data Governance & Quality（新增核心）| 11% |

如果你用的是5月之前的题库刷题，现在的模拟题和真题差距会比较大——Unity Catalog 和性能调优这两块旧版几乎没有覆盖。Databricks Academy 官方免费学习路径已同步更新，补一轮 Unity Catalog 实操 Lab 是最快的补缺方式。

> 来源：[Databricks Community – Data Engineer Associate Certification 2026](https://community.databricks.com/t5/certifications/data-engineer-associate-certification-in-2026/td-p/142852)

## 二、免费 / 限免课程

### 1. freeCodeCamp × Google Cloud Cybersecurity Certificate：12小时备考课 YouTube 全程免费

**适合**：零云安全基础、想备考 Google Cloud Cybersecurity 认证或入行云安全的学员

**链接**：[freecodecamp.org/news/beginners-guide-to-cloud-cybersecurity/](https://www.freecodecamp.org/news/beginners-guide-to-cloud-cybersecurity/)

这门课由 Google Cloud 官方团队制作，内容完整覆盖 Google Cloud Cybersecurity Professional Certificate 的五个模块：

- **Module 1**：云安全原则与数字化转型（Cloud 共享责任模型、GCP 安全边界）
- **Module 2**：云安全风险管理框架——HIPAA、NIST CSF、SOC 的实际应用场景
- **Module 3**：云威胁检测、响应与恢复（监控告警配置、日志分析、事件溯源）
- **Module 4**：针对云环境的攻击类型与防御机制（SQL 注入、DDoS、权限提升）
- **Capstone**：实战场景演练，产出可放 Portfolio 的云安全案例报告

全程在 freeCodeCamp YouTube 频道播放，无需注册账号。看完后可直接去 [Google Skills](https://www.skills.google/paths/419) 注册 Google Cloud Cybersecurity Certificate 认证路径，官方实操 Lab 有 GEAR 计划每月35积分可用（同样免费）；最终认证考试在 Pearson VUE 完成，今年2月已完成从 Kryterion 的迁移，全球5000+考场可预约。

### 2. Hugging Face 官方 AI Agents Course：全程免费，完成后发结业证书

**适合**：有 Python 基础、想掌握 AI Agent 开发或备战 AI 工程师职位的中级开发者

**链接**：[huggingface.co/learn/agents-course](https://huggingface.co/learn/agents-course)

Hugging Face 2026年旗舰公开课，零费用，完成各单元作业后发放可上传 LinkedIn 的结业证书。三大框架全覆盖：

- **smolagents**（Hugging Face 官方轻量 Agent 框架）：Tool Calling、Code Agent、Multi-Step 推理链
- **LangGraph**（多智能体工作流编排）：状态机式 Agent 图、Human-in-the-Loop 设计
- **LlamaIndex**（RAG 与知识库连接）：Retrieval 管道、Index 策略、Hybrid Search

每个单元结构：视频讲解 + 书面教程 + 互动测验 + 实战 Lab（环境在 Hugging Face Spaces，不需要自己配 GPU）。课程最后有带 Leaderboard 的 Agent 竞赛环节，跑出来的排名能直接放简历。

完成 Unit 1（约2小时）即可拿 Fundamentals 证书；完成全部单元并通过评分作业拿 Advanced 证书。内容直接对应 AI-103 的 Agent 编排考域，也是目前市面上覆盖最完整的 AI Agent 免费课程之一。
