## 一、认证动态

### 1. DP-100 倒计时 4 天：Azure Data Scientist Associate 6 月 1 日正式停考

**一句话**：DP-100 及 Azure Data Scientist Associate 认证将于 2026 年 6 月 1 日 23:59 CST 停考，届时无法再报名、无法续期，想拿替代证 AI-300 需另行备考。

如果你手上还没考 DP-100 但有考试计划，现在已经来不及报名正式考——普尔文会及时告知余量，但大多数考点的 6 月 1 日槽位已满。更现实的做法是转换跑道，直接备考新证 AI-300。

已经持有 DP-100 的同学不用慌：现有证书继续有效直到原本的 2 年到期日，会在 Microsoft Learn Profile 上正常显示；**变化是停止续期**——证书到期后就彻底失效，不会自动续成 AI-300。要延续 Azure ML 认证路径，必须单独过 AI-300。

从考试内容看，两张证的侧重差别很大：DP-100 考的是数据科学建模（特征工程、训练管道、Jupyter 实验），AI-300 考的是 MLOps 生产化运维（CI/CD 流水线、模型漂移监测、Azure Foundry 上的 GenAI 应用部署）。转型时间最短估计要 6-8 周认真备考。

> 来源：[Microsoft Learn — Azure Data Scientist Associate](https://learn.microsoft.com/en-us/credentials/certifications/azure-data-scientist/)（搜索结果直接引用：*"The certification, related exam, and renewal assessments will retire on June 1, 2026, and you will no longer be able to earn or renew this certification after this date."*）

### 2. AI-300 MLOps Engineer Associate：本月 GA，接棒 DP-100 的新方向

**一句话**：微软 AI-300 Machine Learning Operations (MLOps) Engineer Associate 从 Beta（3 月开放）进入 GA，预计 5 月内正式上线，考 40–60 题、150 分钟、过线 700/1000。

AI-300 是目前微软官方推荐的 DP-100 继任认证。考试核心能力区分得很清楚：

- **设计并实现安全可扩展的 MLOps 基础设施**（GitHub Actions、Bicep、Azure CLI 自动化资源配置）
- **管理模型版本和注册表**，实现训练、部署端到端自动化
- **使用 Microsoft Foundry 运营 GenAI 解决方案**
- **实现 RAG 管道优化**和微调模型的成本/准确率平衡
- **GenAI 可观测性**（监控幻觉率、延迟、成本漂移）

Beta 阶段（截止 4 月 2 日）的折扣码 `AI300Meridian` 前 300 名 8 折，现已失效；GA 后标准价格与其他 Associate 级认证一致，约 USD 165。

> 来源：[Microsoft TechCommunity — New Certification for MLOps Engineers](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-certification-for-machine-learning-operations-mlops-engineers/4494111)；[Microsoft Learn AI-300 Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/ai-300)

### 3. DP-800 SQL AI Developer Associate：T-SQL 里直接跑 RAG，5 月 GA

**一句话**：DP-800 Developing AI-Enabled Database Solutions 本月从 Beta 进入 GA，专为想把 AI 能力直接内嵌进 SQL Server / Azure SQL 的数据库工程师设计。

这张证覆盖的核心技能在国内外都比较稀缺：

- **T-SQL AI 函数**：直接在 SQL 里调用 `AI_GENERATE_EMBEDDINGS()`、`VECTOR_DISTANCE()` 等新函数生成和检索向量
- **语义搜索**：基于向量索引实现近似最近邻（ANN）搜索，完全在数据库层处理
- **RAG 工作流**：不用出 SQL 就能接 Azure OpenAI，把检索增强生成嵌进存储过程
- **与 Azure AI Services 集成**：文本分类、情感分析、图像描述直接写进 T-SQL pipeline

考试格式：40–60 题，总座位时长 120 分钟（实际作答 100 分钟），过线 700/1000，题型含拖拽、案例题和交互式组件。Beta 期间折扣码 `DP800Belzoni` 前 300 名 8 折（截止 4 月 3 日，已失效）。GA 后价格约 USD 165。

如果你现在的工作是 DBA 或 SQL 开发，DP-800 是横向扩展进 AI 工程的最低摩擦路径——不用重学 Python，SQL 技能直接复用。

> 来源：[Microsoft TechCommunity — SQL AI Developer Associate Certification](https://techcommunity.microsoft.com/blog/skills-hub-blog/new-microsoft-certified-sql-ai-developer-associate-certification/4494113)；[certqueen.com 认证介绍](https://www.certqueen.com/news_Microsoft_Introduces_New_AI_and_Data_Certifications_AI-300_DP-750_and_DP-800_5348.html)

## 二、免费 / 限免课程

### 1. Google Get Certified 2026（GEAR Program）：9 周免费，含 $200 考试券

**适合**：已在 Google Cloud 客户公司工作、想系统拿 GCP 认证但不想自费的工程师

**链接**：[developers.google.com/program/gear/getcertified](https://developers.google.com/program/gear/getcertified)

这个项目的核心卖点：加入完全免费，完成后拿到一张覆盖全部考试费（通常 USD 200）的 voucher。具体安排是每周约 9 小时自学 + 最多 6 次在线直播答疑，大约 9 周跑完一个 cohort。

入选条件有一点需要注意：**必须用公司邮件（Google Workspace 账号）申请**，且公司必须是 Google Cloud 客户。个人邮件和学校邮件不受理。如果满足这个条件，申请后 7 天内会收到是否入选的邮件，cohort 开始前分配名额。

对刚入职使用 GCP 的公司、或者公司 IT 预算紧、不想自掏考试费的同学，这条路值得走一遍。

> 来源：[Google GEAR Get Certified Program](https://developers.google.com/program/gear/getcertified)（搜索结果引用：*"Joining costs nothing, and once you're accepted into the cohort, your training is fully funded, with the opportunity to earn a certification exam voucher when finished"*；*"you must be employed by a Google Cloud customer and use a valid corporate email address linked to a Google Workspace account"*）

### 2. Hugging Face Agents Course：免费 · 官方证书 · 覆盖 LangChain/LlamaIndex/smolagents

**适合**：想系统学 AI Agent 开发框架、目前主要做 Python AI 应用的工程师

**链接**：[huggingface.co/agents-course](https://huggingface.co/agents-course)

Hugging Face 官方推出这门课的理由很清楚：市面上大量"AI Agent 教程"只教 ReAct 的基本循环，不教框架差异和实际部署。这门课的结构是**每单元带直播答疑 + 书面材料 + 交互式测验**，完成后颁发 Hugging Face 官方认证，不是第三方颁的。

课程内容里值得关注的三块：

- **LangChain vs LlamaIndex vs smolagents 对比实现**：同一个任务用三个框架都跑一遍，看清楚各自的抽象层级和适用场景
- **真实业务案例**：自动化 SQL 查询 Agent、代码生成 Agent、复杂文档摘要 Agent
- **工具调用深度**：Tool calling 规范、function schema 设计、错误重试策略

完全免费，按自己节奏学，不限时间。

> 来源：[Hugging Face — Agents Course](https://huggingface.co/agents-course)；[@burtenshaw Hugging Face 公告](https://huggingface.co/posts/burtenshaw/334573649974058)（原文：*"We're launching a FREE and CERTIFIED course on Agents!"*）
