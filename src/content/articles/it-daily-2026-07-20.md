## 一、认证动态

### 1. AI-200（Azure AI Cloud Developer Associate）本月正式开考 — AZ-204 倒计时 11 天必须今天选边

**一句话**：微软 AI-200（Azure AI Cloud Developer Associate）按计划于 2026 年 7 月进入正式发布（GA）阶段，AZ-204 将于 7 月 31 日 11:59 PM CST 停止报名，今天正式进入两张证书"并行倒计时"阶段。

**两条路，今天必须选一条**

还在观望"考 AZ-204 还是 AI-200"的学员：今天是决策截止日。AZ-204 退役日是 7 月 31 日，11 天后。这不是说"还有 11 天可以备考"——Prometric / Pearson VUE 热门考场时段往往提前数天甚至一周满座，实际上今天就应该预约座位了。

AZ-204 和 AI-200 共用约 60% 的考点内容，但方向截然分叉：

| 维度 | AZ-204 | AI-200 |
|------|--------|--------|
| 核心定位 | 通用 Azure 应用开发 | AI 集成 Cloud 开发 |
| 计算 | App Service / ACI / AKS | 同上 + AI Functions |
| 存储 | Blob / CosmosDB / Table | 同上 + 向量索引 |
| 安全 | Key Vault / Managed Identity | 同上 + AI 访问治理 |
| 新增考点 | — | Azure OpenAI / AI Foundry / Responsible AI |

AZ-204 到 AI-200 不是"旧证失效"，微软明确表态持有者的 AZ-204 证书不会被注销，到期续期机制不变。但从就业市场角度，招聘方今年已在 JD 里开始写 AI-200 方向的要求，AZ-204 会逐渐成为"上一代"背书。

**怎么选**：
- **还没开始备考**：直接备考 AI-200，AI-200 内容更新、更符合 2026-2027 岗位要求，11 天从零通过 AZ-204 成功率极低
- **AZ-204 已备考 60% 以上、考试资料基本齐全**：可以冲 7 月 31 日前，但**今天就要预约考场**，不要拖
- **已持有 AZ-204**：无需任何操作，证书有效，下个续期周期再考虑是否迁移到 AI-200

AI-200 考试费用约 $165 USD（以 Microsoft Learn 官方报名页为准，各国有差异）。备考资源：Microsoft Learn 免费路径已覆盖全部考点，详见本文第二部分。

### 2. AWS Certified AI Practitioner（AIF-C01）：Q1 报考量同比 +34%，Agentic AI / MCP 已入题

**一句话**：AWS AIF-C01（AI Practitioner，$100）2026 年一季度报考量同比增长 34%，近期考生反映新增 Agentic AI 架构和 MCP（Model Context Protocol）框架相关考题，用旧版材料备考的学员需更新学习计划。

**AIF-C01 是什么，为什么值得关注**

AWS AIF-C01 是 AWS 认证体系里门槛最低的 AI 方向入口：65 道题、120 分钟时限、考费 $100、不要求任何前置认证或工作经验。按照 AWS 官方统计，有充分备考的候选人通过率约 80%，是 AWS 所有认证里性价比最高的 AI 背书。

考试涵盖五个领域：AI/ML 基础概念、生成式 AI 和 Foundation Models、Amazon Bedrock 及生成式 AI 应用、负责任 AI、AWS AI/ML 服务概览。没有写代码的要求，属于理解层面的认知考试。

**2026 年新增考点（近期考生反馈）**

三个方向的内容在 2026 年明显变重：

1. **Agentic AI 架构**：ReAct（Reasoning + Acting）编排模式原理、Amazon Bedrock Agents 工作流、多 Agent 协作场景下的工具调用
2. **Kiro**：AWS 2026 年推出的 Computer Use / 自动化基础设施产品，考题开始涉及 Kiro 的使用场景和与 Bedrock 的集成方式
3. **MCP（Model Context Protocol）**：开放协议，规范 AI 模型与外部工具/数据源的接入方式；AWS 服务侧（Bedrock、Q Developer）已开始支持 MCP，考题会考你对 MCP 框架设计意图的理解

**备考建议**

刷题用 Tutorials Dojo 的 AIF-C01 题库（持续更新，收费约 $15），基础知识补充看 AWS Skill Builder 上的免费备考路径（搜 "AWS Certified AI Practitioner Exam Prep"，约 6 小时，免费）。Agentic AI 和 MCP 部分直接看 Amazon Bedrock 官方文档的 Agents 章节，比任何第三方材料都准。

全球 AWS 持证者薪资平均比同岗位非持证者高 26%（Global Knowledge 2026 IT Skills & Salary Report 数据）。AIF-C01 是进 AWS AI 生态的第一个门票，后续可以继续升 AIP-C01（Generative AI Developer Professional，考费 $300）。

### 3. Databricks 推出两张新认证：Generative AI Engineer Associate + Context Engineer Associate

**一句话**：Databricks 今年上半年新增了两张针对 AI 工程师的认证，直接命中 2026 年招聘市场里 AI Agent 相关岗位的技能要求，两张证书的免费概述课程在 Databricks Academy 均可领取。

**Generative AI Engineer Associate**

这是 Databricks 针对"在 Databricks Lakehouse 上构建生成式 AI 应用"的工程师设计的认证。考点集中在：
- Mosaic AI 模型服务的部署与监控
- RAG（Retrieval Augmented Generation）管道构建：Unity Catalog + Delta Lake + Vector Search
- 生成式 AI 应用的评估框架（Mosaic AI Evaluation / Judge Metrics）
- Model Registry 管理 LLM 版本和提示模板

适合目标：已经在做数据工程或 ML 工程，想进入 AI 应用开发方向，且公司数据栈基于 Databricks 的人。

**Context Engineer Associate**

这是 Databricks 在 2026 年 AI Agent 热潮下推出的全新方向，也是整个认证市场里最新的品类之一。"Context Engineering"指的是管理和治理 AI Agent 能访问的信息质量：

- Agent 的 context window 信息筛选与优化（减少噪声、增加相关性）
- Unity Catalog 对 Agent 数据读取的权限控制和审计
- 向量数据库的数据新鲜度管理（embedding 版本同步、stale data 检测）
- 多 Agent 系统中 context 传递的标准化设计

用 Databricks 自己的表述："validates your ability to design, assemble, and govern information for AI agent systems on Databricks."

**两张证书怎么选**

如果你是数据工程师想往 AI 方向走：先拿 Data Engineer Associate（2026 版考纲 5 月刚大改，7 个模块），然后视公司技术方向选 GenAI Engineer 或 Context Engineer。两者不互为前置条件。

备考起点都在 [academy.databricks.com](https://academy.databricks.com)，注册账号免费领取各认证的 Overview Course，里面有完整考纲清单和推荐学习路径。

> 来源：[Databricks Certification 官网](https://www.databricks.com/learn/training/certification) · [Databricks Blog: Getting Databricks Certified — Free Overview Courses](https://www.databricks.com/blog/getting-databricks-certified-now-easier-free-overview-courses) · [Tredence: Generative AI Certifications 2026](https://www.tredence.com/blog/generative-ai-certifications-2026)

### 4. Google Professional Machine Learning Engineer：Cloud Next '26 后考纲更新，Gemini 管道入题，薪资溢价 18-25%

**一句话**：Google Cloud Professional Machine Learning Engineer（$200）已将 Google Cloud Next '26 新发布内容纳入考纲，Gemini Enterprise Agent Platform 和 Vertex AI 最新 MLOps 工作流是新增重点考点，通过者平均薪资 $165,000–$195,000（美国市场），溢价幅度 18-25%。

**这张证是什么定位**

Google Cloud Professional ML Engineer 是 GCP 认证体系里技术含量最重的 AI / ML 方向证书，属于 Professional 级（非入门 Foundational 或 Associate 级）。考试要求候选人有 2 年以上实际 ML 项目经验，熟悉 TensorFlow 或 PyTorch，能够端到端设计和部署 ML 系统。

**2026 年考纲新增内容**

Google Cloud Next '26 之后，官方更新了考纲，新增了以下高权重考点：
- **Gemini API 在 Vertex AI 上的调用与微调**：包括 Gemini 1.5 Pro / Flash 的参数微调、API 配额管理、提示工程最佳实践
- **Vertex AI Agent Builder / Gemini Enterprise Agent Platform**：多 Agent 系统的编排与监控
- **Vertex AI Pipelines 的 MLOps 集成**：CI/CD for ML 管道、模型监控和漂移检测
- **生成式 AI 模型的新评估指标**：LLM-as-judge 评分方法、RAGAS 框架应用
- **Responsible AI 在生产系统的落地**：公平性检测工具（Explainable AI / Model Cards）、数据治理合规

**考试细节**
- 考试费：$200 USD
- 题型：50–60 道单选 / 多选题，120 分钟
- 准备时间：有 ML 基础、2 年以上经验的候选人建议 4-6 周，100-120 小时
- 免费备考起点：[Google Cloud Skills Boost — Machine Learning Learning Path](https://cloudskillsboost.google/paths/17)（300+ 免费徽章课程，ML 方向齐全）

Generative AI Leader 和 Professional ML Engineer 的差别：Generative AI Leader 是业务向、无需技术背景、$200 进门票；Professional ML Engineer 是工程向、需要真实项目经验、薪资溢价更高。不同起点的学员选不同方向。

## 三、JR Academy 学员行动建议

1. **今天必须定方向 — AZ-204 还是 AI-200**：11 天后 AZ-204 停考。已备考 60% 以上：今天预约 7 月 31 日前的 Prometric 座位；还没开始：直接备考 AI-200，打开 [Microsoft Learn AI-200 路径](https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-developer/) 开始第一模块。

5. **完全零基础、想走 IT 路的学员**：本周开 [CS50x](https://cs50.harvard.edu/x/)，坚持 11 周完成所有作业，免费拿哈佛 CS 证书。3 个月后你会有实质性的编程能力，比花几千块报付费入门班效果更实在。
