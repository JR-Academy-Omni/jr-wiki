01 · 大模型融资
Anthropic 完成 $65B 系列 H估值直逼 $1 万亿
图 2 · 海报 01
一句话：Altimeter / Sequoia 领投，run-rate 营收已达 $47B

Anthropic 宣布完成 $65B 系列 H 融资，投后估值达 $965B，仅差一步便触及万亿美元门槛。本轮由 Altimeter Capital、Dragoneer、Greenoaks 和 Sequoia Capital 联合领投，时间节点恰好与公司 run-rate 营收突破 $47B 同步披露，向市场传递了强烈的商业化加速信号。

这笔融资的规模在 AI 行业创下新纪录，也宣告了一个重要的竞争格局：AI 大模型市场正在向「OpenAI vs Anthropic」双巨头格局收拢。两年前 Anthropic 还被视为「OpenAI 的有力挑战者」，而今已晋升为同量级的行业塑造者。资本的大规模涌入将直接转化为算力扩张、人才竞争和产品迭代速度。

对于开发者和企业用户而言，此轮融资带来的最直接利好是算力与服务。Anthropic 此前已宣布与 SpaceX 达成算力合作，获得超 220,000 块 NVIDIA GPU 的算力支持，Claude Code 和 Claude API 的并发上限与响应速度将随之提升。估值层面，$965B 的数字也意味着 Anthropic 可能在 12-18 个月内推进 IPO，届时生态开放程度与定价策略或将发生根本性变化。

来源：anthropic.com · 9to5mac.com
02 · 模型发布
Claude Opus 4.8 正式上线代码 Agent 得分升至 69.2%
图 3 · 海报 02
一句话：Fast Mode 速度提升 2.5 倍，独立工作能力再升级

Anthropic 于 5 月 28 日正式发布 Claude Opus 4.8，这是继 Opus 4.6 之后仅相隔约两个月的新一代旗舰模型。官方公告显示，Opus 4.8 在 agentic coding 标准评测中得分从 64.3% 升至 69.2%，提升幅度约 8%；Fast Mode（快速推理模式）速度提升约 2.5 倍，同时模型在长时间独立任务执行方面表现更强，「判断力更锐利、对自身进度的自我感知更诚实」。

这次迭代最值得注意的信号不是数字本身，而是迭代节奏。Anthropic 正在把「两个月一次旗舰更新」变成常态，这与 OpenAI 的 GPT-5.5 Instant 快速推进同步，折射出头部玩家已从「年度大版本」转向「季度迭代」的产品竞争模式。Agentic Coding 得分逼近 70%，意味着模型在无人监督情况下完成多步骤代码任务的可靠性显著提升，「AI 替代初级开发者」的技术可行性正在快速积累。

对于 IT 从业者而言，Claude Opus 4.8 的实际影响体现在三个维度：第一，代码审查、Bug 修复、单元测试生成等场景的准确率提升，减少人工确认轮次；第二，长任务（如重构整个模块、批量处理文档）的中途「卡壳」概率降低；第三，Fast Mode 的速度提升使得实时 pair programming 体验更接近真正的「AI 同事」而非「等待服务」。建议已在使用 Claude Code 的工程师尽快切换到 Opus 4.8 基线。

来源：anthropic.com · 9to5mac.com
03 · 中国 AI
DeepSeek 首轮融资将收官估值 $50B，国家队 + 腾讯入局
图 4 · 海报 03
一句话：大基金三期领投，DeepSeek 正式走向「国家战略资产」

据南华早报援引三位知情人士透露，DeepSeek 即将完成其历史上首轮外部融资，估值高达 $50B。本轮由中国集成电路产业投资基金三期（「大基金三期」）旗下 AI 专项机构牵头，腾讯控股联合参与，资金将主要用于算力扩张与模型研发。此前 DeepSeek 以完全自有资金运营，此次引入国家级投资者，标志着其战略地位从「独立 AI 实验室」升格为「国家核心 AI 竞争资产」。

这一变化的意义超过融资本身。大基金三期的参与历来伴随着国家资源优先分配——包括芯片采购通道、政府数据授权以及与国有企业的优先合作权。更重要的是，此前在美国制裁环境下 DeepSeek 自主研发低成本高效模型（V3、V4 系列）已被证明是技术可行路径；获得国家背书后，DeepSeek 在算力短缺约束下仍能保持竞争力的底气将更足。DeepSeek-V4 系列尽管在 NIST 评测中落后顶级硅谷模型约 8 个月，但凭借极致的成本效益仍吸引了大量 API 用户。

对开发者和企业而言，DeepSeek 国家化意味着双重信号：利好层面，开放 API 定价可能进一步下探，低成本高质量推理将更容易获取；风险层面，地缘政治敏感度上升，使用 DeepSeek 作为核心业务底座的澳洲企业需要评估数据主权与合规风险，特别是在政府和金融行业客户场景下。

来源：scmp.com · bloomberg.com
04 · 科技巨头
Meta 裁员 8000 人押注 AI同步测试 $7.99/月 AI 订阅
图 5 · 海报 04
一句话：10% 员工离职，7000 人转岗 AI 相关岗位

Meta 5 月启动自公司成立以来规模最大的一次战略重组：约 8000 名员工（占全体员工约 10%）被裁员，另有 7000 人的工作职责将向 AI 相关方向迁移。与此同时，Meta 宣布将在新加坡、危地马拉、玻利维亚等市场试点 AI 订阅服务，定价分为 Meta One Plus（$7.99/月）和 Meta One Premium（$19.99/月）两档，这是 Meta 首次尝试广告以外的 AI 直接付费商业模式。

从宏观视角看，这是一次罕见的「同步进行裁员与商业模式创新」操作。Meta 的逻辑是：AI 替代了部分重复性内容审核、数据标注和基础工程工作，同时 AI 功能（图像生成、智能助手、个性化推荐）正成为平台核心竞争力，因此人才结构必须随之重塑。从 Threads 到 WhatsApp 到 Instagram，Meta AI 的触点已遍及超 30 亿日活用户；订阅服务的尝试本质上是在测试「用户为 AI 付费」的意愿，而非纯商业化试验。

对 IT 求职者而言，这次裁员短期内会向人才市场释放一批来自 Meta 的工程师和产品经理，特别是数据工程、推荐系统和内容理解领域，建议关注相关招聘动态。长期来看，Meta AI 开放的 API 生态（Llama 系列持续更新）和订阅模式若成功将提升整个行业对「AI 功能溢价」的预期，对从事 B2C AI 产品开发的创业者是重要参考信号。

05 · 行业预判
DeepMind CEO：AGI 最快 2029 年Google I/O 推 Gemini 3.5 Flash 降价
图 6 · 海报 05
一句话：「处于奇点山脚下」，Ultra 订阅从 $250 降至 $200

Google DeepMind CEO Demis Hassabis 在 Google I/O 2026 开发者大会上表示，人类正站在「奇点山脚」，AGI 的到来时间他预期在 2030 年前后，但 2029 年越来越成为一个现实可能。他认为自主 Agent 将是下一代 AI 的核心形态——能够跨平台串联计划、执行真实世界任务（如预订为期一周的行程、执行新药研发方案）。与此同时，Google 在 I/O 上推出更便宜的 Gemini 3.5 Flash，并将 Gemini Ultra 订阅从 $250/月降至 $200/月，同步宣布以约 $80-90M 授权费挖角 Contextual AI 20 余名研究员。

AGI「4 年内」的公开承诺是本次 I/O 最值得关注的信号。历史上 AI 领域的 timeline 预测屡遭质疑，但本轮 Hassabis 的表态具有不同寻常的背景：一是当前架构（Transformer + RLHF + large-scale RL）已被证明在多数知识密集任务上接近或超越人类水平；二是自主 Agent 的实际工程化正在快速推进（包括 Claude Managed Agents、OpenAI Codex、Grok Build 等）；三是算力、数据和训练方法论均处于加速改进期。这使得「4 年」不再是科幻预言，而是越来越接近可操作的工程路线图。

对 IT 专业人士的启示最为直接：若 AGI 在 2029 年前后成真，留给技术人员完成「人机协作能力建设」的窗口期仅剩 3 年。Gemini Flash 降价则是更立即的信号——AI API 成本持续下降意味着「用 AI 做产品」的门槛已不再是财务问题，而是能否设计出真正有价值的 AI 应用场景。建议正在学习 AI 开发的同学优先关注 Agent 框架、工具调用设计和长任务规划能力，这将是未来 3 年最抢手的工程技能方向。
