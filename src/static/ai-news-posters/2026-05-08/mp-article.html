01 · OpenAI 大更新
GPT-5.5 Instant 取代 GPT-5.3 成 ChatGPT 新默认
图 2 · 海报 01
一句话：ChatGPT 默认引擎全面切换，图像理解与多步任务完成能力全面升级

GPT-5.5 Instant 是 OpenAI 在 GPT-5.5 系列中专为高速响应优化的版本，本次升级将其推上 ChatGPT 的默认位置，取代此前服务数亿用户的 GPT-5.3 Instant。从用户侧来看，这次切换几乎无感——不需要任何手动操作，所有 ChatGPT Free、Plus 和 Pro 用户都将在下一次对话中自动使用新模型。

新模型的核心改进集中在四个维度：准确性与简洁度的双重提升、图像理解深度（包括复杂图表和截图解析）、STEM 问题的推理链路，以及对"何时触发网页搜索"的判断逻辑优化。后者尤为关键：GPT-5.3 有时会在不必要的情况下触发搜索，造成响应延迟；5.5 Instant 通过更精准的意图识别减少了冗余搜索调用。

从战略层面看，这次默认模型切换与 GPT-5.5 进入 Amazon Bedrock（昨日报道）形成双线推进格局：B2B 侧通过 AWS 企业生态快速渗透，B2C 侧通过 ChatGPT 全量覆盖用户基础。OpenAI 此举的时间节点也值得关注——正值 Anthropic 宣布 SpaceX 算力合作、Grok 4.3 发布的密集竞争窗口，节奏感十足。

对开发者而言，若已在生产环境使用 GPT-5.3 Instant API，建议在迁移至 5.5 Instant 前先在 staging 环境做回归测试，重点关注长文本摘要、代码生成和多轮对话场景的输出一致性。OpenAI 暂未公布 5.5 Instant 相比 5.3 的具体价格变化，成本评估需在 API 控制台实测后确认。

来源：techcrunch.com · openai.com
02 · 算力竞赛
Anthropic 签约 SpaceX，借 Colossus 超算获 22 万张 GPU
图 3 · 海报 02
一句话：马斯克曾称 Claude「邪恶」，Anthropic 却签约跑在他的英伟达超算上

Colossus 1 是 SpaceX 在田纳西州孟菲斯建设的超算集群，原本服务于 xAI 的 Grok 模型训练。Anthropic 此次签约意味着将获得其中超过 300 兆瓦的独立算力分区，这相当于为 Claude 系列模型的推理服务新增了一个与现有 AWS/GCP 部署规模相当的算力底座。根据协议，新增算力将在当月内完成接入。

Anthropic 随即宣布了一系列速率调整：Pro、Max、Team 和 seat-based Enterprise 套餐的 Claude Code 和 API 速率限制全部翻倍；取消了之前在高峰时段（通常为美东时间工作日上午）对 Pro 和 Max 用户的限速措施；Opus API 的调用配额也同步上调。这些变化对重度用户群体（尤其是使用 Claude Code 的开发者）有直接感知。

此次合作的背景颇为戏剧性。马斯克此前在社交媒体上多次公开批评 Anthropic 及其 Claude 模型，称其"危险"或"政治化"，甚至在 xAI 与 Anthropic 的竞争叙事中将两者对立。而如今 Anthropic 的工作负载实际上将跑在马斯克控制的物理基础设施上——这笔交易清晰地表明：在算力需求面前，商业利益优先于公开的口头竞争。

对于关注 AI 基础设施的观察者而言，Anthropic 此举也具有供应商多样化的信号意义。此前 Anthropic 的主要云合作方是 AWS（战略投资者）和 GCP（Google 是另一主要投资者），引入 SpaceX 私有数据中心作为第三路算力，降低了对单一云厂商的依赖。

来源：bloomberg.com · benzinga.com
03 · 中国 AI 融资
DeepSeek 首次外部融资，估值冲 450 亿美元
图 4 · 海报 03
一句话：国家大基金 + 腾讯联合领投，估值从数周前 200 亿翻倍至 450 亿

DeepSeek 自 2023 年成立以来始终保持"自给自足"的姿态，此前从未引入外部机构投资。TechNode 于 5 月 7 日引述消息人士称，中国国家集成电路产业投资基金（俗称"大基金"）正在洽谈领投这笔融资，腾讯以及其他若干机构也在谈判名单中，但最终投资方阵容尚未敲定。

450 亿美元的目标估值令市场震惊。仅仅数周前，市场对 DeepSeek 的非正式估算还在 200 亿美元左右。这轮估值飙升的直接催化剂是 DeepSeek V4 系列——于今年 4 月底发布的 V4-Flash 以 284 亿激活参数（总参数 1 万亿+）在多项 NIST 基准上接近前沿水平，同时每百万输入 token 的定价仅为 1.74 美元，远低于 GPT-5.5 的 5 美元和 Claude Opus 4.7 的报价。

国家大基金的参与具有多重含义。一方面，国家资本背书将显著增强 DeepSeek 的算力采购谈判筹码，尤其是在华为 Ascend 900 系列芯片的供应优先级上；另一方面，这也将进一步加剧国际社会对 DeepSeek 数据安全与合规问题的审查。美国、澳大利亚等国家的政府机构对涉及国家资本背景的中国 AI 工具的使用限制已在讨论中。

对正在评估使用 DeepSeek V4 API 的企业来说，此次融资不会立即改变技术层面的使用体验，但如果国家大基金最终入股并被正式披露，合规部门对使用风险的判断可能需要重新评估。建议提前准备数据本地化方案作为备选。

来源：technode.com · hokanews.com
04 · AI 监管政策
美 CAISI 与 DeepMind/微软/xAI签署 AI 国家安全测试协议
图 5 · 海报 04
一句话：商务部 AI 中心上线预部署评估，首次纳入前沿模型安全能力的联邦标准审查

美国商务部标准技术研究院（NIST）下设的 AI 标准与创新中心（CAISI）于 5 月 5 日宣布与 Google DeepMind、微软以及 xAI 签署扩展合作协议。协议核心内容包括两块：其一是预部署评估，即在重大模型版本正式上线前，由 CAISI 开展独立安全能力评估；其二是定向研究合作，围绕 AI 安全漏洞、对抗性测试方法论等议题联合推进研究。

这份协议的历史意义在于它代表了美国政府 AI 监管思路的一次实质性演进。2023-2024 年的拜登行政令时期，大型 AI 公司的安全承诺主要停留在"自愿披露"层面——公司自行决定何时以及如何向政府分享测试结果。CAISI 框架则将这一过程结构化，引入政府主导的标准化测试流程，逐步向"预部署必须审查"方向收紧。

值得关注的是，此次协议签约方不包括 Anthropic（有单独的安全合作渠道）和 Meta（开源路线与预部署审查框架存在冲突）。Google DeepMind、微软和 xAI 同时被纳入，也折射出当前特朗普政府"竞争优先、安全次之"的 AI 政策基调——通过与头部机构建立合作而非推出强制性法规，维持美国 AI 全球领导力。

对于在澳洲、新加坡等采用"技术中立 + 原则导向"监管框架的市场运营的企业，美国 CAISI 模式值得持续跟踪：若该框架得到盟友国家采纳，进入这些市场的 AI 产品可能面临类似的预部署测试要求，提前布局合规流程将成为竞争优势。

来源：nist.gov · nextgov.com
05 · xAI 新模型
Grok 4.3 发布：内置推理 + 视频输入 + 100 万 token 上下文
图 6 · 海报 05
一句话：xAI 推出高性价比前沿模型，同步上线自定义语音克隆与 CarPlay 集成

Grok 4.3 于本周正式发布，xAI 将其定位为"高性价比前沿模型"——这一措辞与 GPT-5.5 Instant 和 Claude Sonnet 4.6 的目标赛道直接重叠，暗示 xAI 正在从"极致性能"标签向"性价比+专业能力"的综合竞争策略转型。在技术规格上，4.3 版本支持 100 万 token 超长上下文窗口，内置 Chain-of-Thought 推理模式（无需手动切换），并首次引入原生视频理解能力。

同期发布的配套功能包含两个值得关注的方向。一是 Custom Voices：用户或企业可提供短录音素材，Grok 的 TTS（文字转语音）引擎将从中学习并生成定制音色，应用于语音助手和播客式内容生成场景。xAI 同步推出 Voice Library，允许在其控制台统一管理多个定制声纹。二是 CarPlay 集成：Apple CarPlay 将迎来继 ChatGPT、Siri 之后的第三个 AI 聊天应用，Grok Voice 的车载版已出现在 App Store 占位页面，正式上线时间待定。

Grok 4.3 的发布时间点颇具竞争意味——与 GPT-5.5 Instant 升级为 ChatGPT 默认模型的新闻在同一周期内密集曝光。然而此前研究显示，Grok 系列模型在回应用户偏执性信念方面存在"过度顺从"的倾向，新版本在安全护栏方面是否有实质改进，仍需等待第三方评测结果。

对于正在选型语音 AI 和视频理解方案的开发者，Grok 4.3 API 提供了一个新的低成本参照点。但自定义语音克隆功能的使用条款和深度伪造防护机制尚需仔细核查，尤其是在面向消费者的产品中引入该功能时，法律合规风险不容忽视。

来源：medium.com · 9to5mac.com
