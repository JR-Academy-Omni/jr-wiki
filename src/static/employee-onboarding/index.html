    Step 4 · 写文案 / 报价 / 答学员前必读

    公司产品 & Membership 速查

    最容易踩的坑：把会员权益当成独立产品推销。以下数字以代码为准：jr-academy/src/modules/membership/membership-plan.service.ts。

    🚨 红线：考证匠 ≠ 独立订阅产品。 考证匠是 BASIC / PLUS / PREMIUM 三档 membership 全部包含的权益之一。绝不能写"考证匠 ¥X/月" / "考证匠订阅" / "买考证匠送会员"，正确话术是"$8/月 BASIC 会员含考证匠"。
    Tier 月度 (AUD) 年度 (AUD) 年度省 核心权益（除考证匠 + VPN + Tech Interview Handbook）
    BASIC
    基础版    		 $8 $60 37% 课程 95 折 / 内推 / 付费活动免费 / VIP 席位
    PLUS
    进阶版    		 $19.90 $119 50% + AI Tutor 无限 / 大学课 AI 辅导 / 复习助手 / 资料库 2GB
    PREMIUM
    高级版    		 $88 $550 — + 1 对 1 导师 / 模拟面试 / 简历修改 / 岗位代投 10 次

    ❌ 错的角度

    • "考证匠订阅 $X/月"
    • 把考证匠当成猿题库竞品
    • 推考证匠 = 转化考证匠 DAU
    • BASIC + PLUS 权益叠加（PLUS 没有内推 / VIP 席位）
    • 同时推三档

    ✅ 对的角度

    • "$8/月 BASIC 会员含考证匠"
    • 把考证匠当作"留学生综合服务包"切入点
    • 推考证匠 = 转化 BASIC 会员订阅
    • 漏斗：BASIC → PLUS / PREMIUM 升级路径
    • iOS IAP 价格另算（30% 抽成可能 gross-up）

    完整权益对照见 company-products-pricing.md。改价必须同步 3 处：代码 / skill 文档 / 本页。

    Step 5

    工作模式 / 文档 / 找谁

    下面这三块平时翻得最多。不熟没关系，第一周看一遍就有印象。

    🧭 三大工作铁律

    1. Single Responsibility + due date。每条任务 1 个负责人 + 1 个截止时间，进 Notion Projects，群里口头分配不算数。
    2. 求助 4 阶段。阻塞 1 小时内必须求助；先 Google + ChatGPT + 文档；同级 → 上级 → 跨部门；带"问题描述 + 试过什么 + 推荐方案"。
    3. Invoice 流程。报销 / 第三方账单一律发 invoice@jiangren.com.au，写清支出内容 + 附收据，不发个人邮箱。

    🚨 内容红线

    • 禁说"副业 / 月入 X / 包就业"（监管 + 信任）
    • 禁"在当今 / 准备好了吗"等 AI 味句
    • 禁模版化群公告 / 课表海报
    • 禁微信群带工作机密（薪资 / 价格策略）

    ⚠️ Anti-pattern

    • 卡问题 2 小时没人知道你在卡
    • 群 @ 老板问工资（应 1:1 / HR）
    • 紧急用邮件等回复（用电话 / Zoom）
    • 同时 @ 5 人问同一问题（破坏 single responsibility）
