## 🎨 统一设计风格(所有图通用)

**唯一视觉 SoT = `jr-academy-brand/`**(`DESIGN.md` + Brand Board `index.html` + token `tokens/tokens.css`)。下面是从 brand v4.2 token 抄出来的,做图直接照用,别自己编 hex:

| 用途 | 值 |
|------|----|
| 尺寸 | 1242 × 1660(竖版 3:4) |
| 页面底色 | `#FFFCF6`(暖白 `--jr-surface-canvas`) |
| 卡片/面板底 | `#FFFDF8`(`--jr-surface-panel`) |
| 细边框 | `#F3D8B5`(浅橙 `--jr-border-warm`,1–2px) |
| 主文字 / 标识 / 主标题 | `#0D0F12`(`--jr-black`) |
| 强调 / 编号 / "坏消息"/danger | `#FF4D4F`(`--jr-red`,**只做点缀,绝不做大块背景**) |
| 红底配白字时 | 用 `#D92D20`(`--jr-red-aa`,保证对比度) |
| AI / 进度 / tag | `#7B61FF`(紫 `--jr-purple`) |
| 圆角 | 卡片/按钮 8px,大面板 16px |
| Logo 角标 | 取 `jr-academy-brand/assets/logo/jr-box.svg`,放右上或右下小角标 |

**字号(竖版海报,最小 24px,别再小):** display 100+ / headline 60 / title 40 / subtitle 32 / label 30 / body 28 / meta 24。

**基调:** 暖白底 + 黑字 + 红点缀,干净留白,一张图一个核心信息。不要紫蓝当主色、不要黑粗边 offset 阴影、不要全 pill 按钮(那是被废弃的旧 neo-brutalism)。

## 🖼 第 6 张 · 开发者注意(API)

**文字:**
> 接入要改 3 处(开发者收藏)
>
> ① model 换成 `claude-fable-5`
> ② `thinking:{type:"disabled"}` 不能传了,传了报错 → 直接不写这参数
> ③ 要处理"拒答":它会拒某些请求,返回 refusal(不是报错)
> 🆕 新参数 `fallbacks`:被拒自动换别的模型重试

**设计:** 白卡,稍微"代码感"。代码片段(`claude-fable-5` 等)用深色 code 底(`#0D0F12` 底 + 浅字)。①②③ 编号用红色圆标。"🆕" 那行可加紫色 tag 标"新"。

## 📝 正文(发在图文下方)

> Claude 又又又出新模型了。这次不是小升级——Anthropic 6/9 上线的 **Fable 5** 直接架在 Opus 之上,成了公开能用到的天花板。
>

>
> 但它没那么"香":价格是 Opus 4.8 的整整 2 倍($10/$50),还换了新分词器,同样内容多烧约 30% token。所以它不是默认升上去的模型,是"挑硬骨头才用"的。
>
> 简单结论:日常活继续 Sonnet / Opus,真正卡住的超长 agent 任务、最高难度推理再上 Fable 5。开发者接入记得改 model id、删掉 `thinking disabled`、处理 refusal。
