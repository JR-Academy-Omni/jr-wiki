P2 新闻 1 · 科研合作
01 · 科研合作

UMelb × Cyclotek 共建 LARS 放射化学实验室
加速癌症诊断与靶向治疗 · Parkville 校区 · Prof Donnelly 主导

墨尔本大学与放射制药商 Cyclotek 在 Parkville 校区开放 Laboratory for Advanced Radiochemical Science（LARS），将领先科研与学生培训和分子靶向疗法先进制造结合，加速癌症诊断与治疗的全链条研发转化。
实验室定位
LARS 把 UMelb 基础科研能力与 Cyclotek 放射制药制造对接，实现从分子设计到临床样品生产的整合；加强澳洲放射制药本土自主能力，支持临床试验管线扩张
主导研究员
Professor Paul Donnelly（Richard Robson Chair of Chemistry）在 LARS 开展新型放射性药物分子的设计与合成研究，方向指向癌症放射免疫治疗
学生机会
LARS 让学生和研究人员直接接触工业和临床挑战，在 Parkville 校区内与 Cyclotek 工程师协作，大幅缩短发现到临床的转化路径
对你的影响
走 Chemistry / Biomedical Science / Pharmaceuticals / Medical Imaging 方向的同学：UMelb × Cyclotek 是澳洲罕见的「高校 + 产业」放射制药联合实验室，PhD 和 Honours 课题可直接切入癌症放化治疗前沿

P3 新闻 2 · 科研合作
02 · 科研合作

墨大与 AWS 签战略合作意向书
云基础设施 + 研究商业化 + 学生体验 · DVC Research Prof Cassidy 主导

墨尔本大学与 Amazon Web Services（AWS）签署合作意向书，为正式「Strategic Collaboration Agreement」框架打基础，聚焦数字基础设施转型、研究商业化、创业支持和学生体验提升四大方向。
协作框架
意向书明确 AWS 将提供：Solutions Architects 技术支持 + AWS Training and Certification + 创新项目参与（云迁移 / 可持续发展审查）+ 研究和创新项目 AWS 积分资助
副校长表态
副校长（研究）Professor Mark Cassidy 表示，这一合作将助力大学实现研究与教育战略目标；协作超越单纯云迁移，覆盖「加速发现、提升学生体验、推进研究转化」
全校可用
所有 UMelb 研究人员（含研究生）和外部研究合作者，均可通过大学 AWS 研究计算服务平台访问云资源，满足定制化科研需求
对你的影响
走 Cloud / Data Science / AI / IT / 创业 方向的同学：UMelb + AWS 合作意味着更系统的云资源对接和 AWS 认证培训机会；创业方向的同学也有机会通过 AWS 创新项目获得技术资源支持

P4 新闻 3 · 医学研究
03 · 医学研究

全国糖尿病研究中心 ACADI 获 $1460 万联邦医学资助
MRFF 四年资助 · 60+ 机构协作网络 · 全国医学研究策略新信号

澳大利亚全国糖尿病研究中心 ACADI（总部设于墨尔本大学）获联邦政府医学研究未来基金（MRFF）四年资助 $1460 万，整合超 60 个组织的虚拟协作网络，加速改善糖尿病患者生活质量的创新解决方案研发。
资助规模
$14.6 million（约合 $1460 万澳元，4 年），来自联邦政府 Medical Research Future Fund（MRFF）；ACADI 是由 UMelb 主导、整合超 60 个大学 + 医疗机构 + 患者组织 + 产业界的虚拟协作网络
政策背景
属于联邦政府发布「全国卫生与医学研究策略」并加大医学研究投入的一部分；同期 Florey Institute 旗下澳洲癫痫项目亦获 $3000 万拨款，在全澳推行癫痫护理新标准
研究方向
聚焦从基础发现→成果转化→卫生成效的完整链条；UMelb 表示欢迎这一战略方向，将加强从发现到转化到健康成效的整体研究链条
对你的影响
走 Medicine / Health Sciences / Public Health / Biomedical Engineering / Data Science 方向的同学：UMelb 在慢性病管理和医疗创新转化这条线上政策影响力和资金规模突出，是选择科研平台时重要的参考维度
